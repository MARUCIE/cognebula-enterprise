# 场景六：电商企业合规检查

**政策依据**：《电子商务法》第28条；财税〔2018〕103号（跨境电商零售出口退税）；国家税务总局2019年第31号公告

## 第一步：初步判别——电商企业税务特殊性

电商企业涉及**多平台销售、刷单退货、促销折扣、跨境交易**等特殊场景，税务处理较传统企业更复杂。

## 第三步：佐证/验证——电商高频风险

1. **平台数据与税务申报不一致**：金税系统可调取平台数据比对
2. **刷单金额计税**：刷单产生的"收入"需要从收入中剔除，对应的"支出"也不可扣除
3. **促销折扣不合规**：折扣未在发票上体现 → 全额计税
4. **赠品未视同销售**：随单赠品未做视同销售增值税处理

## 第四步：结论/建议

- **数据打通**：电商ERP与财务系统数据对接，自动核对收入
- **发票管理**：平台费用发票月度催收，不留缺口
- **退货台账**：建立退货专项台账，与红字发票一一对应
- **合规刷单处理**：如有刷单行为（虽不建议），须在账务中正确处理
- **跨境优先合规**：跨境电商退税审核严格，单证必须齐全
