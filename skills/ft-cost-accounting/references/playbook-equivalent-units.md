# 场景四：约当产量计算

**政策依据**：《企业产品成本核算制度（试行）》（财会〔2013〕17号）第16条

## 第一步：初步判别——何时需要约当产量

**问题**：月末在产品（未完工产品）如何参与成本分配？

**适用场景**：
- 月末有在产品且金额不容忽略
- 各月在产品数量变动较大
- 产品需经过连续加工过程

**不需要约当产量的情形**：
- 月末在产品数量极少（可全部计入完工品成本）
- 在产品只消耗材料不消耗加工费（直接按材料成本计价）

## 第二步：深入分析——约当产量计算方法

**核心公式**：

在产品约当产量 = 在产品数量 × 完工程度（%）

单位成本 = (月初在产品成本 + 本月投入) ÷ (完工产量 + 在产品约当产量)

**案例**：
- 月初在产品：100件（完工40%），成本20,000元
- 本月投入：加工费80,000元
- 本月完工：400件
- 月末在产品：200件（完工60%）

**计算**：
1. 月末在产品约当产量 = 200 × 60% = 120件（约当）
2. 单位加工费 = (20,000 + 80,000) ÷ (400 + 120) = 192.31元/件
3. 完工品加工费 = 400 × 192.31 = 76,923元
4. 月末在产品加工费 = 120 × 192.31 = 23,077元

## 第三步：佐证/验证——完工程度的确定

**直接材料的完工程度**：
- 生产开始一次投入全部材料 → 在产品材料完工程度=100%（不论加工到哪一步）
- 随加工进度逐步投入 → 按实际投入比例确定

**加工费用（人工+制造费用）的完工程度**：
- 均匀发生：完工程度=实际工序位置/总工序数
- 非均匀发生：按各工序加工工时比重计算

**确定完工程度的方法**：
| 方法 | 适用 | 准确性 |
|-----|-----|-------|
| 实地测定法 | 直接评估完工百分比 | 最准但成本高 |
| 工序折算法 | 按经过的工序数/总工序数 | 适中 |
| 约定百分比法 | 统一按50%（简化处理） | 最简但误差大 |

**工序折算法详例**：
- 产品经3道工序，各工序定额工时为10h、20h、10h（合计40h）
- 第2道工序在产品完工程度 = (10 + 20×50%) / 40 = 50%
- （假设每道工序内部均匀消耗，本工序按50%计）

## 第四步：结论/建议

- **材料与加工分别计算**：材料和加工费用的完工程度通常不同，须分别计算约当产量
- **工序折算最常用**：有工艺路线和定额工时数据时，工序折算法最合理
- **月末盘点**：约当产量计算依赖准确的在产品数量和所在工序，月末必须实地盘点
- **ERP自动化**：成熟的ERP系统可根据MES工序报工数据自动计算约当产量
- **简化处理**：在产品变动不大时，可采用约定50%完工程度（减少核算工作量但需管理层批准）
