# 场景：法规时效性验证

**政策依据**：《立法法》；《规章制定程序条例》；税务规范性文件管理相关规定

## 第一步：法规状态分类

| 状态 | 含义 | 识别方法 |
|------|------|----------|
| 现行有效 | 法规仍在执行 | 官方法规库标注"有效" |
| 全文废止 | 被新法规完全替代 | 新法规明确列出废止文件清单 |
| 部分修订 | 部分条款被修改但主体有效 | 修订公告指明修改的具体条款 |
| 到期失效 | 政策执行期限已过 | 法规原文注明有效期至某日 |
| 暂停执行 | 临时暂停但未正式废止 | 特殊通知 |

## 第二步：时效验证流程

**5步验证法**：

1. **查看法规原文**：确认是否注明执行期限（如"本通知自X年X月X日起执行，至X年X月X日止"）

2. **查找废止/修订公告**：
   - 在国家税务总局法规库搜索"废止 [原文件号]"
   - 查看财政部/税务总局定期发布的废止和失效税收规范性文件目录

3. **查找延续公告**：
   - 临时性优惠政策到期前通常会发布延续公告
   - 搜索"延续 [政策关键词] [年份]"

4. **查看上位法变动**：
   - 法律修订后，下位法可能自动失效
   - 如《个人所得税法》2018年修订后，大量原有规范性文件失效

5. **交叉验证**：在至少两个权威渠道验证（官方法规库 + 税务总局官网 + 中国法律法规信息库）

## 第三步：常见过期/废止陷阱

| 常被错误引用的过期政策 | 实际状态 |
|-----------------------|----------|
| 财税〔2008〕121号（房产税原值含土地） | 仍然有效 |
| 国税发〔2009〕2号（特别纳税调整） | 部分被42号公告替代 |
| 财税〔2016〕12号（教育费附加免征） | 已被整合到综合减免政策 |
| 个人所得税"起征点3500元" | 已废止（2019年起为5000元） |
| 增值税发票认证360天期限 | 已取消（2020年起无期限限制） |

## 第四步：管理建议

- 建立常用法规清单并标注有效期，每季度更新一次
- 订阅国家税务总局/财政部官方公告推送
- 对客户引用的法规，必须先验证时效再回复
- 知识图谱中的法规节点必须包含"有效期"和"废止标记"属性
- 发现过期法规被引用时，同步更新KG和Skill中的引用

**风险提示**：
- 引用已废止法规出具的咨询意见可能导致客户做出错误的税务处理决策
- 部分修订的法规，未修订的条款仍然有效，不可因部分修订而整体弃用
- 临时性优惠政策到期未延续时，企业若继续按优惠执行将面临补税+滞纳金

**关联技能协同**：
- 查询到的法规涉及具体税种适用 --> `ft-tax-advisor` 确认当前执行口径
- 涉及税收优惠政策有效期 --> `ft-tax-incentive` 确认优惠政策当前状态
- 法规变更影响合规检查标准 --> `ft-compliance-checklist` 更新检查清单模板
