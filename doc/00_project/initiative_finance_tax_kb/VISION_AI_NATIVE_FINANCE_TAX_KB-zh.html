AI-Native 知识引擎 vs 传统法规数据库 -- 从"人搜索信息"到"Agent 自主执行业务"的范式迁移。

02
四层架构: 从法规存储到智能决策

原三层架构(法规/操作/合规)基础上新增 L4 智能中心，完成从"数据仓库"到"知识引擎"的升维。

L4

智能中心 Intelligence NEW

Graph RAG 混合检索 + Multi-Swarm 4 层质控 + 法规变更实时检测。将底层三层的静态知识转化为动态推理能力。Cypher 图遍历 + LanceDB ANN 向量召回 + MCP 12 工具链。

L3

合规中心 Compliance

量化风险评分体系 + 行业预警指标 + 处罚案例关联。覆盖增值税税负率、企业所得税贡献率、印花税等 20+ 核心指标，每个指标含行业基准区间和异常阈值。

L2

操作中心 Operation

Agent 做账指令集 -- "看到这张发票，借什么贷什么"。法规条款到会计科目的自动映射(全球独创)，覆盖常见业务场景 500+ 种分录模板。

L1

法规中心 Regulation

50K+ 法规全文 + 条款级拆分(M2 目标 150 万条款节点)。涵盖国家税务总局、财政部、各省级税务机关全量法规，按条款粒度建模，每条款含生效日期、废止状态、关联法规。

03
核心技术栈

五大技术支柱，构建从数据采集到智能推理的完整链路。

Graph Database

KuzuDB 嵌入式图数据库

Cypher 查询语言，500K 节点舒适区。嵌入式架构零运维，单文件部署。123 张表 / 24 种节点类型，支持多跳关系遍历和 OLAP 分析。

Embedding

Gemini Embedding 2 Preview

768 维 Matryoshka 编码，支持 128-3072 维灵活配置。多模态能力(文本/图像/PDF)。LanceDB ANN 索引，13,445 向量，毫秒级语义检索。

Data Ingestion

OpenClaw 24/7 不间断抓取

14 个专用 fetcher 覆盖国家税务总局、财政部、12366 热线、chinaacc 培训、各省级税务机关。CF Browser Rendering 处理 JS 渲染页面。VPS 76 个 cron job 全自动运行。

Quality Control

Multi-Swarm AI Synthesis

4 层流水线: Generator(生成) -- Fact Checker(事实核查) -- Domain Expert(领域专家) -- Red Team(红队攻击)。49% 淘汰率确保知识质量。每条 AI 合成知识必须通过全部 4 层才能入库。

Retrieval & Reasoning

Graph RAG 混合检索

KuzuDB Cypher 图遍历(结构化精确查询) + LanceDB ANN 向量召回(语义模糊检索) + MCP 12 工具链(Agent 原生接口)。三路融合，覆盖从精确法规条款查询到模糊语义联想的全谱段检索需求。

做账

BookKeeping

→
Graph Traversal

→
Output

完整会计分录 + 法规依据 + 风险提示

Agent Output Example

借: 原材料 1,000,000 / 应交税费-应交增值税(进项税额) 130,000
贷: 应付账款 1,130,000
法规依据: 财税[2016]36号 第二十五条 | 风险提示: 需在 360 日内认证抵扣

报税

TaxFiling
Input

"本月需要申报哪些税？"

→
Graph Traversal

Company → Industry → TaxType → FilingStep → FilingForm → Deadline

→
Output

申报清单 + 操作步骤 + 截止日期 + 表单填写指南

Agent Output Example

本月申报清单 (3 项):

2. 企业所得税(季度预缴) -- 截止 4/15 -- 填写 A200000 表
3. 个人所得税(代扣代缴) -- 截止 4/15 -- 自然人电子税务局申报

审计

Audit
Input

"检查制造业增值税税负率是否异常"

→
Graph Traversal

Industry(制造业) → RiskIndicator(税负率) → Threshold(1.5%-3.5%) → Compare(实际 2.1%)

→
Output

正常/异常判断 + 行业基准对比 + 整改建议

Agent Output Example

判断: 正常 (在行业基准区间内)
实际税负率: 2.1% | 行业基准: 1.5%-3.5% | 分位: P38
建议: 当前税负率偏低端，建议核查进项税额抵扣是否存在超期风险

合规

Compliance
Input

"高新技术企业在上海自贸区做软件出口，有哪些叠加优惠？"

→
Graph Traversal

3-hop 遍历 → TaxIncentive(高新 15%) + TaxIncentive(自贸区) + TaxIncentive(软件即征即退) + ExportRefund

→
Output

4 项叠加优惠清单 + 适用条件 + 法规原文 + 会计处理

Agent Output Example

可叠加优惠 (4 项):
1. 高新技术企业所得税优惠 -- 税率 15% (国发[2007]40号) -- 条件: 高新证书有效期内
2. 自贸区企业所得税优惠 -- 15% 或免税 (沪府发[2013]75号) -- 条件: 注册地在自贸区
3. 软件产品增值税即征即退 -- 超 3% 部分即征即退 (财税[2011]100号) -- 条件: 软件检测报告
4. 出口退税 -- 增值税零税率/免抵退 (财税[2012]39号) -- 条件: 出口报关单
综合税负: 约 8.2% (对比标准 25% 所得税率，节税 67%)

08
五大不可复制优势

CogNebula 的核心壁垒不在数据量，而在知识工程方法论和 AI-Native 架构设计。

01

Graph-Native 多跳推理

知识以图谱而非文档存储。支持 3-hop 以上的复杂关联推理 -- 从"高新企业 + 自贸区 + 软件出口"一次查询得到 4 项叠加优惠。传统数据库需要 4 次独立查询。

02

法规到分录自动映射

全球独创。

从法规条款到会计科目到完整分录的端到端自动映射。Agent 看到一张发票，知识引擎直接输出"借什么贷什么"，而非"请参考某某法规第几条"。

03

AI Agent 原生接口

MCP (Model Context Protocol) 12 工具链。Agent 不需要"理解"法规，只需要调用工具。知识引擎是 Agent 的"专业大脑"，Agent 是知识引擎的"手和脚"。

04

量化风险评分体系

20+ 核心风险指标，每个含行业基准区间和异常阈值。不是"可能有风险"的模糊提示，而是"税负率 2.1%，行业基准 1.5%-3.5%，P38 分位"的精确量化。

05

100x 价格优势 + 开放架构

威科先行年费数万元，面向大型企业。CogNebula 嵌入式架构零运维，MCP 开放接口，个人开发者和小微企业均可接入。从"奢侈品"到"基础设施"。
