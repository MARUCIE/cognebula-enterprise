# 场景二：季度申报清单

**政策依据**：《企业所得税法》第54条（季度预缴）；增值税小规模纳税人按季申报规定

## 第一步：初步判别——季度申报涉及的税种

| 税种 | 申报期限 | 适用纳税人 |
|-----|---------|----------|
| 企业所得税预缴 | 季度终了15日内 | 所有企业所得税纳税人 |
| 增值税（季度） | 季度终了15日内 | 小规模纳税人 |
| 附加税（随增值税） | 季度终了15日内 | 小规模纳税人 |
| 房产税 | 各地规定（多为半年/季度） | 有房产的纳税人 |
| 城镇土地使用税 | 各地规定（多为半年/季度） | 有土地的纳税人 |
| 印花税（按季） | 季度终了15日内 | 选择按季申报的纳税人 |

**注意**：企业所得税季度预缴对所有企业适用，是季度最重要的申报事项。

## 第二步：深入分析——季度预缴要点

**企业所得税预缴方法**（两种）：
1. **按实际利润预缴**（多数企业）：按季度实际利润×税率计算
2. **按上年应纳税所得额均摊**：上年应纳税所得额÷4×税率

**预缴申报表关键项**：
- 营业收入：按季度累计数
- 营业成本：按季度累计数
- 利润总额：收入-成本-费用（会计利润）
- 特定业务计算的应纳税所得额
- 减免所得税额（小微优惠/高新优惠等，预缴即可享受）
- 已预缴税额（之前季度已缴的）

**季度预缴优化**：
- 研发加计扣除可在预缴时享受（2023年起）
- 小微企业优惠可在预缴时享受
- 固定资产一次性扣除可在预缴时享受

## 第三步：佐证/验证——季度申报操作日历

```
季度终了后第1-5日：
  □ 季度财务数据汇总
  □ 企业所得税预缴利润测算
  □ 减免税项目确认

季度终了后第6-10日：
  □ 企业所得税预缴申报表填写
  □ 小规模纳税人增值税申报
  □ 房产税/土地使用税申报（如在本季度）
  □ 印花税季度申报

季度终了后第11-15日：
  □ 申报数据复核+提交
  □ 税款缴纳
  □ 申报回执保存

季度终了后第16日起：
  □ 本季度税负率计算与分析
  □ 下季度预计税款
  □ 税务台账更新
```

## 第四步：结论/建议

- **预缴不低估**：季度预缴少缴税款可能被加收滞纳金（部分地区执行严格）
- **优惠即享**：符合条件的优惠在预缴阶段即可享受，不要等到年度汇算
- **房产/土地税**：各地申报频率不同，须确认当地规定
- **四季度注意**：第四季度（10-12月）预缴后还有年度汇算清缴，预缴不是最终
- **现金流安排**：季度末提前预估税款金额，确保账户资金充足
