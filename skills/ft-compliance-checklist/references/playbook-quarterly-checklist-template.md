# 场景二：季度合规检查模板

**政策依据**：《企业所得税法》第54条；企业内部控制基本规范

## 第一步：初步判别——季度检查与月度检查的差异

季度检查在月度检查基础上增加**分析层面**的检查，关注趋势和指标。

## 第二步：深入分析——检查清单

```
═══ 季度税务合规检查表 ═══
检查期间：____年第____季度
检查人：________  复核人：________  日期：________

一、企业所得税预缴检查
  □ 1.1 季度利润计算准确
  □ 1.2 减免税项目计算正确（小微/高新/加计扣除）
  □ 1.3 预缴申报表填报完整
  □ 1.4 已预缴税额累计正确
  □ 1.5 弥补以前年度亏损计算无误

二、增值税季度分析
  □ 2.1 季度增值税税负率计算
  □ 2.2 税负率与行业基准对比（偏离>20%须说明）
  □ 2.3 进项构成分析（异常供应商排查）
  □ 2.4 留抵税额变动分析
  □ 2.5 免税/简易计税收入占比分析

三、收入完整性检查
  □ 3.1 银行流水与申报收入对比（差异<5%）
  □ 3.2 预收账款变动分析（长期挂账排查）
  □ 3.3 关联交易价格合理性
  □ 3.4 视同销售事项排查

四、成本费用合理性检查
  □ 4.1 毛利率季度对比（波动>5%须说明）
  □ 4.2 期间费用率季度对比
  □ 4.3 限额费用累计测算（招待费/广告费）
  □ 4.4 大额咨询费/服务费核查
  □ 4.5 关联方采购价格核查

五、往来款检查
  □ 5.1 应收账款账龄分析
  □ 5.2 其他应收款余额排查
  □ 5.3 股东借款状态检查
  □ 5.4 预付账款合理性检查

六、特殊税种检查
  □ 6.1 房产税/土地税申报（如在本季度）
  □ 6.2 印花税缴纳情况（按季申报的）
  □ 6.3 其他地方税种

七、指标分析报告
  本季度关键指标：
  - 增值税税负率：___% （行业均值：___%）
  - 所得税税负率：___% （行业均值：___%）
  - 毛利率：___% （上季度：___%）
  - 期间费用率：___% （上季度：___%）
```

## 第三步：佐证/验证——季度分析报告

每季度检查后出具简要分析报告，包含：
1. 各项指标与行业/上期对比结果
2. 异常项目清单和原因说明
3. 整改建议和完成时限
4. 下季度关注事项

## 第四步：结论/建议

- **季度预缴审慎**：预缴不宜过低（被加收滞纳金）也不宜过高（资金占用）
- **趋势分析**：季度检查重点看趋势变化，单一季度数据不足以判断
- **管理层汇报**：季度检查结果向财务负责人汇报
- **外部对照**：有条件的企业可委托外部税务师做季度复核
- **年度汇缴准备**：Q3检查后即可开始评估年度汇缴的重大调整事项
