# 场景四：常设机构判定

**政策依据**：各双边税收协定第5条；《企业所得税法》第2条、第3条；国税发〔2010〕75号；OECD税收协定范本第5条注释

## 第一步：初步判别——是否存在常设机构风险

1. **固定场所型PE**：境外企业在中国是否有固定的营业场所（办公室、工厂、车间、矿场等）？
2. **工程型PE**：在中国从事建筑、安装、装配、监督管理活动是否超过规定月数（通常6个月或12个月）？
3. **服务型PE**：在中国提供劳务/服务是否在任何12个月中超过183天？
4. **代理型PE**：在中国是否有代理人习惯性代表境外企业签订合同或接受订单？

**后果**：一旦构成PE → 归属于PE的利润须在中国缴纳25%企业所得税（不再是10%预提税）

## 第二步：深入分析——三类PE判定标准

**固定场所型PE**（最常见）：
- 三要素：固定性（非临时）+ 营业性（从事经营活动）+ 场所性（有物理存在）
- **排除项**：仅用于储存/展示/交付/采购/辅助性/准备性活动的场所不构成PE
- **共享办公/子公司场所**：境外母公司员工长期使用子公司办公室 → 可能构成PE

**工程型PE**（门槛因协定而异）：
| 对方国家 | 工程PE门槛 |
|---------|----------|
| 多数协定 | 6个月（183天） |
| 日本/韩国 | 6个月 |
| 新加坡 | 6个月 |
| 美国 | 6个月 |

- 计算方法：从开工到竣工或最终停止，包括暂停期间
- 分拆合同反避税：关联企业同一项目分拆的多个合同合并计算

**代理型PE**：
- 依赖代理人：经常性代表境外企业签订合同或安排合同签订
- 独立代理人除外：法律和经济上独立 + 在正常业务范围内代理

## 第三步：佐证/验证——常见风险情形

1. **外派员工常驻**：境外公司派员工到中国子公司"协助"，实际从事经营管理 → 固定场所PE
2. **远程办公**：疫情后境外员工在中国远程办公 → 可能被认定为固定场所
3. **电商服务器**：在中国设有服务器 → 部分协定下可能构成PE（OECD范本注释有争议）
4. **合同分拆**：将18个月工程分拆为3个6个月合同 → 税务机关可合并计算
5. **子公司员工双重角色**：子公司员工同时为母公司签订合同 → 代理型PE

**证据收集清单**：
- 员工出入境记录、工作签证类型
- 办公场所租赁合同、使用记录
- 合同签订地和签约人身份
- 项目时间表、进度报告

## 第四步：结论/建议

- **风险评估先行**：境外企业在华活动前，评估PE风险并设计合规方案
- **天数管控**：服务型PE严格控制境内服务天数（<183天/12个月），建立出入境台账
- **合同管理**：明确合同签约主体和签约地点，避免代理型PE
- **工程筹划**：大型工程项目拆分须有真实商业理由，同一项目关联合同将被合并
- **一旦构成PE**：须办理税务登记、据实申报纳税，可抵扣归属于PE的合理费用
