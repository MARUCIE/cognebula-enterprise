# 场景：功能货币（记账本位币）判定

**政策依据**：《企业会计准则第19号——外币折算》第五至九条

## 第一步：理解功能货币的概念

功能货币（准则称"记账本位币"）是指企业经营所处的主要经济环境中的货币。按准则第五条，企业通常应选择人民币为记账本位币。但业务收支以外币为主的企业，可以选择外币作为记账本位币。

功能货币的选择不是任意的，应当反映企业主要经济环境的货币。一旦确定，不得随意变更，除非企业经营所处的主要经济环境发生重大变化。

## 第二步：判断主要经济环境的货币

按准则第六条，企业选定记账本位币应当考虑以下因素：
1. **收入因素**：主要影响商品和劳务销售价格的货币（通常是商品定价和结算的货币）
2. **成本因素**：主要影响商品和劳务成本的货币（通常是人工、材料、其他费用的计价和结算货币）
3. **融资活动货币**：融资取得的货币以及保存经营活动收入的货币
4. **竞争环境**：所在国家或地区的市场竞争格局和监管要求

如果上述因素指向的货币不一致，管理层应综合判断。通常"收入+成本"因素优先级高于"融资"因素。

## 第三步：境外子公司功能货币判定

境外子公司的功能货币可能与母公司不同。按准则第八条，境外经营的记账本位币额外考虑：
- 其活动是母公司的延伸还是具有相当程度的自主性
- 与母公司交易在其全部活动中的比重
- 其现金流量是否直接影响母公司的现金流量
- 其活动产生的现金流量是否足以偿还现有及可预期的债务

如果境外子公司是"销售中心"（产品从母公司进、在当地卖），功能货币可能仍是人民币。如果是独立运营的海外业务，功能货币可能是当地货币。

## 第四步：变更处理与审计要点

功能货币变更按准则第九条采用变更当日的即期汇率将所有项目折算为变更后的记账本位币，折算后金额作为新的记账基础（无追溯调整）。

审计要点：功能货币选择是否有充分的业务支撑依据（而非为了操控汇兑损益而人为选择）。典型风险：选择弱势货币减少负债折算压力，或选择强势货币虚增资产。应获取企业收入/成本/融资币种构成数据，独立验证功能货币选择的合理性。

关键输出：功能货币判定分析（含收入/成本/融资币种占比）+ 境外子公司逐户评估 + 政策说明。
