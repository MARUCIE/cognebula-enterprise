# 场景九：综合风险评估报告模板

**政策依据**：国税发〔2005〕43号（纳税评估管理办法）；企业内部控制基本规范

## 第一步：初步判别——报告适用场景

1. **定期自查**：年度汇算清缴前的全面税务健康检查
2. **专项检查前**：收到税务风险提示或评估通知后的应对准备
3. **并购尽调**：目标企业的税务风险评估
4. **上市筹备**：IPO前的税务合规全面审查

## 第二步：深入分析——报告框架

```
═══════════════════════════════════════
        税务综合风险评估报告
═══════════════════════════════════════

一、企业基本信息
  - 企业名称/税号/注册地/行业/纳税人类型
  - 评估期间/报告日期

二、财务概况
  - 近三年主要财务数据（收入/成本/利润/资产/负债）
  - 增长趋势分析

三、税种分析

  3.1 增值税风险
    - 税负率及行业对比
    - 进项构成分析
    - 异常发票排查结果
    - 风险等级：高/中/低

  3.2 企业所得税风险
    - 税负率及行业对比
    - 收入完整性检查
    - 成本费用合规性检查
    - 优惠政策享受合规性
    - 风险等级：高/中/低

  3.3 个人所得税风险
    - 工资薪金申报完整性
    - 股东借款/分红处理
    - 个税全员申报合规性
    - 风险等级：高/中/低

  3.4 其他税种
    - 印花税/房产税/城镇土地使用税
    - 风险等级：高/中/低

四、专项风险评估
  - 关联交易风险
  - 往来款项风险
  - 发票管理风险
  - 优惠政策合规风险

五、风险汇总矩阵
  | 风险项 | 涉及金额 | 风险等级 | 补税估算 | 整改建议 |
  |-------|---------|---------|---------|---------|

六、整改建议与行动计划
  - 立即整改事项（高风险）
  - 近期整改事项（中风险）
  - 持续关注事项（低风险）

七、附件
  - 指标计算底稿
  - 行业基准数据来源
```

## 第三步：佐证/验证——评分标准

**综合风险评分（100分制）**：
| 维度 | 权重 | 评分标准 |
|-----|-----|---------|
| 增值税合规 | 25分 | 税负率正常+进项合规+发票合规 |
| 所得税合规 | 25分 | 收入完整+成本合理+优惠合规 |
| 个税/其他税 | 15分 | 全员申报+代扣合规+小税种完整 |
| 往来款管理 | 15分 | 无长期挂账+股东借款合规 |
| 内控制度 | 20分 | 税务档案+发票管理+申报流程 |

- 90-100分：低风险，继续保持
- 70-89分：中风险，需针对性整改
- <70分：高风险，须立即启动全面整改

## 第四步：结论/建议

- **年度报告**：建议每年出具一份综合风险评估报告
- **第三方参与**：中大型企业建议聘请税务师事务所出具鉴证性报告
- **跟踪闭环**：报告中的整改建议须分配责任人和完成时限
- **管理层报告**：风险评估结果须向管理层汇报（税务风险最终是企业风险）
- **持续改进**：每期报告与上期对比，追踪风险项改善情况
