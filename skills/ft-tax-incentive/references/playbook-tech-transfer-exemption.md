# 场景六：技术转让所得减免

**政策依据**：《企业所得税法》第27条第4款；《企业所得税法实施条例》第90条；国家税务总局公告2015年第82号

## 第一步：初步判别——适用条件

**减免标准**：
- 技术转让所得≤500万元 → **免征**企业所得税
- 技术转让所得>500万元的部分 → **减半征收**

**技术转让范围**：
1. 居民企业转让专利技术（含国防专利）
2. 转让计算机软件著作权
3. 转让集成电路布图设计专有权
4. 转让植物新品种权
5. 转让生物医药新品种
6. 5年以上（含5年）非独占许可使用权

**注意**：技术转让 ≠ 一般货物买卖，必须涉及技术成果的权属或使用权转移

## 第二步：深入分析——所得计算

**技术转让所得** = 技术转让收入 - 技术转让成本 - 相关税费

| 项目 | 内容 |
|-----|-----|
| 技术转让收入 | 转让方获得的价款（含技术咨询/技术服务/技术培训等配套收入） |
| 技术转让成本 | 该项无形资产的账面净值（已摊销部分不计） |
| 相关税费 | 除企业所得税和可抵扣增值税以外的各项税费 |

**案例**：企业转让一项专利，收入800万，该专利账面净值100万，相关税费20万
- 技术转让所得 = 800 - 100 - 20 = 680万
- 免税部分 = 500万
- 减半部分 = (680 - 500) × 25% × 50% = 22.5万
- **实际缴纳所得税 = 22.5万**（若无优惠则需170万）

## 第三步：佐证/验证——备案与申报要求

1. **技术合同登记**：技术转让合同须到省级以上科技行政部门认定登记
2. **技术出口许可/备案**：向境外转让的须办理技术出口许可或备案
3. **年度申报**：在企业所得税年度纳税申报时填报相关减免税附表
4. **留存备查材料**：
   - 技术转让合同（副本）
   - 科技部门登记证明
   - 技术出口许可/备案文件（如涉及）
   - 技术所有权证明
   - 转让方取得收入的凭证

**关键要求**：
- 不能将非技术转让的其他收入混入技术转让收入
- 配套服务收入（如安装调试）如在合同中明确与技术转让相关，可纳入
- 分期收款的，按合同约定确认收入

## 第四步：结论/建议

- **合同设计**：技术转让合同中明确技术内容、转让范围、价款构成
- **登记先行**：转让前完成技术合同登记（未登记的无法享受减免）
- **500万卡位**：技术转让所得在500万附近时，可通过合理的成本费用归集优化
- **许可vs转让**：5年以上非独占许可等同于转让可享受减免，期限<5年不适用
- **关联交易**：关联方间的技术转让须符合独立交易原则，定价畸低/畸高将被调整
