# 场景三：受益所有人判定

**政策依据**：国家税务总局公告2018年第9号（关于税收协定中"受益所有人"有关问题的公告）；国税函〔2009〕601号（已被9号公告替代但历史参考）

## 第一步：初步判别——何时需要做受益所有人判定

1. **触发场景**：非居民企业申请适用协定优惠税率（股息/利息/特许权使用费）
2. **判定主体**：支付方（扣缴义务人）需评估收款方是否为该所得的"受益所有人"
3. **核心问题**：收款方是否对所得有实质处置权，还是仅作为"导管"传递给第三方

**受益所有人 ≠ 法律登记所有人**。关键是看实质支配权和经济受益权。

## 第二步：深入分析——不利因素与安全港

**2018年9号公告的不利因素清单**：

存在以下因素的，不利于受益所有人判定：
1. 申请人有义务在规定时间内将所得的全部或绝大部分支付或派发给第三国居民
2. 除持有所得据以产生的财产或权利外，申请人没有或几乎没有其他经营活动
3. 申请人资产规模、人员配置、收入规模与其所得金额不匹配
4. 对于有关所得，申请人没有或几乎没有控制权或处置权，也不承担或很少承担风险

**安全港规则**（9号公告第4条，直接判定为受益所有人）：
- 缔约对方政府
- 缔约对方上市公司
- 缔约对方居民且与申请人同一国的个人
- 申请人被上述人100%直接/间接持有（需满足中间层均为中国/对方居民）

## 第三步：佐证/验证——穿透分析方法

**实务操作**：逐层分析资金流向和决策链条

```
境内企业 → 支付股息 → 香港SPV（申请协定5%优惠）→ 实际流向BVI控股公司
```

1. **香港SPV是否有实质**：
   - 员工人数？（仅1-2名秘书 → 不利）
   - 办公场所？（虚拟办公室 → 不利）
   - 年度营业额？（仅收取股息/利息 → 不利）
   - 董事会是否在港召开？
2. **所得处置权**：SPV收到股息后是否有权自主决定使用还是必须上缴BVI母公司？
3. **"导管安排"认定**：如果SPV无实质经营 + 所得自动流转至第三方 → 否定受益所有人

**有利因素补强**：
- SPV有独立管理团队和决策能力
- SPV除持股外有其他经营业务
- SPV过去3年有利润留存记录
- SPV承担实质性商业风险

## 第四步：结论/建议

- **架构设计**：中间控股公司须注入实质（真实员工、办公场所、经营活动）
- **证据准备**：保留董事会会议记录、银行账户流水、员工雇佣记录、租赁合同
- **利润留存**：控股公司应有合理利润留存比例，不应100%自动分配
- **安全港优先**：如可能，让上市公司或政府实体直接持股以获得安全港保护
- **提前沟通**：大额支付（>1,000万元）建议提前与主管税务机关沟通受益所有人判定
