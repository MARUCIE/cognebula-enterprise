# 场景：跨区迁址流程

**政策依据**：《市场主体登记管理条例》第二十四条；《税务登记管理办法》第二十六条至第二十九条

## 第一步：判断迁址类型

| 类型 | 定义 | 复杂度 |
|------|------|--------|
| 同区变更 | 同一区/县范围内迁址 | 低：仅需工商变更地址 |
| 跨区迁移 | 不同区/县之间迁址 | 中：工商+税务同步迁移 |
| 跨市/跨省迁移 | 跨地级市或跨省迁址 | 高：税务清算后重新登记 |

**决策规则**：跨区迁址最复杂的环节是税务迁移。税务机关需完成税务检查/清算后才放行。

## 第二步：工商迁址流程

1. 在迁入地市场监管局申请迁入（取得迁入通知书）
2. 持迁入通知书到迁出地市场监管局办理迁出
3. 迁出地将企业档案移交迁入地
4. 在迁入地市场监管局办理变更登记，领取新营业执照

**注意**：部分城市已实现全程网办，无需线下跑两地市场监管局。

## 第三步：税务迁移流程（核心难点）

**迁出地（原主管税务机关）**：
1. 申请税务迁出
2. 税务机关对企业进行税务检查（可能触发稽查）
3. 清缴所有欠税、罚款、滞纳金
4. 缴销全部未使用发票和税控设备
5. 开具《税务事项通知书》（准予迁出）

**迁入地（新主管税务机关）**：
1. 持迁出税务通知书办理税务迁入
2. 重新核定税种、发票票种
3. 重新签订三方协议
4. 申领新的发票和税控设备

**时间预估**：跨区税务迁移通常需要1-3个月，取决于税务检查的复杂程度。

## 第四步：风险与注意事项

- **税务稽查风险**：迁址触发税务检查是惯例。建议迁址前自查补缴所有税款
- **增值税留抵**：迁出时增值税留抵税额原则上可转移至迁入地继续抵扣（但各地执行有差异）
- **社保/公积金**：需同步办理社保、公积金的机构变更或迁移
- **银行变更**：基本户通常需要在迁入地重新开户（部分银行支持变更）
- **合同/许可证变更**：迁址后需更新所有涉及地址的合同、许可证、资质证书
- **发票空窗期**：迁移过程中可能出现无法开票的空窗期，建议提前储备足够发票量
