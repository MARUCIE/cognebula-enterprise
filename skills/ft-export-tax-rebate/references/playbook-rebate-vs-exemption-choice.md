# 场景十：退税 vs 免税选择

**政策依据**：财税〔2012〕39号第6-7条；国家税务总局公告2012年第24号第9条

## 第一步：初步判别——出口退税的三种政策

1. **退（免）税**：出口环节免征增值税 + 退还购进环节已缴增值税 → 最优
2. **免税**：出口环节免征增值税，但购进环节进项不退（进项转出）
3. **征税（视同内销）**：出口不免不退，按内销征增值税

**适用退（免）税是默认最优选择，但部分情形下企业须在退税和免税间选择。**

## 第二步：深入分析——何时免税比退税更优

**免税适用场景**：
| 场景 | 政策依据 |
|-----|---------|
| 小规模纳税人出口 | 征收率3%已很低，免税即可 |
| 退税率为0的商品 | 只能免税或征税 |
| 来料加工复出口 | 免税不退（进口料件本身免税） |
| 出口企业未在规定期限内申报退税 | 可选择免税处理 |
| 特殊企业（软件出口等） | 政策规定适用免税 |

**选择免税的经济逻辑**：
- 当进项税额极少时（如服务出口、软件出口），退税金额小但申报成本高 → 免税更经济
- 当企业为小规模纳税人时，进项本就不可抵扣 → 免税是唯一选项

## 第三步：佐证/验证——退税 vs 免税对比测算

**案例**：出口商品100万，购进成本80万（进项10.4万），退税率10%

| 项目 | 退（免）税方案 | 免税方案 |
|-----|-------------|---------|
| 出口环节增值税 | 免征 | 免征 |
| 进项退还 | 80×10%=8万 | 0（进项转出） |
| 征退差额成本 | 80×3%=2.4万（转出） | 10.4万（全额转出） |
| 企业实际获得 | 退税8万 | 无退税 |
| 税务成本 | 2.4万 | 10.4万 |
| 申报管理成本 | 较高（需齐全单证） | 较低 |

**结论**：绝大多数情况下，退（免）税优于免税（差距8万/100万出口额=8%）

**免税更优的极端情况**：
- 出口金额极小（退税<1万）但单证准备成本高
- 进项极少（如纯服务型出口无法取得进项发票）
- 商品退税率为0（无选择余地）

## 第四步：结论/建议

- **默认选退税**：有退税资格的商品/企业，应尽一切可能办理退税
- **免税备用**：退税条件不满足时（逾期/单证缺失/退税率为0），选择免税而非视同内销
- **避免视同内销**：视同内销是最差结果（出口还要缴增值税），须极力避免
- **一经选择不可撤回**：选择免税后不可再改为退税（同一笔出口业务），决策前须测算
- **小规模注意**：有出口业务且金额较大时，考虑升级为一般纳税人以获得退税资格
