# 场景：少数股东权益计算

**政策依据**：《企业会计准则第33号——合并财务报表》第三十二条、第四十条

## 第一步：确定少数股东持股比例

少数股东权益 = 子公司所有者权益中不属于母公司的份额。计算基础：子公司所有者权益（按公允价值持续计量基础），而非母公司个别报表中的长期股权投资。

注意多层控股时的有效持股比例：如母公司持A公司80%，A公司持B公司70%，则母公司对B的有效持股=80%×70%=56%，B公司少数股东权益比例=44%（含A公司少数股东通过A间接持有的部分）。

## 第二步：计算期末少数股东权益

少数股东权益 = 子公司期末所有者权益（公允价值基础）× 少数股东持股比例。

子公司期末所有者权益各组成部分：
- 股本/实收资本
- 资本公积（含购买日公允价值调整）
- 盈余公积（购买日后累积提取）
- 未分配利润（购买日后累积，含公允价值调整影响）
- 其他综合收益

少数股东权益在合并资产负债表中单独列示，属于所有者权益项目但不属于归属于母公司的部分。

## 第三步：计算少数股东损益

少数股东损益 = 子公司当期净利润（公允价值基础调整后）× 少数股东持股比例。

在合并利润表中，"净利润"下方分为：归属于母公司所有者的净利润 + 少数股东损益。

特殊情况处理（准则第四十条）：子公司少数股东分担的当期亏损超过其在子公司期初所有者权益中享有的份额时，超出部分仍然冲减少数股东权益（少数股东权益可以为负数），除非少数股东有义务承担额外亏损且有能力弥补。

## 第四步：逆流交易与其他调整

逆流交易（子→母/子→子）中的未实现利润需要在母公司和少数股东之间分担。例如：子公司向母公司销售存货产生未实现利润100万，母公司持股80%：少数股东承担 100×20%=20万，调减少数股东损益和少数股东权益。

其他综合收益的归属：子公司的其他综合收益变动（如外币报表折算差额、可供出售金融资产公允价值变动）也需按比例分配给少数股东。子公司向少数股东分红时：减少少数股东权益，不影响合并利润表。

关键输出：少数股东权益计算表 + 少数股东损益计算表 + 逆流交易分担计算 + 合并所有者权益变动表中少数股东列。
