# 场景：营运资金管理

**政策依据**：财务管理通用方法论；《企业会计准则》相关科目核算要求

## 第一步：营运资金核心指标

**营运资金** = 流动资产 - 流动负债

**现金周转周期（CCC）** = 存货周转天数 + 应收周转天数 - 应付周转天数

| 指标 | 计算 | 含义 |
|------|------|------|
| 存货周转天数 | 365 × 平均存货/营业成本 | 库存占用资金的时间 |
| 应收周转天数 | 365 × 平均应收/营业收入 | 回款需要的时间 |
| 应付周转天数 | 365 × 平均应付/营业成本 | 付款给供应商的时间 |

**CCC越短越好**：说明企业从付出现金到回收现金的时间越短，资金效率越高。CCC为负表示企业先收到客户的钱再付供应商（如零售行业）。

## 第二步：三大杠杆优化

**缩短应收周转天数**：
- 缩短信用期（如从60天改为30天）
- 加速催收（逾期30天即跟进）
- 提供现金折扣（如2/10，net/30）
- 应收账款保理/贴现

**缩短存货周转天数**：
- 优化采购频次（小批量多频次）
- 清理滞销库存（促销/打折/报废）
- 提升需求预测准确性
- JIT（准时制）采购

**延长应付周转天数**（合理范围内）：
- 与供应商协商更长账期
- 优先使用商业信用而非银行贷款
- 利用承兑汇票延长付款

## 第三步：现金缺口测算

```
每日营业成本 = 年营业成本 / 365
现金周转周期 = CCC天数
所需营运资金 = 每日营业成本 × CCC天数
```

**案例**：年营业成本1000万，CCC=90天
- 所需营运资金 = 1000万/365 × 90 ≈ 247万
- 这意味着企业至少需要247万的流动资金维持正常运转

**资金缺口** = 所需营运资金 - 现有营运资金
- 缺口为正 → 需要外部融资
- 缺口为负 → 资金充裕

## 第四步：监控与税务关联

**月度监控看板**：
- CCC趋势图（12个月滚动）
- 三大周转天数分解
- 资金缺口预警

**税务关联**：
- 应付账款长期不付（超过3年）→ 可能被税务要求转入营业外收入缴纳所得税
- 应收账款长期挂账 → 坏账准备与税务扣除的差异处理
- 预收账款 → 增值税纳税义务可能已发生（服务类）
- 库存积压 → 关注是否需要计提跌价准备（税会差异）
