# 场景：审计报告模板

**政策依据**：《中国注册会计师审计准则第1501号——对财务报表形成审计意见和出具审计报告》；CAS 1502-1504

## 第一步：确定审计意见类型

四种审计意见（CAS 1501/1502）：
- **无保留意见**：财务报表在所有重大方面按适用的财务报告编制基础编制，公允反映
- **保留意见**：存在重大错报但不具广泛性，或无法获取充分适当的审计证据但影响不具广泛性
- **否定意见**：存在重大错报且具广泛性
- **无法表示意见**：无法获取充分适当的审计证据且影响具广泛性

决策路径：是否存在重大错报？→ 是否具广泛性？→ 对应意见类型。

## 第二步：编制审计报告要素

标准审计报告包含以下要素（CAS 1501第二十条起）：
1. 标题：含"独立"二字
2. 收件人：按审计业务约定书确定
3. 审计意见段：明确所审计的财务报表、适用的财务报告编制基础、审计意见
4. 形成审计意见的基础段：说明审计准则依据、独立性声明、证据充分性声明
5. 管理层责任段
6. 注册会计师责任段
7. 签章和盖章、事务所地址、报告日期

关键控制点：报告日期不得早于获取充分适当证据的日期。

## 第三步：处理特殊事项

**关键审计事项**（CAS 1504）：上市公司审计必须包含，选取审计中最为重要的事项，说明为何认定及如何应对。**强调事项段**（CAS 1503）：不影响审计意见，但需提请使用者关注（如持续经营重大不确定性）。**其他事项段**：与审计相关但财务报表未列示的事项。

判断标准：是否影响审计意见 → 影响则修改意见类型；不影响则考虑强调事项段。

## 第四步：复核与出具

经项目合伙人复核后签发。确保报告与管理层声明书日期一致。审计报告应当由两名注册会计师签名盖章，并加盖会计师事务所公章。对于上市公司，还需关注CAS 1504关键审计事项的披露充分性。内部审计报告可参照上述结构简化，但应保留发现、建议、管理层回复三段式结构。

关键输出：审计报告（标准格式）+ 管理层声明书 + 关键审计事项说明。
