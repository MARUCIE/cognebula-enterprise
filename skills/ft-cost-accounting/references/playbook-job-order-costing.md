# 场景一：分批法成本核算

**政策依据**：《企业会计准则第1号——存货》第8-10条；《企业产品成本核算制度（试行）》（财会〔2013〕17号）

## 第一步：初步判别——是否适用分批法

**适用条件**：
1. 小批量、多品种生产
2. 按客户订单组织生产
3. 产品批次之间差异较大
4. 能按批次归集直接费用

**典型行业**：机械制造、船舶建造、定制家具、印刷、广告、软件项目

**不适用**：大规模连续生产（应用分步法）、单品种大量生产（应用品种法）

## 第二步：深入分析——分批法核算流程

**成本计算对象**：每一批次订单（生产工单/Job Order）

**核算步骤**：

1. **开立成本计算单**：按生产工单编号建立成本单，记录批次、产品名、数量、开工日期
2. **归集直接材料**：按领料单直接计入对应批次（实际发出成本）
3. **归集直接人工**：按工时记录直接计入对应批次（工时×工时费率）
4. **分配制造费用**：按预定分配率或实际分配率计入各批次

**制造费用分配**：
- 预定分配率 = 预计全年制造费用 ÷ 预计全年分配基数（机器工时/人工工时/人工成本）
- 某批次分配的制造费用 = 该批次实际分配基数 × 预定分配率

**成本计算单示例**：
| 费用项目 | 批次A（100件） | 批次B（50件） |
|---------|--------------|-------------|
| 直接材料 | 50,000 | 30,000 |
| 直接人工 | 20,000 | 15,000 |
| 制造费用（按工时） | 16,000 | 12,000 |
| **合计** | **86,000** | **57,000** |
| **单位成本** | **860** | **1,140** |

## 第三步：佐证/验证——月末在产品处理

**跨月批次的处理**：
- 月末未完工：全部成本留在"生产成本"科目，不分配完工/在产
- 月末完工：全部成本转入"库存商品"
- 部分完工：按完工数量/在产数量比例分摊（或约当产量法）

**检查要点**：
- 领料单与生产工单编号一一对应
- 工时记录准确（打卡/日报/MES系统）
- 制造费用预定分配率年初确定，年末调整差异
- 废料/报废品成本的处理（见playbook-scrap-and-rework）

## 第四步：结论/建议

- **工单管理**：每个客户订单对应唯一生产工单号，作为成本归集的唯一标识
- **实时归集**：材料领用和工时记录随生产过程实时记录，不要月末集中补记
- **费用分配基准**：选择与制造费用驱动因素最相关的分配基准（机器密集型选机时，人力密集型选工时）
- **差异分析**：年末实际制造费用与已分配制造费用的差异须处理（金额小→计入当期损益，金额大→按比例分摊至在产品/产成品/已销产品）
