# 场景：HS编码归类方法

**政策依据**：《中华人民共和国进出口税则》；《商品名称及编码协调制度国际公约》；海关总署令第252号

## 第一步：理解HS编码结构

HS编码（Harmonized System）是国际通用的商品分类编码。中国进出口税则采用10位编码：
- 前6位：国际统一（HS公约规定）
- 第7-8位：中国子目
- 第9-10位：具体商品代码

HS编码共97章（第77章预留未用），分为21类：动物产品(1-5章)、植物产品(6-14章)、化工品(28-38章)、塑料橡胶(39-40章)、纺织品(50-63章)、机器设备(84-85章)、车辆(86-89章)等。

编码直接决定：关税税率、增值税税率、是否征消费税、是否需要进口许可证、是否受贸易管制。

## 第二步：归类基本规则

《税则》归类总规则六条（与国际一致）：
1. **规则一**：按品目条文和类/章注释确定（最基本原则）
2. **规则二(a)**：不完整品、未组装品、未制成品按其完整品归类
3. **规则二(b)**：混合材料/物质按使税则赋予最基本特征的材料归类
4. **规则三**：多个品目适用时，按最具体描述→基本特征→编号最后的品目
5. **规则四**：其他规则无法归类时，按最相似的货品归类
6. **规则五**：包装容器随商品归类（除非可重复使用）
7. **规则六**：子目归类在同级子目间比较

实操优先级：先看品目条文是否有明确列名 → 再看类注/章注是否有排除或包含 → 最后用归类规则解决。

## 第三步：常见归类争议与应对

**机电产品**（84-85章）：注意功能归类原则——多功能机器按主要功能归类。例如带打印功能的扫描仪，如果扫描是主功能则归8471（数据处理设备外围设备），打印是主功能则归8443。

**零部件**：专用零部件随主机归类（第十五类注释二），通用零部件按材质归类（如螺栓归7318）。

**新型产品**：无明确品目的新技术产品，应按功能和用途归类到最接近的品目。例如智能手环可能归9029（计数器）或8517（通信设备），取决于其主要功能。

遇到归类不确定时，可向海关申请预裁定（海关总署令第236号），海关在60日内出具具有法律约束力的归类裁定。

## 第四步：归类工具与风险管理

归类工具：海关总署网站"税则税率查询"、中国海关归类数据库、WCO归类意见查询。企业应建立归类台账，记录每种进出口商品的HS编码及依据。

风险管理：归类错误可能导致——补缴关税和增值税（加收滞纳金，每日万分之五）、海关处罚（归类申报不实）、影响出口退税（退税率与编码挂钩）。建议对归类有争议的商品主动申请预裁定，或咨询报关行/关务顾问。

关键输出：商品HS编码归类分析 + 归类规则适用说明 + 预裁定申请建议 + 归类台账模板。
