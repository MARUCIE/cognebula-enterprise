# 场景八：集团企业合规检查

**政策依据**：《企业所得税法》第41-44条（关联交易）；国家税务总局公告2016年第42号（关联申报）；国家税务总局公告2017年第6号（特别纳税调整）

## 第一步：初步判别——集团企业特殊合规领域

集团企业涉及**关联交易、转让定价、成本分摊、跨区汇总纳税**等特殊问题。

## 第二步：深入分析——检查清单

```
═══ 集团企业合规检查表 ═══
集团名称：________
检查期间：____年度
检查人：________  日期：________

一、关联交易合规
  □ 1.1 关联方清单更新（持股/控制关系变动）
  □ 1.2 各类关联交易汇总
        - 货物购销：____万
        - 服务提供：____万
        - 资金借贷：____万
        - 特许权使用费：____万
        - 其他：____万
  □ 1.3 各类交易定价方法合理（可比非受控/成本加成/利润分割等）
  □ 1.4 关联交易价格与市场价差异在合理范围内

二、转让定价文档
  □ 2.1 是否达到同期资料准备门槛（>2亿）
  □ 2.2 主体文档准备状态
  □ 2.3 本地文档准备状态
  □ 2.4 国别报告（>55亿合并收入）
  □ 2.5 特殊事项文档（成本分摊/资本弱化）

三、集团内部服务费
  □ 3.1 管理服务费收取有真实服务协议
  □ 3.2 服务内容有具体可验证的交付物
  □ 3.3 收费标准有合理的分摊基准
  □ 3.4 不得扣除的"股东活动"已排除

四、资金借贷合规
  □ 4.1 关联方借款利率不超过金融企业同期同类贷款利率
  □ 4.2 债资比合规（一般企业2:1，金融5:1）
  □ 4.3 超出部分利息不得税前扣除
  □ 4.4 资本弱化特殊事项文档（如适用）

五、汇总纳税管理
  □ 5.1 总分机构分配表准确（三因素法）
  □ 5.2 分支机构预缴按期完成
  □ 5.3 新设/撤销分支机构处理正确
  □ 5.4 亏损弥补在集团层面统一计算

六、关联申报
  □ 6.1 年度关联业务往来报告表填报
  □ 6.2 与所得税汇缴同步提交（5月31日前）
  □ 6.3 关联交易金额与财务数据一致
```

## 第三步：佐证/验证——集团税务风险矩阵

| 风险领域 | 风险等级 | 检查频率 |
|---------|---------|---------|
| 转让定价 | 高 | 年度 |
| 管理费扣除 | 高 | 半年度 |
| 资金借贷 | 中 | 季度 |
| 汇总纳税 | 中 | 季度 |
| 成本分摊 | 中 | 年度 |

## 第四步：结论/建议

- **集团税务架构**：设立集团税务管理部门或指定税务统筹岗
- **定价政策统一**：制定集团关联交易定价政策文件，各成员统一执行
- **同期资料**：每年6月30日前完成上年度同期资料
- **关联申报**：与汇算清缴同步，5月31日前提交
- **税务总监报告**：每年度向集团管理层出具税务风险管理报告
