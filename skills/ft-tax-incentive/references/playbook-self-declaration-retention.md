# 场景十：自行判别申报享受留存备查

**政策依据**：国家税务总局公告2018年第23号（企业所得税优惠事项管理办法）；各项优惠政策中的备案/备查要求

## 第一步：初步判别——新制度核心变化

**2018年23号公告的重大改变**：
- **旧制度**：事前审批或事前备案 → 税务机关审核同意后才能享受
- **新制度**：自行判别 → 申报享受 → 相关资料留存备查

**三步走**：
1. **自行判别**：企业自行判断是否符合优惠条件
2. **申报享受**：在纳税申报时直接填报优惠，无需事前提交材料
3. **留存备查**：相关资料由企业保存，税务机关后续核查时提供

**适用范围**：企业所得税各项优惠事项（高新、小微、研发加计、技术转让、西部大开发等）

## 第二步：深入分析——留存备查材料体系

**通用材料**（所有优惠事项）：
1. 企业享受优惠事项的相关合同/协议
2. 相关指标的统计数据和计算过程
3. 自行判别的说明文件（判别理由和依据）

**各优惠事项专项材料**：
| 优惠事项 | 专项材料 |
|---------|---------|
| 高新技术企业 | 高新证书、知识产权证书、研发费明细、科技人员名单 |
| 小型微利企业 | 从业人数和资产总额的计算过程 |
| 研发加计扣除 | 研发项目立项文件、辅助账、研发费归集表 |
| 技术转让减免 | 技术合同、科技部门登记证明、收入凭证 |
| 西部大开发 | 鼓励类产业收入占比计算、实质经营证明 |
| 加速折旧/一次性扣除 | 固定资产购置合同、发票、折旧计算表 |
| 软件企业优惠 | 软件企业认定证书、产品检测报告 |

## 第三步：佐证/验证——风险管控要点

**留存备查期限**：优惠事项发生年度的企业所得税汇算清缴完毕后**10年**

**后续管理方式**（税务机关核查）：
1. **案头审核**：通过申报数据比对分析发现疑点
2. **调取资料**：要求企业限期提供留存备查材料
3. **实地核查**：现场核实经营情况和资料真实性
4. **第三方核实**：向相关部门（科技/工信/人社等）核实信息

**常见风险**：
| 风险 | 后果 |
|-----|-----|
| 不符合条件但申报享受 | 补缴税款+滞纳金（自申报之日起）|
| 材料不完整无法提供 | 视为不符合条件，取消优惠 |
| 未按规定保存10年 | 材料灭失时视为不符合条件 |
| 享受条件发生变化未及时调整 | 补缴差额税款 |

**不构成偷税的情形**：
- 诚信申报但对政策理解有偏差 → 补税+滞纳金，不处罚
- 与"故意欺骗隐瞒"的偷税有本质区别

## 第四步：结论/建议

- **判别文档化**：每项优惠的自行判别过程须形成书面说明，留档备查
- **材料即时归集**：享受优惠当年即开始归集备查材料，不要等到核查时才补
- **10年保存**：建立专门的税收优惠档案，电子+纸质双重保存
- **年度复核**：每年度汇算清缴前复核享受条件是否仍然满足
- **风险评估**：对把握不准的优惠事项，建议聘请税务师事务所出具鉴证报告（增强合规确信度）
