# 场景：预提费用税前扣除

**政策依据**：《企业所得税法实施条例》第九条（权责发生制）；国家税务总局公告2011年第34号

## 第一步：预提费用的税务认定

**会计处理**：按权责发生制，费用在发生期间确认为负债（应付账款/其他应付款/预提费用等）。

**税法态度**：税法同样认可权责发生制，但附加条件——费用必须"实际发生"且有合理凭证。

| 预提类型 | 税务可扣除性 | 关键条件 |
|----------|-------------|----------|
| 已发生、年底未付款未取得发票 | 预缴时暂按暂估扣除，汇算前补齐发票可扣 | 5月31日前取得发票 |
| 已确认义务但金额为估计 | 合理估计可扣，最终金额确定时调整 | 估计须有依据 |
| 计提但未实际发生 | 不可扣除，须纳税调增 | 典型：计提未发的年终奖 |
| 准备金类（坏账/减值/保修） | 不可扣除（特殊行业除外） | 银行/保险/证券有特殊规定 |

## 第二步：常见预提费用处理

| 费用项目 | 会计计提 | 税务扣除规则 |
|----------|----------|-------------|
| 年终奖 | 12月计提 | 次年5月31日前实际发放的可在计提年度扣除 |
| 工资 | 按月计提 | 次年汇算清缴前实际发放的可扣除 |
| 水电费 | 按月计提（账单滞后） | 取得发票后可扣除 |
| 利息 | 按月计提 | 需有合同+利息支付凭证 |
| 租金 | 按月分摊 | 一次性收付的按合同期均匀确认 |
| 维修费 | 发生时计提 | 取得发票后可扣除 |
| 产品保修费 | 按销售额比例计提 | 实际发生保修支出时才能扣除 |

## 第三步：汇算清缴关键时点

**34号公告明确**：
- 企业当年度实际发生的相关成本费用，由于各种原因未能及时取得该成本费用的有效凭证，企业在预缴季度所得税时可暂按账面发生金额核算
- 但在汇算清缴时（次年5月31日），应补充提供该成本费用的有效凭证

**决策规则**：
- 5月31日前取得凭证 → 在费用发生年度扣除
- 5月31日后取得 → 追补或在取得年度扣除
- 最终未取得且无法证明实际发生 → 永久不可扣除

## 第四步：风险提示

- **年终奖特别规则**：12月计提但次年6月以后才发放的年终奖，不能在计提年度扣除，须纳税调增
- **预提准备金**：产品质量保证金、售后服务费等，会计预提但税法不认可（除特殊行业外），须全额调增
- **费用真实性**：大额预提费用是税务稽查重点。年底突然大额计提的费用须有充分依据
- **跨期调整**：预提金额与实际发生差异较大的，需在实际发生年度调整
