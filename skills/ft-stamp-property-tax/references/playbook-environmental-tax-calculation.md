# 场景：环保税计算——排放量x税额

**政策依据**：《中华人民共和国环境保护税法》（2018年1月1日施行）；《环境保护税法实施条例》

## 第一步：确定应税污染物与纳税义务

应税污染物四大类：大气污染物、水污染物、固体废物、噪声。纳税义务人：直接向环境排放应税污染物的企事业单位和其他生产经营者。

**不征税情形**（环保税法第四条）：
- 向依法设立的污水集中处理场所排放应税污染物（即入管网处理的）
- 向依法设立的生活垃圾集中处理场所排放应税固体废物
- 在符合标准的设施/场所贮存或处置固体废物
- 达标排放的规模化畜禽养殖（有省级标准的除外）

## 第二步：计算应税排放量

各类污染物排放量的确定方法（按优先级）：
1. **自动监测数据**：安装在线监测设备的，以实际监测数据为准
2. **监测机构数据**：环保部门委托的监测机构出具的数据
3. **排污系数/物料衡算**：按照国务院生态环境主管部门公布的排污系数和物料衡算方法计算
4. **抽样测算**：按抽样测算的方法核定

大气/水污染物：按污染当量数（排放量÷当量值）从大到小排列，大气取前三项、水取前五项（第一类水污染物全部计征）。固体废物：按排放量（吨）。噪声：按超标分贝数。

## 第三步：适用税额标准与计税

各类污染物税额标准（法定最低标准，各省可在此基础上上浮最多10倍）：
- **大气污染物**：每污染当量1.2-12元
- **水污染物**：每污染当量1.4-14元
- **固体废物**：煤矸石5元/吨、尾矿15元/吨、危险废物1000元/吨、冶炼渣/粉煤灰/炉渣/其他25元/吨
- **噪声**：超标1-3dB每月350元，每递增3dB翻倍，至超标16dB以上每月11200元

计税公式：
- 大气/水：应纳税额 = 污染当量数 × 适用税额
- 固体废物：应纳税额 = 排放量（吨）× 适用税额
- 噪声：应纳税额 = 超标档次对应月税额

## 第四步：减免优惠与申报

减征优惠（环保税法第十三条）：
- 排放浓度低于排放标准30%的：减按75%征收
- 排放浓度低于排放标准50%的：减按50%征收

申报：按月计算、按季申报（季度终了后15日内）。纳税地点：排放地主管税务机关。申报时需填报排放口信息、污染物种类和排放量、适用税额和减免情况。

环保部门与税务部门信息共享：环保部门将排污许可、监测数据等信息共享给税务部门，税务部门据此核实申报数据。

关键输出：应税污染物识别 + 排放量计算底稿 + 污染当量换算 + 税额计算表 + 减征优惠适用分析。
