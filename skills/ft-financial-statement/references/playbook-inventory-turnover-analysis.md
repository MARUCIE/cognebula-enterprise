# 场景：存货周转分析

**政策依据**：《企业会计准则第1号——存货》；财务分析通用方法论

## 第一步：核心指标计算

| 指标 | 公式 | 含义 |
|------|------|------|
| 存货周转率 | 营业成本 / 平均存货 | 存货一年周转几次 |
| 存货周转天数 | 365 / 存货周转率 | 存货从入库到卖出的平均天数 |
| 平均存货 | (期初存货 + 期末存货) / 2 | |

**行业参考**：
| 行业 | 周转天数参考 |
|------|-------------|
| 快消/食品 | 30-60天 |
| 零售/商贸 | 45-90天 |
| 制造业 | 60-120天 |
| 建筑/工程 | 90-180天 |
| 奢侈品/珠宝 | 180-360天 |

## 第二步：异常信号识别

| 异常表现 | 可能原因 | 风险 |
|----------|----------|------|
| 周转天数持续增加 | 滞销/过量采购/市场萎缩 | 跌价风险+资金占用 |
| 周转天数骤降 | 少计存货/成本多结转 | 账务可能有误 |
| 存货占资产比过高（>50%） | 库存积压严重 | 流动性风险 |
| 存货账面与实际严重不符 | 管理不善/盘点不规范 | 减值+税务风险 |
| 原材料周转正常但产成品积压 | 生产与销售不匹配 | 需调整生产计划 |

## 第三步：存货结构分析

将存货按类别拆分分析：

```
存货类别 | 金额 | 占比 | 周转天数 | 同比变动
---------|------|------|---------|--------
原材料   | XX万 | XX%  | XX天    | +/-XX天
在产品   | XX万 | XX%  | XX天    | +/-XX天
产成品   | XX万 | XX%  | XX天    | +/-XX天
库存商品 | XX万 | XX%  | XX天    | +/-XX天
```

**决策规则**：
- 原材料周转慢 → 采购过量或生产计划不合理
- 在产品周转慢 → 生产效率低或产线瓶颈
- 产成品/库存商品周转慢 → 销售不力或产品滞销

## 第四步：管理建议与税务关联

**管理行动**：
- 建立安全库存量标准，避免过量采购
- 对滞销品（超过正常周转天数2倍的）进行促销清理或计提跌价准备
- 每月盘点+每季全面盘点

**税务关联**：
- 存货跌价准备不得税前扣除，需做纳税调增
- 存货非正常损失（管理不善导致的霉烂变质/被盗等）对应进项税额需转出
- 存货报废需留存内部审批单+报废清单+评估报告等备查
- 期末存货金额骤降但无对应销售收入或报废记录的，可能被质疑存货去向
