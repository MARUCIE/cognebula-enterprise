# 场景：专精特新认定

**政策依据**：工信部《优质中小企业梯度培育管理暂行办法》（工信部企业〔2022〕63号）

## 第一步：理解专精特新三级梯度

工信部建立了优质中小企业梯度培育体系（63号文）：
- **创新型中小企业**（省级认定，入门级）：基本创新能力达标
- **专精特新中小企业**（省级认定，核心层）：专业化、精细化、特色化、新颖化
- **专精特新"小巨人"企业**（国家级认定，顶层）：市场占有率高、创新能力强、掌握关键核心技术

逐级培育逻辑：先认定创新型中小企业 → 培育为专精特新 → 优中选优为"小巨人"。

## 第二步：专精特新认定核心指标

按63号文第七至九条，专精特新中小企业认定条件：

**基本条件**：在中国境内注册、近2年未被列入经营异常名录或严重违法失信名单、提供的产品不属于禁限类。

**专项条件**（满足其中一项）：
1. 近3年获得过省级以上科技奖励
2. 获得高新技术企业认定且研发投入占营收比>=3%
3. 至少1项主营业务相关的发明专利或2项实用新型专利
4. 主导或参与制修订至少1项国家/行业/团体标准

**通用条件**（评分制，满分100分，60分以上）：
- 专业化：从事特定细分领域年限>=3年、主营业务收入占比>=60%
- 精细化：资产负债率适中、净利润率正增长
- 特色化：产品差异化特征明显、区域市场占有率
- 新颖化：研发费用占比、研发人员占比、科技成果转化

## 第三步：申报流程与材料

申报渠道：各省中小企业公共服务平台（如"优质中小企业梯度培育平台"）。申报窗口：通常每年1-2批，各省时间不同。

核心材料：企业基本信息表、营业执照、财务报表（近3年）、研发费用专项审计报告、知识产权证书、产品检测/认证报告、主要客户/合同证明、荣誉证书（省级以上奖励）。

评审流程：企业在线填报 → 区/县初审 → 市级复审 → 省级终审 → 公示 → 认定公告。认定有效期3年，到期重新评审。

## 第四步：认定后的政策红利

专精特新企业可享受：
- **财政奖励**：各地10-200万不等（"小巨人"奖励更高，部分地方达300-600万）
- **融资便利**：专精特新企业专属贷款产品（利率优惠0.5-1个百分点）
- **上市绿色通道**：北交所优先服务专精特新企业
- **政府采购优先**：中小企业政府采购预留份额中优先
- **人才政策**：部分地方给予核心人才落户/住房优惠

关键输出：梯度定位评估 + 认定条件自查表 + 申报材料准备清单 + 评分预估 + 后续政策红利规划。
