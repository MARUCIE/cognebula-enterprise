# 场景七：资产重组特殊性税务处理

**政策依据**：《企业所得税法》第2章；财税〔2009〕59号；财税〔2014〕109号；国家税务总局公告2010年第4号；国家税务总局公告2015年第48号

## 第一步：初步判别——重组类型识别

1. **企业法律形式改变**：有限→股份制等
2. **债务重组**：以非货币资产/股权清偿债务
3. **股权收购**：收购被收购企业股权≥50%
4. **资产收购**：收购被收购企业资产≥50%
5. **合并**：吸收合并或新设合并
6. **分立**：存续分立或新设分立

**决策路径**：先判断重组类型 → 再判断能否适用特殊性税务处理 → 计算税负差异

## 第二步：深入分析——特殊性税务处理五大条件

同时满足以下条件方可适用（财税〔2009〕59号第5条）：

| 条件 | 具体要求 |
|-----|---------|
| 合理商业目的 | 不以减少/免除/推迟缴纳税款为主要目的 |
| 比例要求 | 股权/资产收购比例≥50% |
| 对价形式 | 股权支付比例≥85%（股权收购/资产收购） |
| 经营连续性 | 被收购/合并企业12个月内不改变原实质性经营活动 |
| 股权连续性 | 取得股权的股东12个月内不得转让 |

**特殊处理核心效果**：资产/股权按原计税基础确认 → 暂不确认转让所得 → 递延纳税

**测算（资产收购1亿，原计税基础4,000万）**：
- 一般处理：(10,000 - 4,000) × 25% = 1,500万 EIT
- 特殊处理：暂不纳税，计税基础延续4,000万

## 第三步：佐证/验证——申报与备案要点

1. **备案时限**：重组业务完成当年企业所得税年度申报时，向主管税务机关报送资料
2. **必备材料**：重组协议/合同、股东会决议、评估报告、工商变更证明
3. **跨年重组**：注意重组日期的确认（工商变更登记日 vs 协议生效日）
4. **亏损弥补**：合并中被合并企业亏损可在限额内由合并企业弥补（年限额 = 被合并企业净资产 × 截至合并时的国债利率）
5. **后续管理**：12个月锁定期内不得转让/改变经营，否则需追溯调整

**风险点**：
- 不满足85%股权支付比例 → 全额按一般处理纳税
- 12个月内转让股权/改变经营 → 追溯纳税+滞纳金

## 第四步：结论/建议

- **交易架构设计**：确保股权对价≥85%，现金对价控制在15%以内
- **时序安排**：重组完成后12个月锁定期的经营和股权安排提前规划
- **增值税协同**：资产重组中的不动产、土地使用权转让可能涉及增值税（非56号文范围）
- **土增税联动**：企业合并/分立中房产土地转移，满足条件可暂不征土增税（财税〔2018〕57号）
- **专业评估**：重组交易建议聘请税务顾问出具税务尽调报告，避免因条件不满足导致巨额补税
