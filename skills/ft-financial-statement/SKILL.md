# 财务报表编制与分析

三大报表编制/关键比率计算/报表间勾稽关系验证/报表异常识别。代理记账最终输出的核心产品——报表质量直接反映记账水平。

## Core Knowledge

### 三大报表结构

**资产负债表(时点数，如12月31日)**
```
资产 = 负债 + 所有者权益

资产:
  流动资产: 货币资金 + 应收票据 + 应收账款 + 预付账款 + 存货 + 其他流动资产
  非流动资产: 固定资产 + 无形资产 + 长期股权投资 + 在建工程 + 递延所得税资产

负债:
  流动负债: 短期借款 + 应付票据 + 应付账款 + 预收账款 + 应付职工薪酬 + 应交税费
  非流动负债: 长期借款 + 应付债券 + 递延所得税负债

所有者权益: 实收资本 + 资本公积 + 盈余公积 + 未分配利润
```

**利润表(期间数，如1-12月)**
```
营业收入
- 营业成本
= 毛利
- 税金及附加
- 销售费用
- 管理费用
- 研发费用
- 财务费用
+/- 投资收益/公允价值变动/资产处置/营业外收支
= 利润总额
- 所得税费用
= 净利润
```

**现金流量表(期间数)**
```
经营活动现金流(OCF): 销售收款 - 采购付款 - 工资 - 税费
投资活动现金流(ICF): 处置资产收入 - 购置资产支出 - 投资支出
筹资活动现金流(FCF): 借款收入 - 还款支出 - 分红支出
现金净增加额 = OCF + ICF + FCF
期末现金 = 期初现金 + 现金净增加额
```

### 勾稽关系(必须验证)

| 关系 | 验证公式 | 容差 |
|---|---|---|
| 资产负债表平衡 | 资产合计 = 负债合计 + 所有者权益合计 | 0(必须绝对平衡) |
| 利润表→资产负债表 | 期末未分配利润 = 期初未分配利润 + 净利润 - 分红 - 盈余公积计提 | 0 |
| 现金流量表→资产负债表 | 期末货币资金 = 现金流量表期末现金及等价物余额 | 0(理论上，实务中可能有外币折算差异) |
| 间接法验证 | 净利润 + 折旧摊销 + 减值 + 财务费用 +/- 营运资本变动 ≈ 经营活动现金流 | 允许小额差异 |
| 所得税验证 | 利润表所得税费用 ≈ 应交所得税 + 递延所得税变动 | 0 |

### 关键财务比率

| 类别 | 比率 | 公式 | 健康范围 | 注意 |
|---|---|---|---|---|
| 偿债 | 流动比率 | 流动资产/流动负债 | 1.5-2.0 | <1 有短期偿债风险 |
| 偿债 | 速动比率 | (流动资产-存货)/流动负债 | 0.8-1.0 | 排除存货变现不确定性 |
| 偿债 | 资产负债率 | 总负债/总资产 | 40%-60% | 行业差异大 |
| 盈利 | 毛利率 | (收入-成本)/收入 | 行业不同 | 连续下降是预警信号 |
| 盈利 | 净利率 | 净利润/收入 | 行业不同 | 关注与毛利率的差距 |
| 盈利 | ROE | 净利润/平均所有者权益 | >8% | 杜邦分析可拆解 |
| 营运 | 应收账款周转天数 | 365/(收入/平均应收) | <90天 | >180天关注坏账风险 |
| 营运 | 存货周转天数 | 365/(成本/平均存货) | 行业不同 | 零售<30天/制造<90天 |
| 营运 | 总资产周转率 | 收入/平均总资产 | >0.5 | 轻资产行业>1.0 |
| 现金 | 经营现金流/净利润 | OCF/净利润 | >1.0 | <0.5说明利润质量差 |

### 小企业会计准则 vs 企业会计准则

| 差异项 | 小企业会计准则 | 企业会计准则 |
|---|---|---|
| 适用对象 | 不对外筹资、规模较小 | 所有企业(上市/大中型) |
| 报表格式 | 简化版(项目更少) | 完整版 |
| 公允价值 | 不使用 | 金融资产/投资性房地产使用 |
| 资产减值 | 实际发生时确认，不计提 | 期末评估计提减值准备 |
| 所得税 | 应付税款法(不确认递延) | 资产负债表债务法(确认递延) |
| 长期股权投资 | 成本法 | 成本法/权益法/公允价值 |

## Decision Framework

| 场景 | 处理方式 | 关键判断 |
|---|---|---|
| 月度报表编制 | 先做凭证→试算平衡→出报表 | 资产负债表必须平衡 |
| 季度申报用报表 | 资产负债表+利润表(不需现金流) | 检查与上季度逻辑连续性 |
| 年度报表 | 三表+附注(完整版) | 勾稽关系全部验证 |
| 银行贷款用报表 | 重点关注偿债指标 | 银行看资产负债率和现金流 |
| 税务检查用报表 | 与申报表数据一致 | 收入/成本/利润与申报表交叉核对 |
| 投资尽调用报表 | 需要审计报告 | 超出代账范围，建议请事务所 |

## Gotchas

1. **现金流量表不是"现金收支记录"** -- 现金流量表是按经营/投资/筹资三类对现金收支做重分类。归还贷款本金毫无疑问是筹资活动。利息支出的分类有两种做法：CAS 31允许将利息支出列入筹资活动或经营活动。国内实务中大多数企业将借款利息归入"分配股利、利润或偿付利息支付的现金"(筹资活动)，但也有企业(特别是金融机构)放在经营活动。关键是保持一致性并在附注中披露分类政策。

2. **资产负债表是时点数，利润表是期间数，不能直接相除** -- 计算ROE时，分子(净利润)是1-12月的累计数，分母(所有者权益)是12月31日的时点数。正确做法是用平均所有者权益=(期初+期末)/2。直接用期末数会低估ROE。应收账款周转率同理。

3. **合并报表的少数股东权益不是负债** -- 少数股东权益在合并报表中列示在所有者权益部分，不是负债。但计算"资产负债率"时，有的分析师把它当负债处理(因为"不是母公司的钱")，这是一种分析口径选择，不是会计错误——但必须标注口径。

4. **报表附注和报表同等重要** -- 审计师最先看附注，不是报表数字。附注说明会计政策选择(折旧方法/收入确认/坏账计提比例)、重大事项(诉讼/担保/关联交易)、期后事项。出报表不出附注等于半成品。

5. **小企业会计准则和企业会计准则的报表格式不同** -- 两套准则的资产负债表和利润表项目不完全相同(小企业准则没有递延所得税、没有公允价值变动)。不能用小企业准则的账套直接套企业会计准则的报表模板，会出现项目对不上。切换准则需要追溯调整。

6. **经营活动现金流为负但净利润为正 = 高风险信号** -- 说明企业赚的是"纸面利润"(应收账款增加、存货积压)，实际没有收到钱。连续2个季度出现这种情况，必须在报表分析中高亮标注，并建议客户加强回款管理。

7. **期末应交税费为负数不一定是错** -- 应交税费借方余额(负数)通常表示预缴税款或留抵税额。增值税留抵、预缴企业所得税都会导致这个科目出现借方余额。但如果长期为负且金额很大，要排查是否有多缴/错缴/未及时退税的情况。

8. **毛利率突然大幅波动需要解释** -- 毛利率从30%突然跳到50%或降到10%，最常见的原因是：收入和成本的确认期间不匹配(如12月确认了收入但成本延到1月)、成本归集错误(把管理费用放进了生产成本)、关联交易定价异常。不是所有波动都是问题，但都需要附注说明原因。

## When to Escalate

- 资产负债表不平衡：停止出报表，先查账(通常是凭证漏录或科目余额方向错误)
- 客户要求出具审计报告：超出代账范围，需要注册会计师事务所
- 涉及合并报表编制：子公司抵销和少数股东权益处理复杂，需要高级会计师
- 需要追溯调整以前年度(准则切换/差错更正)：影响以前年度报表，需要人工判断调整方案
- 客户的现金流量表间接法和直接法差异超过5%：说明分类有系统性错误，需要逐笔核查
