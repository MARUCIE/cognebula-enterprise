# 场景：进项税额转出6种情形

**政策依据**：《增值税暂行条例》第十条；财税〔2016〕36号附件1第二十七条、第二十九条

## 第一步：明确6种必须转出的情形

1. **用于简易计税项目**：原已抵扣进项的货物/服务转用于简易计税项目
2. **用于免征增值税项目**：如免税农产品、残疾人福利企业等
3. **用于集体福利或个人消费**：如购买物资用于员工食堂、节日礼品分发
4. **非正常损失的购进货物及运输**：因管理不善被盗/丢失/霉烂变质
5. **非正常损失的在产品/产成品所耗用的购进货物**
6. **不动产/在建工程发生非正常损失**：涉及其耗用的购进材料、设计、施工等

## 第二步：转出金额计算

| 情形 | 计算方法 |
|------|----------|
| 直接转用（可明确归属） | 按购进时的进项税额全额转出 |
| 不动产/固定资产转用 | 按净值比例转出 = 净值 ÷ 原值 × 已抵扣进项 |
| 无法准确划分（兼营） | 按销售额比例计算 = 不得抵扣销售额 ÷ 全部销售额 × 当期无法划分进项 |
| 农产品（计算抵扣的） | 按买价 × 扣除率 转出 |
| 免税购进（进口环节已抵扣） | 按进口环节已抵扣税额转出 |

**不动产净值比例转出**详细公式：
```
转出税额 = 不动产净值 ÷ 不动产原值 × 已抵扣进项税额
不动产净值 = 原值 - 已计提折旧
```

## 第三步：账务处理

**转出分录**：
```
借：管理费用/营业外支出/在建工程/应付职工薪酬（视转出原因定科目）
    贷：应交税费——应交增值税（进项税额转出）
```

**申报处理**：在增值税申报表附表二"进项税额转出额"栏填报。

## 第四步：风险与注意事项

- **非正常损失的严格定义**：仅限管理不善造成的被盗/丢失/霉烂变质及违法被没收。自然灾害损失不需转出
- **混用资产重新转回**：原已转出进项的资产又转为可抵扣用途的（如不动产从免税项目转为应税项目），允许按净值比例重新抵扣进项
- **常见遗漏**：外购礼品赠送客户（视同销售可抵扣，但赠送员工属集体福利需转出）；报废产品如属管理不善需转出
- **稽查重点**：进销比对中发现进项长期大于销项但无留抵退税申请的企业，常被检查是否存在应转出未转出
