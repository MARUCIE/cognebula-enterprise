# 场景七：特许权使用费 vs 服务费区分

**政策依据**：《企业所得税法实施条例》第20条；OECD范本第12条注释；国家税务总局公告2018年第11号（关于税收协定执行若干问题的公告）

## 第一步：初步判别——付费性质是什么

1. **特许权使用费**：使用或有权使用版权、专利、商标、设计模型、秘密配方/工艺、工业/商业/科学设备而支付的款项
2. **服务费/技术服务费**：为完成特定服务或提供技术支持而支付的报酬
3. **混合付费**：一份合同中同时包含技术许可和技术服务

**核心区别**：是"使用权的转让"还是"人力/技术的投入"？

## 第二步：深入分析——分类判定标准

| 判定要素 | 特许权使用费 | 技术服务费 |
|---------|------------|----------|
| 付费对象 | 技术/知识产权的使用权 | 人员/团队的服务 |
| 技术转移 | 技术方案/配方/源代码交付给使用方 | 服务方自行运用技术完成任务 |
| 使用方能力 | 使用方获得技术后可自行实施 | 使用方仍依赖服务方 |
| 排他性 | 通常有排他/独占使用权 | 同一技术可同时服务多个客户 |
| 成果归属 | 技术成果由使用方所有/控制 | 服务成果交付即完成 |

**税务后果差异**：
- 特许权使用费 → 预提所得税10%（或协定优惠税率） + 增值税6%
- 技术服务费 → 通常无预提所得税（除非构成PE） + 增值税6%
- **差异可达10%以上的税负**

## 第三步：佐证/验证——高风险灰色地带

**国家税务总局公告2018年第11号**明确的重点场景：

1. **软件许可 vs SaaS服务**：
   - 买断/授权使用软件 → 特许权使用费
   - 在线订阅、软件始终在服务方运行 → 服务费

2. **技术支持/咨询**：
   - 提供技术配方+后续技术支持 → 可能拆分为特许权使用费+服务费
   - 纯技术问题解答 → 服务费

3. **设备租赁 vs 技术许可**：
   - 仅出租设备使用权 → 租金（另一条协定条款）
   - 随设备附带技术许可 → 可能包含特许权使用费成分

4. **培训费**：
   - 技术转让合同配套培训 → 特许权使用费的一部分
   - 独立的技术培训服务 → 服务费

**举证要求**：
- 合同条款中关于知识产权/技术交付的描述
- 技术文档交付清单
- 使用方是否有独立实施能力的证据

## 第四步：结论/建议

- **合同拆分**：如同时涉及技术许可和技术服务，在合同中分别约定金额
- **合理定价**：拆分后各部分定价须符合独立交易原则
- **文档准备**：保留技术交付物清单、服务工作记录，作为定性依据
- **SaaS优化**：纯SaaS模式（服务方运行、用户通过网络访问）更有利于认定为服务费
- **协定核查**：部分协定对特许权使用费的定义范围不同，需逐条核对
