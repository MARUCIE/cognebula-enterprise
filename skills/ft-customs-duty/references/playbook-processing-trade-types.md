# 场景：进料加工vs来料加工

**政策依据**：《海关加工贸易货物监管办法》（海关总署令第219号）

## 第一步：区分两种加工贸易模式

**进料加工**：企业自行进口料件（自有外汇购买），加工成成品后出口。料件所有权属于加工企业。企业承担市场风险（定价权、库存风险、坏账风险）。

**来料加工**：由外商提供全部或部分原料/零部件/元器件/包装物料，委托国内企业加工装配成成品后交外商销售。料件和成品所有权均属于外商。国内企业仅收取加工费，不承担市场风险。

核心区别总结：

| 维度 | 进料加工 | 来料加工 |
|------|---------|---------|
| 料件所有权 | 国内企业 | 外商 |
| 成品定价权 | 国内企业 | 外商 |
| 风险承担 | 国内企业 | 外商 |
| 利润模式 | 销售利润（差价） | 加工费 |
| 税务处理 | 免抵退税 | 免税不退税 |

## 第二步：税务处理差异

**进料加工——免抵退税**：
- 进口料件：保税进口（免征关税和进口增值税）
- 出口成品：免征增值税
- 国内采购环节增值税：可计算免抵退税额（退税或抵减内销税额）
- 计算公式中需扣除"免税购进原材料"（保税料件金额按计划分配率或实际分配率计算）

**来料加工——免税不退税**：
- 进口料件：免征关税和进口增值税
- 出口成品：免征增值税
- 国内采购的辅料/水电等：进项税额不得抵扣（因为出口免税而非零税率）
- 收取的加工费：免征增值税

## 第三步：海关监管要求

两种模式均需办理加工贸易手册（电子账册或纸质手册）。海关监管重点：
- **单耗管理**：每种成品消耗的料件数量（单耗）需在手册中备案，海关核销时按单耗核算
- **期限管理**：手册有效期内必须完成加工出口，通常12个月（可申请延期）
- **核销管理**：手册到期后需向海关办理核销，进口料件数量 × 单耗 = 应出口成品量

内销处理：加工贸易产品如需内销，须经海关审批并补缴进口环节税。进料加工按料件补税，来料加工按成品价格补税。

## 第四步：模式选择建议

选择进料加工的场景：企业有自主品牌/客户渠道、能承担市场风险、利润空间较大、希望享受出口退税。

选择来料加工的场景：企业以加工能力为主、无自主品牌/客户、不愿承担市场风险、加工环节附加值低。

趋势：进料加工是主流（占加工贸易总量80%+），来料加工逐年萎缩。政策导向鼓励从来料加工转向进料加工（提升产业链位置）。

关键输出：加工贸易模式比较分析 + 税务处理差异计算 + 手册管理要点 + 模式选择建议。
