# 场景：进口消费税计算

**政策依据**：《中华人民共和国消费税暂行条例》第七条、第九条；海关总署相关公告

## 第一步：判断是否属于消费税应税商品

消费税为特定消费品税种，进口以下商品需缴纳消费税：烟（卷烟/雪茄）、酒（白酒/啤酒/其他酒）、高档化妆品、贵重首饰及珠宝玉石、鞭炮焰火、成品油、摩托车（250ml以上）、小汽车、高尔夫球及球具、高档手表（10,000元以上）、游艇、木制一次性筷子、实木地板、电池、涂料。

非上述列举商品不征消费税。大部分进口商品仅涉及关税和增值税，消费税涉及面较窄但税额通常较高。

## 第二步：确定消费税计税方式

消费税有三种计税方式：
- **从价定率**：组成计税价格 × 比例税率（大部分应税品）
- **从量定额**：数量 × 单位税额（啤酒、黄酒、成品油）
- **复合计征**：从价+从量同时征收（卷烟、白酒）

从价定率的组成计税价格 = (关税完税价格 + 关税) ÷ (1 - 消费税比例税率)

分母中扣除消费税率的原因：消费税属于价内税（含在价格中），需要还原为含税价格。

## 第三步：计算实例

**示例一：进口小汽车**（排量2.5L，消费税率9%，关税率15%，增值税率13%）
- 完税价格：200,000元
- 关税：200,000 × 15% = 30,000
- 消费税组成计税价格：(200,000 + 30,000) ÷ (1 - 9%) = 252,747
- 消费税：252,747 × 9% = 22,747
- 增值税组成计税价格：200,000 + 30,000 + 22,747 = 252,747
- 增值税：252,747 × 13% = 32,857
- 税负合计：30,000 + 22,747 + 32,857 = 85,604（占完税价格42.8%）

**示例二：进口白酒**（复合计征，从价20%+从量0.5元/500ml）
- 完税价格：100,000元，数量2,000瓶（500ml/瓶），关税10%
- 关税：10,000
- 从量税：2,000 × 0.5 = 1,000
- 从价组成计税价格：(100,000 + 10,000 + 1,000) ÷ (1 - 20%) = 138,750
- 从价消费税：138,750 × 20% = 27,750
- 消费税合计：1,000 + 27,750 = 28,750

## 第四步：消费税的后续处理

进口消费税不可抵扣（消费税非增值税进项）。但在以下情况可以抵减：进口应税消费品用于连续生产应税消费品的，已纳进口消费税可以从应纳消费税中抵扣（如进口烟丝用于卷烟生产）。

会计处理：进口消费税计入存货成本（构成采购成本的一部分）。借：库存商品/原材料（含关税和消费税）/ 应交税费—应交增值税（进项税额）/ 贷：银行存款。

关键输出：消费税应税判定 + 计税方式确定 + 分步计算底稿 + 各税种金额汇总 + 账务处理分录。
