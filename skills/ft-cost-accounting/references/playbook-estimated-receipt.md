# 场景九：暂估入库与红冲

**政策依据**：《企业会计准则第1号——存货》第6条；国税函〔2010〕79号第5条（暂估入库的税前扣除）；财会〔2016〕22号（增值税会计处理）

## 第一步：初步判别——何时需要暂估入库

**暂估入库**：材料/商品已验收入库，但采购发票尚未到达，月末暂估入账

**触发条件**：
1. 月末存货已入库但未收到发票（很常见）
2. 发票延迟原因：供应商未及时开票、审批流程、快递延误等
3. 会计核算需按权责发生制确认存货和负债

**暂估金额**：按合同价格或最近采购价格暂估（不含增值税进项税额）

## 第二步：深入分析——暂估入库与红冲流程

**月末暂估入库**：
```
借：原材料/库存商品  100,000（不含税暂估价）
  贷：应付账款—暂估应付账款  100,000
```

**次月初红冲（冲回暂估）**：
```
借：原材料/库存商品  -100,000（红字）
  贷：应付账款—暂估应付账款  -100,000（红字）
```

**收到发票后正式入账**：
```
借：原材料/库存商品  98,000（实际不含税金额）
借：应交税费—增值税（进项税额）  12,740
  贷：应付账款—XX供应商  110,740
```

**暂估与实际的差异处理**：
- 差异金额小：直接按实际金额入账（暂估100,000 vs 实际98,000 → 差额2,000自然消化）
- 差异金额大：如暂估价与实际价差异超过5% → 需分析原因并追溯调整已转成本

## 第三步：佐证/验证——税务处理要点

**企业所得税**（国税函〔2010〕79号第5条）：
- 企业在汇算清缴期结束前未取得发票的暂估成本 → 不得税前扣除
- 汇算清缴期（次年5月31日）前取得发票 → 可追溯至成本发生年度扣除
- 汇算清缴期后取得 → 追补至成本发生年度扣除（5年内有效）

**增值税**：
- 暂估入库不含增值税进项（未取得发票不得抵扣）
- 取得发票后正式认证抵扣

**常见风险**：
| 风险 | 后果 |
|-----|-----|
| 长期挂暂估不取得发票 | 税前扣除被调增 |
| 暂估金额虚高 | 多转成本少交所得税 |
| 暂估科目年末余额异常大 | 审计/税务关注 |
| 暂估未月初红冲导致重复 | 存货和成本虚增 |

## 第四步：结论/建议

- **月结纪律**：月末统一暂估，次月初统一红冲（不要跨多月不红冲）
- **发票催收**：暂估超过60天未取得发票的须启动催票流程
- **年末清理**：年末重点清理暂估余额，确保汇算清缴前发票到位
- **台账管理**：建立暂估入库台账（供应商/入库日期/暂估金额/发票到位日期）
- **系统设置**：ERP系统设置暂估自动红冲和发票匹配功能，减少手工错误
