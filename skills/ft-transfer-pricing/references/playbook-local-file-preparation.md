# 场景：本地文档准备清单

**政策依据**：国家税务总局公告2016年第42号《关于完善关联申报和同期资料管理有关事项的公告》第十五至十八条

## 第一步：确定是否需要准备本地文档

按42号公告第十三条，同时满足以下条件的企业应准备本地文档：
- 年度关联交易金额符合以下标准之一：有形资产所有权转让金额超过2亿元、金融资产转让金额超过1亿元、无形资产所有权转让金额超过1亿元、其他关联交易金额（劳务/特许权等）超过4,000万元
- 如果已签订预约定价安排（APA）且覆盖当年，可免于准备该交易的本地文档

准备时限：次年6月30日前完成。税务机关可在20日前通知企业限期提供。

## 第二步：本地文档核心内容

按42号公告第十五条，本地文档应包含以下内容：

**（一）企业概况**：组织架构、管理架构、经营策略变化、主要竞争对手

**（二）关联交易情况**：关联交易描述（类型、参与方、金额、合同条款）、关联交易的商业理由、关联方的功能和风险分析

**（三）可比性分析**：可比性因素分析（资产/功能/风险/条款/经济环境）、可比信息来源及筛选过程

**（四）转让定价方法的选用与应用**：选择方法的理由、被测试方的选择、利润水平指标的选择、可比数据和区间、被测试方利润率与可比区间的对比结论

## 第三步：支撑数据与附件准备

本地文档应附具以下支撑材料：
- 关联交易合同/协议（关键条款摘要即可）
- 功能风险分析问卷（由业务部门确认）
- 可比公司筛选截图/数据库报告
- 财务数据提取说明（ERP导出路径）
- 行业分析报告（第三方市场研究）
- 以前年度转让定价报告（如有连续性参考）

所有财务数据应与年度审计报告/企业所得税申报表可勾稽。

## 第四步：质量检查与归档

完成本地文档后执行质量检查：
- 数据一致性：文档中的关联交易金额是否与关联申报表（G114和G115系列）一致
- 方法连续性：如果往年使用同一方法，分析是否连续可比
- 结论明确性：是否明确给出"关联交易定价合理"的结论
- 更新充分性：与上年文档相比，变化内容是否充分更新

本地文档以中文编制，按税务机关要求可能需提供外文翻译件。文档应保存至关联交易发生年度的次年6月1日起10年。

关键输出：本地文档正文 + 功能风险分析 + 可比性分析 + 转让定价方法应用 + 支撑材料附件。
