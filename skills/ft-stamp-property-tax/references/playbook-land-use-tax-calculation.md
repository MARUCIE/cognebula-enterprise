# 场景：城镇土地使用税计算

**政策依据**：《中华人民共和国城镇土地使用税暂行条例》（国务院令第483号修订）

## 第一步：确定纳税义务人与计税面积

纳税义务人：在城市、县城、建制镇和工矿区范围内使用土地的单位和个人。拥有土地使用权的为纳税人；使用权不明确的实际使用人为纳税人。

计税面积：以实际占用的土地面积为计税依据。确定方式优先级：不动产权证/土地使用证记载面积 > 测绘机构实测面积 > 政府部门核定面积。

共有土地：按各自实际使用面积纳税。如果无法区分，按比例分摊。例如写字楼内企业按购买面积占整体面积比例分摊。

## 第二步：确定适用税额标准

城镇土地使用税实行分级幅度定额税率（条例第四条）：
- 大城市：1.5-30元/平方米·年
- 中等城市：1.2-24元/平方米·年
- 小城市：0.9-18元/平方米·年
- 县城、建制镇、工矿区：0.6-12元/平方米·年

具体标准由各省市在幅度内确定。例如上海一类地区30元/m²、六类地区1.5元/m²。企业需查阅当地财政/税务部门公布的分区税额标准，按照房产所在地段的等级确定适用税额。

## 第三步：计算应纳税额与减免

应纳税额 = 实际占用面积（平方米）× 适用税额标准（元/平方米·年）

法定减免（条例第六条）：
- 国家机关、军队、学校、寺庙自用土地免征
- 市政街道、广场、绿化地带等公共用地免征
- 直接用于农林牧渔的生产用地免征
- 经批准开山填海、改造废弃土地，从使用月份起免征5-10年

企业常见减免情形：物流企业大宗商品仓储设施用地减按50%（持续延期中）、城镇土地使用税困难减免（企业确有困难可申请减免，需经省级税务机关审批）。

## 第四步：纳税义务时间与申报

纳税义务发生时间：
- 出让/转让方式取得：自合同约定交付土地之次月起
- 新征用耕地：自批准征用之日起满一年开始
- 新征用非耕地：自批准征用之次月起

申报：按年计算、分期缴纳（一般半年或按季）。与房产税通常一起申报。需要在财产和行为税综合申报表中填报。地块信息变化（面积调整、用途变更、取得新地块）应及时更新税源信息。

关键输出：土地使用税计算表（地块面积×税额）+ 地段等级确认 + 减免资格判断 + 年度缴纳计划。
