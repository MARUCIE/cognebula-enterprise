# 场景：交易净利润法（TNMM）

**政策依据**：国税发〔2009〕2号第二十三条第（四）项；OECD转让定价指南第二章

## 第一步：理解TNMM原理

交易净利润法（Transactional Net Margin Method）以关联交易的营业利润率（或其他净利润指标）与可比非关联交易的相应指标进行比较。核心假设：在可比条件下，从事类似功能、承担类似风险的企业应获得相似的营业利润率。

TNMM是实务中使用最广泛的转让定价方法，因为它对产品差异的容忍度高于CUP法和再销售价格法。

常用利润水平指标（PLI）：
- 营业利润率（OM）= 营业利润 ÷ 营业收入
- 完全成本加成率（FCMU）= 营业利润 ÷ 完全成本
- Berry Ratio = 毛利 ÷ 营业费用（适用于分销/服务）

## 第二步：确定被测试方与PLI

**选择被测试方**：通常为关联交易中功能较简单、风险较低的一方（如受托加工方、有限风险分销商）。被测试方的选择应使分析更可靠，减少需要进行的可比性调整。

**选择PLI**：
- 制造型企业：完全成本加成率（FCMU）
- 分销型企业：营业利润率（OM）或Berry Ratio
- 服务型企业：完全成本加成率或营业利润率

选择依据：PLI应最不受关联交易影响的差异因素影响。

## 第三步：可比公司筛选与四分位距分析

筛选步骤（详见comparable-analysis playbook）：
1. 行业筛选 → 2. 功能风险筛选 → 3. 定量筛选 → 4. 手工剔除

取可比公司多年（通常3-5年）PLI数据，计算加权平均或中位数。编制四分位距：
- P25（下四分位）：合理区间下限
- P50（中位数）：税务调整基准点
- P75（上四分位）：合理区间上限

被测试方PLI落在P25-P75区间内，则定价合理。低于P25，税务机关可能按P50调整。

## 第四步：多年数据与经济周期考量

TNMM应使用多年数据（至少3年）以平滑经济周期波动。如果被测试方或可比公司存在亏损年度，应分析亏损原因：行业普遍低迷（可接受）vs 个别经营问题（应排除）。对于新设企业的亏损期，应关注是否属于市场开拓期的合理亏损。

新冠/贸易摩擦等特殊经济事件年度的数据，可以考虑排除或单独分析。

关键输出：被测试方功能风险分析 + PLI选择说明 + 可比公司筛选 + 四分位距结果 + 合理性结论。
