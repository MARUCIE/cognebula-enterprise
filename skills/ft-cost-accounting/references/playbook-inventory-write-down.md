# 场景八：存货跌价准备

**政策依据**：《企业会计准则第1号——存货》第15-19条；《企业所得税法》第10条（税前扣除限制）

## 第一步：初步判别——何时需要计提存货跌价准备

**计提条件**：存货的可变现净值低于其账面成本

**可变现净值** = 估计售价 - 至完工时估计将要发生的成本 - 估计的销售费用和税金

**触发信号**：
1. 市场价格持续下跌且短期内无回升迹象
2. 产品过时/技术淘汰
3. 存货积压超过保质期或正常周转周期
4. 客户取消订单导致专用存货滞销
5. 原材料价格下跌导致产成品售价低于成本

## 第二步：深入分析——计提方法与计算

**计提原则**：通常按单个存货项目计提，但在某些情况下可以按存货类别汇总

**产成品可变现净值计算**：
```
可变现净值 = 估计售价 - 估计销售费用 - 估计相关税费
```

**在产品/原材料可变现净值计算**：
```
可变现净值 = 最终产品估计售价 - 至完工估计成本 - 估计销售费用和税费
```

**案例**：
| 存货 | 账面成本 | 可变现净值 | 跌价准备 |
|-----|---------|----------|---------|
| 产品A | 100万 | 80万 | 20万 |
| 产品B | 50万 | 60万 | 0（不计提） |
| 材料C（用于产品A） | 30万 | 28万* | 2万 |

*材料C的可变现净值：产品A售价下跌导致材料C可变现净值也下降

**重要**：如产品售价正常（可变现净值>成本），即使原材料市场价低于成本，也不对该材料计提跌价

## 第三步：佐证/验证——税会差异处理

**会计处理**：
```
计提时：
  借：资产减值损失—存货跌价损失
    贷：存货跌价准备

转回时（跌价因素消失）：
  借：存货跌价准备
    贷：资产减值损失—存货跌价损失

处置已计提跌价的存货：
  借：主营业务成本
  借：存货跌价准备
    贷：库存商品
```

**税务处理**：
- **计提年度**：存货跌价准备**不得**税前扣除（《企业所得税法》第10条）→ 纳税调增
- **实际处置/报废年度**：实际发生的损失可税前扣除 → 纳税调减
- **资产损失申报**：实际报废须按国家税务总局公告2011年第25号办理资产损失税前扣除申报

## 第四步：结论/建议

- **定期测试**：至少每季度末评估存货可变现净值，年末必须全面测试
- **证据充分**：计提依据须充分（市场报价、销售记录、客户订单取消证明等）
- **不过度计提**：审计师和税务机关均关注过度计提（操纵利润嫌疑）
- **税务调整台账**：建立跌价准备的税会差异台账，逐年跟踪调增/调减
- **存货周转**：高龄存货定期清理处置，避免累积大额跌价准备
