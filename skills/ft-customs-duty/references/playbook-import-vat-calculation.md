# 场景：进口环节增值税计算

**政策依据**：《中华人民共和国增值税暂行条例》第十四条；《海关法》第五十六条

## 第一步：确定进口增值税计税依据

进口环节增值税的组成计税价格：组成计税价格 = 关税完税价格 + 关税 + 消费税（如涉及）

对于不征消费税的货物：组成计税价格 = 关税完税价格 + 关税
对于征收消费税的货物：组成计税价格 = (关税完税价格 + 关税) ÷ (1 - 消费税税率)

这意味着进口增值税的计税基础包含了关税，形成"税上税"效应。

## 第二步：计算进口增值税

进口增值税 = 组成计税价格 × 增值税税率

增值税税率：一般货物13%、农产品/图书/饲料/化肥等9%。部分跨境电商零售进口适用特殊政策（个人限值内增值税70%征收）。

计算示例：进口设备完税价格100万元，关税税率10%，增值税税率13%。
- 关税 = 100万 × 10% = 10万
- 组成计税价格 = 100万 + 10万 = 110万
- 进口增值税 = 110万 × 13% = 14.3万
- 进口环节税负合计 = 10万 + 14.3万 = 24.3万

## 第三步：进口增值税的抵扣

进口增值税可作为进项税额抵扣（增值税暂行条例第八条）。凭证：海关进口增值税专用缴款书。

抵扣流程（现行规定）：取得海关缴款书后，在增值税发票综合服务平台进行"海关缴款书信息比对"。比对相符的，可在当期或后续期间申报抵扣。比对不符的，需核查原因（常见：缴款书号码录入错误、企业名称/税号不一致）。

注意：代理进口时，缴款书上的纳税人为代理公司，实际货主（委托方）需通过代理公司转交缴款书并在自己的申报中抵扣。需要在缴款书备注栏注明委托方信息。

## 第四步：特殊情况处理

**减免税进口**：享受关税减免的设备（如鼓励类外商投资项目），进口增值税也相应减免。减免的增值税不能作为进项抵扣。

**退运/退关**：已缴纳进口增值税的货物退运出口的，可申请退还已缴税款（需在进口之日起1年内申请）。已抵扣的进项税额需做转出处理。

**跨境电商零售进口**：个人年度限值26,000元内，关税税率暂设为0，增值税/消费税按法定税率的70%征收。

关键输出：进口增值税计算表 + 海关缴款书核对 + 抵扣申报操作指引 + 减免税/退运处理方案。
