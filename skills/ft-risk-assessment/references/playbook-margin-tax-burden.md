# 场景："两率一负"税务风险分析

**政策依据**：《税收征收管理法》第三十五条（核定征收适用情形）；《企业所得税法》第四十四条（特别纳税调整）；国家税务总局关于纳税评估管理办法（试行）的通知（国税发〔2005〕43号）

**适用场景**（入口条件）：年度汇算清缴前自查、收到税务风险提示后应对、新客户首次接入税务健康评估。需要至少一个完整年度的财务数据和对应行业的税负率基准值。

**数据不足时的处理**：如缺少行业基准值，先通过 `ft-regulation-lookup` 查询当地税务局发布的行业税负率参考；如无法获取，使用全国参考值（见 `ft-business-analyst` playbook-industry-classification）并标注"数据可靠性待验证"。

## 核心指标定义

- **毛利率** = (销售收入 - 销售成本) / 销售收入 x 100%
- **利润率** = 利润总额 / 销售收入 x 100%
- **所得税税负率** = 企业所得税 / 销售收入 x 100%

## 分析步骤

### 步骤一：计算核心指标
根据用户提供的财务数据计算毛利率、利润率、所得税税负率。

### 步骤二：利润率判别

**2.1 实际利润率 > 同行利润率 (偏高)**
- 所得税税负率 < 同行 → 可能存在税收优惠、合法性需查看是否存在税收优惠政策适用
- 所得税税负率 >= 同行 → 正常

**2.2 实际利润率 < 同行利润率 (偏低)**
- 先判断实际毛利率是否异常：
  - 若实际毛利率 >= 同行毛利率 且 实际税负率 < 同行 → 销售收入正常但成本偏高，**可能费用虚大**，核查费用真实性
  - 若实际毛利率 < 同行毛利率 → 销售收入或成本可能有异常，查收入隐匿或虚增成本

### 步骤三：毛利率判别

**3.1 实际毛利率 > 同行毛利率 (偏高)**
- 所得税税负率 < 同行 → 可能存在费用虚大及其他税收优惠，需查费用异常或税收优惠、抵扣原因

**3.2 实际毛利率 < 同行毛利率 (偏低)**
- 重点查是否存在隐匿收入、虚增成本

### 步骤四：异常等级判定
- 异常偏差比率 >= 50% → 高风险
- 异常偏差比率 >= 20% → 中风险
- 未达异常标准但各指标在同行区间明显偏移 → 低风险

### 步骤五：输出综合结论
1. 毛利率 = X% (同行Y%)
2. 利润率 = X% (同行Y%)
3. 所得税税负率 = X% (同行Y%)
4. 异常等级: 高/中/低
5. 判别结论: 具体的潜在风险点
6. 建议: 整改措施或留存备查资料

## 注意事项
1. 分析前先确认数据来源和行业基准值的可靠性
2. 如缺少特定行业税负基准，可用搜索工具查询行业协会数据或咨询当地税务师事务所
3. 所有结论以实际为准，本次分析仅提供风险预判

**完成标准**（出口条件）：输出包含上述6项结论要素的风险分析报告，且每项异常指标均附有具体整改建议。高风险项须明确整改责任人和时限。

**关联技能协同**：
- 异常等级为高风险 --> `ft-compliance-auditor` 启动专项自查
- 毛利率偏低疑似隐匿收入 --> `ft-compliance-auditor` 排查私户收款（playbook-personal-account-risk）
- 费用虚大嫌疑 --> `ft-invoice-manager` 审查大额费用发票真实性
- 涉及关联交易导致利润偏低 --> `ft-transfer-pricing` 进行可比性分析（playbook-comparable-analysis）
