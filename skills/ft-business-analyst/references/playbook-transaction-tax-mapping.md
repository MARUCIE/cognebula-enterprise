# 场景：交易类型→税种映射表

**政策依据**：《增值税暂行条例》；《企业所得税法》；《印花税法》；《契税法》等

## 第一步：货物销售交易

| 交易行为 | 涉及税种 | 税率/要点 |
|----------|----------|-----------|
| 销售自产货物 | 增值税13%+所得税+印花税 | 购销合同万分之三 |
| 批发零售货物 | 增值税13%+所得税+印花税 | 同上 |
| 出口货物 | 零税率/免税+出口退税 | 退税率因商品而异 |
| 进口货物 | 进口增值税+关税+消费税（如适用） | 海关代征 |
| 以物易物 | 双方各视同销售 | 按公允价值分别计税 |

## 第二步：服务交易

| 交易行为 | 增值税税率 | 其他税种 |
|----------|-----------|----------|
| 技术开发/技术转让 | 免增值税 | 所得税（技术转让500万免征） |
| 信息技术服务 | 6% | 技术合同印花税万分之三 |
| 建筑服务 | 9% | 建筑合同印花税万分之三 |
| 运输服务 | 9% | 运输合同万分之三 |
| 租赁服务（不动产） | 9% | 房产税12%/印花税 |
| 租赁服务（动产） | 13% | 印花税 |
| 金融服务 | 6% | 印花税 |
| 餐饮/住宿 | 6% | 无特殊 |

## 第三步：资产交易

| 交易行为 | 涉及税种 | 关键要点 |
|----------|----------|----------|
| 转让不动产 | 增值税9%+土地增值税+所得税+印花税+契税（买方） | 个人满2年免增值税（普通住房） |
| 转让土地使用权 | 增值税9%+土地增值税+所得税+印花税+契税 | |
| 转让股权 | 所得税+印花税 | 不涉及增值税 |
| 转让无形资产（非土地） | 增值税6%+所得税 | 技术转让可能免征 |
| 出租不动产 | 增值税9%+房产税12%+印花税 | 个人出租住房1.5%简化 |

## 第四步：快速判定规则

**拿到合同后的5步税种识别法**：
1. **交易标的是什么**：货物→13%，服务→6%/9%，不动产→9%
2. **是否跨境**：出口/进口有特殊规定
3. **是否涉及资产转移**：不动产→土增税+契税，股权→印花税+所得税
4. **是否涉及人员**：劳务报酬→个税代扣
5. **是否涉及印花税**：对照《印花税法》税目表逐一排查

**风险提示**：一项合同可能涉及多个税种。尤其是大额合同（如EPC总包），须逐项拆分税种分析。
