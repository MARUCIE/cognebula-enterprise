# 场景：混合销售vs兼营判定

**政策依据**：《增值税暂行条例实施细则》第五条；财税〔2016〕36号第四十条

## 第一步：概念区分

**混合销售**：一项销售行为既涉及货物又涉及服务，且二者具有从属关系。
- 例：销售电梯并负责安装；销售空调并负责安装
- 从属关系判定：服务是为了完成货物销售而提供的，不可独立存在

**兼营**：纳税人经营的业务中同时存在不同税率/征收率的货物、服务，但二者相互独立、无从属关系。
- 例：商场既卖家电（13%）又提供餐饮（6%）；建材公司既卖建材也提供装修设计
- 独立性判定：各项业务可独立存在，面向不同客户或同一客户的不同需求

## 第二步：税率适用规则

**混合销售**：
- 从事货物生产、批发、零售为主的企业 → 全部按货物缴纳增值税（13%）
- 其他企业 → 全部按服务缴纳增值税（6%/9%等服务适用税率）
- "为主"判定：货物销售额占比超过50%

**兼营**：
- 应分别核算不同税率业务的销售额
- 未分别核算的，从高适用税率

**决策规则**：是否"从属"是区分混合销售与兼营的唯一标准。如果两项业务签在同一合同中、不可拆分报价，大概率是混合销售；如果可以分别签合同、独立定价，则是兼营。

## 第三步：常见场景判定表

| 业务场景 | 分类 | 税率处理 |
|----------|------|----------|
| 卖电脑+送货上门 | 混合销售 | 全额13% |
| 酒店住宿+早餐 | 混合销售 | 全额6% |
| 超市卖商品+出租场地 | 兼营 | 分别13%和9% |
| 建筑公司包工包料 | 混合销售 | 全额9%（建筑） |
| 装修公司卖材料+装修 | 视合同 | 分别签→兼营；一揽子→混合 |

## 第四步：风险评估

- **人为拆分风险**：将本属混合销售的业务人为拆分为兼营以适用低税率，被税务稽查认定为不合理的，将从高补税
- **合同设计**：兼营业务务必在合同中分别列明货物金额和服务金额，分别开票、分别核算
- **36号文特殊规定**：销售活动板房、机器设备等自产货物的同时提供建筑安装服务的，应分别核算货物和服务销售额，不按混合销售处理（财税〔2016〕36号附件2第一条第(一)项）
