# 场景一：月度合规检查模板

**政策依据**：《税收征收管理法》第25条；企业内部控制基本规范

## 第一步：初步判别——月度检查范围

月度检查聚焦于**操作层面合规**，确保当月税务处理无遗漏、无错误。

## 第二步：深入分析——检查清单

```
═══ 月度税务合规检查表 ═══
检查期间：____年____月
检查人：________  复核人：________  日期：________

一、增值税合规（一般纳税人）
  □ 1.1 销项税额与开票系统数据一致
  □ 1.2 未开票收入是否已申报
  □ 1.3 进项发票勾选确认完成
  □ 1.4 不得抵扣进项已做转出（福利/免税/简易计税用途）
  □ 1.5 红字发票开具合规（有信息表编号）
  □ 1.6 税控设备抄报税已完成
  □ 1.7 免税/不征税收入申报正确

二、个人所得税合规
  □ 2.1 全员全额申报（含离职当月人员）
  □ 2.2 工资薪金与账面数据一致
  □ 2.3 劳务报酬代扣代缴完成
  □ 2.4 专项附加扣除信息已更新
  □ 2.5 社保/公积金扣除金额正确

三、发票管理
  □ 3.1 当月收到的发票已全部入账
  □ 3.2 发票抬头/税号/金额/品目正确
  □ 3.3 无不合规发票入账（查验真伪）
  □ 3.4 发票开具时限合规（应开尽开）
  □ 3.5 作废发票处理合规

四、账务处理
  □ 4.1 收入确认时点正确（纳税义务发生时间）
  □ 4.2 成本费用归期正确
  □ 4.3 往来款变动合理
  □ 4.4 银行对账单已核对
  □ 4.5 暂估入库（如有）已记录

五、申报与缴款
  □ 5.1 各税种按期申报完成
  □ 5.2 税款已及时缴纳
  □ 5.3 申报回执已保存
  □ 5.4 无逾期申报记录
```

## 第三步：佐证/验证——异常处理

发现异常时的处理流程：
1. 记录异常项目和具体差异
2. 分析原因（操作失误/政策理解/数据错误）
3. 确定整改措施和责任人
4. 跟踪整改完成情况

## 第四步：结论/建议

- **每月固定日**：建议每月5日前完成上月合规检查
- **双人复核**：检查人和复核人不得为同一人
- **异常台账**：建立月度异常问题台账，跟踪闭环
- **归档保存**：检查表签字后归档保存（建议保存5年）
- **持续优化**：每季度评审检查清单内容，补充新发现的风险点
