# 场景：一般注销全流程

**政策依据**：《公司法》第二百二十九条至第二百四十条（2023修订）；《税务登记管理办法》第二十八条至第三十一条

## 第一步：注销前置条件确认

**启动注销的法定情形**：
1. 公司章程规定的营业期限届满或其他解散事由出现
2. 股东会决议解散（有限公司2/3以上表决权通过）
3. 因合并或分立需要解散
4. 被依法吊销营业执照、责令关闭或被撤销
5. 人民法院依法裁定解散

**前置检查**：
- 是否有未结清的税款/罚款/滞纳金
- 是否有未结案的司法案件
- 是否有未到期的合同/担保
- 是否有在审行政许可事项

## 第二步：清算程序（法定必经步骤）

**成立清算组**：股东会决议解散之日起15日内成立，由股东组成（或法院指定）。

**清算组职责**：
1. 通知债权人并公告（30日内通知已知债权人，45日报纸/公示系统公告）
2. 清理公司财产、编制资产负债表和财产清单
3. 处理与清算有关的公司未了结业务
4. 清缴所欠税款以及清算过程中产生的税款
5. 清理债权、债务
6. 处分剩余财产

**清偿顺序**：
清算费用 → 职工工资/社保/经济补偿 → 欠缴税款 → 普通债权

## 第三步：税务注销

**清算所得税**（国税函〔2009〕684号）：
- 清算期间作为独立纳税年度
- 清算所得 = 全部资产可变现价值或交易价格 - 资产计税基础 - 清算费用 - 相关税费 + 债务清偿损益
- 按25%税率缴纳企业所得税
- 提交清算企业所得税申报表

**税务注销流程**：
1. 完成当年度企业所得税汇算清缴
2. 完成清算期所得税申报
3. 缴销全部发票和税控设备
4. 申请税务注销（电子税务局或窗口）
5. 取得《清税证明》

## 第四步：工商注销及后续

**工商注销**：
1. 持清税证明到市场监管局提交注销申请
2. 提交：股东会决议+清算报告+清税证明+营业执照正副本+报纸/公示截图
3. 核准注销，收缴营业执照

**后续事项**：
- 注销银行账户
- 注销社保/公积金账户
- 缴销公章（部分城市要求上交公安局）
- 注销各类许可证/资质证书

**整体时间**：一般注销全流程3-6个月。如有税务遗留问题，可能延长至1年以上。
