# 场景五：新设企业首次申报

**政策依据**：《税务登记管理办法》；《税收征收管理法》第15条；国家税务总局公告2015年第27号

## 第一步：初步判别——新企业的申报起始期

**关键时间点**：
1. **税务登记日**：领取营业执照后30日内办理税务登记（现多为"多证合一"自动关联）
2. **税种认定日**：税务机关核定应申报的税种和申报期限
3. **首次申报期**：从税种认定的次月起开始申报

**注意**：即使没有业务收入，从税种认定后也须按期"零申报"。未申报将产生逾期记录。

## 第二步：深入分析——首次申报准备工作

**税种认定后需确认**：
| 确认事项 | 内容 |
|---------|-----|
| 纳税人类型 | 一般纳税人/小规模纳税人 |
| 申报税种 | 增值税/所得税/个税/附加税/印花税等 |
| 申报周期 | 月度/季度 |
| 所得税预缴方式 | 按实际利润/按上年均摊（新企业通常按实际利润） |
| 社保/公积金 | 开户并确认缴费基数 |

**首次申报操作清单**：
```
□ 开通电子税务局账号（法人/财务负责人/办税人实名认证）
□ 签订三方协议（银行+税务+企业，用于自动扣税）
□ 确认税种认定信息（有误须及时更正）
□ 安装开票软件（如需开具发票）
□ 首次增值税申报（即使零申报）
□ 首次个税申报（即使仅有法人1人）
□ 首次企业所得税预缴（按季度，可能首次为零）
```

## 第三步：佐证/验证——常见首次申报错误

1. **遗漏税种**：税务登记后未及时做税种认定 → 系统无申报义务但实际已产生纳税义务
2. **零申报误操作**：零申报 ≠ 不申报，须在系统中提交零申报表
3. **个税遗漏**：成立当月即使未发工资，也应做个税零申报
4. **印花税遗漏**：注册资本实缴/签订合同即产生印花税义务
5. **所得税首期**：新企业第一个季度可能不满3个月，仍须预缴申报

**新企业特殊政策**：
- 小微企业判定：新设企业当年按实际经营期判定（非全年数据）
- 高新企业优惠：新设企业须经认定后才能享受，不能追溯

## 第四步：结论/建议

- **前3个月关键**：成立后前3个月是建立规范申报习惯的窗口期
- **会计建账**：同步建立规范的会计核算体系（不要等有业务再建账）
- **发票管理**：首次领票时了解发票额度/份数限制，不够时及时申请增量
- **代理记账**：初创企业如无专职财务，务必委托代理记账公司按期申报
- **信用保护**：连续3个月未申报可能被税务列为非正常户，影响企业信用
