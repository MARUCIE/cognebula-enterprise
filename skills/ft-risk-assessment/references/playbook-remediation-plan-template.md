# 场景十：整改方案模板

**政策依据**：国税发〔2005〕43号第18条（纳税评估结果处理）；《行政处罚法》第33条（首违不罚+整改）

## 第一步：初步判别——何时需要整改方案

1. **自查发现问题**：内部税务风险评估发现的合规缺陷
2. **税务机关要求**：收到风险提示、评估通知或自查通知后
3. **稽查立案后**：配合稽查提交的主动整改方案
4. **外部审计发现**：年度审计或专项审计发现的税务问题

## 第二步：深入分析——整改方案模板

```
═══════════════════════════════════════
          税务整改方案
═══════════════════════════════════════

一、整改背景
  - 问题来源（自查/税务通知/审计发现）
  - 涉及期间
  - 问题概述

二、问题清单

  序号 | 问题描述 | 涉及税种 | 涉及金额 | 风险等级
  -----|---------|---------|---------|--------
  1    | [具体问题] | [增值税/所得税等] | [估算金额] | [高/中/低]
  2    | ...     | ...     | ...     | ...

三、原因分析
  - 每个问题的根本原因（制度缺陷/人员失误/政策理解偏差）
  - 是否为系统性问题还是偶发性问题

四、整改措施

  4.1 立即整改（高风险，7日内完成）
    - 补缴税款XX元（附计算过程）
    - 补申报XX期XX税种
    - 更正XX期纳税申报表

  4.2 近期整改（中风险，30日内完成）
    - 完善XX流程/制度
    - 培训相关岗位人员
    - 建立XX台账/核算体系

  4.3 长期改进（低风险，90日内完成）
    - 建立/完善内控制度
    - 定期自查机制
    - 系统改造/升级

五、补缴方案
  | 税种 | 期间 | 补缴税款 | 滞纳金 | 合计 |
  |-----|-----|---------|-------|-----|
  | ... | ... | ...     | ...   | ... |
  | 合计 |     |         |       |     |

六、内控完善计划
  - 发票管理制度修订
  - 收入确认流程优化
  - 成本费用审核强化
  - 税务申报复核机制

七、责任分工
  | 整改事项 | 责任人 | 完成时限 | 验收标准 |
  |---------|-------|---------|---------|

八、跟踪机制
  - 每周/月进度汇报
  - 整改完成后内部验收
  - 向税务机关提交整改完成报告（如适用）
```

## 第三步：佐证/验证——整改执行要点

1. **补税计算**：须精确计算补缴金额，包含税款+滞纳金（0.05%/天）
2. **申报更正**：通过电子税务局提交更正申报，保留更正前后对比
3. **制度修订**：涉及制度层面的问题须修订相关管理制度并正式发文
4. **培训记录**：相关人员培训须有签到表和培训内容记录
5. **效果验证**：整改后至少跟踪3个月，确认同类问题不再发生

## 第四步：结论/建议

- **态度积极**：整改方案体现的态度比方案本身更重要（税务机关看态度量裁量）
- **数据准确**：补缴金额须计算准确，误差不超过千元
- **时限明确**：每项整改措施须有明确完成时限和责任人
- **闭环管理**：整改不是提交方案就结束，须跟踪到每项措施执行完毕
- **留档备查**：整改全过程文档保存10年，作为合规态度的证明
