# 场景六：碳交易损益确认

**政策依据**：财会〔2019〕22号；《企业所得税法》第6条（收入总额）；全国碳排放权交易市场相关规则

## 第一步：初步判别——碳交易损益的触发场景

1. **出售富余配额**：企业排放量<获得配额 → 出售富余部分获利
2. **购买补足配额**：企业排放量>获得配额 → 必须购买补足，产生成本
3. **投机交易**：在碳市场低买高卖（目前主要是控排企业和少量投资机构）
4. **CCER抵消**：使用中国核证自愿减排量抵消排放

## 第二步：深入分析——损益确认规则

**出售配额的收益**：
- 免费配额出售：售价全额为收益（成本为零）
  ```
  借：银行存款  50万
    贷：营业外收入  50万
  ```
- 购买配额出售：售价-购买成本=损益
  ```
  借：银行存款  60万
    贷：碳排放权资产  40万（购买成本）
    贷：营业外收入  20万
  ```

**履约使用配额的成本**：
- 使用免费配额履约：不确认成本（成本为零）
- 使用购买的配额履约：按购买成本确认营业外支出

**CCER抵消**：
- 购买CCER用于履约抵消排放：
  ```
  借：营业外支出  XX万
    贷：银行存款  XX万
  ```
- CCER价格通常低于碳配额价格（因此有成本优势）

## 第三步：佐证/验证——税务处理要点

**企业所得税**：
| 交易类型 | 收入/扣除 | 处理 |
|---------|----------|-----|
| 出售配额收益 | 营业外收入 | 并入应纳税所得额 |
| 购买配额用于履约 | 营业外支出 | 可税前扣除（真实合理） |
| 配额作废损失 | 资产损失 | 按资产损失申报扣除 |
| CCER购买成本 | 营业外支出 | 可税前扣除 |

**增值税**：
- 全国碳市场：碳配额交易暂未明确增值税政策
- 建议：关注财政部/税务总局后续文件，目前暂不申报增值税
- 区域碳市场（如北京/上海/广东等试点）：部分按"金融商品转让"6%征收

**碳交易价格参考**（全国碳市场）：
- 2024年均价约80-100元/吨
- 趋势：逐年上升（配额收紧）

## 第四步：结论/建议

- **损益对冲**：配额出售收益与履约成本可能在同一年度内对冲
- **时点选择**：碳价上升趋势下，富余配额可延后出售获取更高收益
- **税务申报**：碳交易损益在企业所得税年度申报中单独披露
- **CCER策略**：关注CCER重启后的价格差，利用CCER抵消降低合规成本
- **会计政策**：制定企业碳资产会计政策（成本结转方法/损益分类），保持一致性
