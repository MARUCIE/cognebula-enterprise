# 场景六：跨境服务费税务处理

**政策依据**：财税〔2016〕36号附件一第12-13条（增值税境内应税行为判定）；《企业所得税法》第3条；国家税务总局公告2013年第19号

## 第一步：初步判别——跨境服务费涉及的税种

1. **增值税**：境外向境内提供服务 → 是否属于"在境内销售服务"？
2. **企业所得税（预提税）**：是否构成来源于中国的所得？是否有PE？
3. **个人所得税**：如服务提供方为个人，是否有境内个税义务？
4. **源泉扣缴义务**：支付方（境内企业）是否承担扣缴义务？

**关键区分**：服务发生地在境内 vs 境外 → 直接决定增值税是否征收

## 第二步：深入分析——增值税境内判定规则

**财税〔2016〕36号第12条**，以下情形视为在境内销售服务：
- 服务的**销售方或购买方**在境内
- 所销售的服务（不动产除外）的具体应税行为发生在境内

**第13条排除项**（不属于在境内销售服务）：
- 境外单位向境内单位销售完全在境外发生的服务
- 境外单位向境内单位出租完全在境外使用的有形动产
- 财政部和国家税务总局规定的其他情形

| 服务类型 | 增值税处理 | 说明 |
|---------|-----------|-----|
| 境外咨询（远程提供给境内） | 代扣代缴6% | 购买方在境内 |
| 境外维修（在境外完成） | 不征 | 完全在境外发生 |
| 境外培训（人员出境培训） | 不征 | 完全在境外发生 |
| 境外设计（成果在境内使用） | 代扣代缴6% | 购买方在境内 |
| 境外研发（成果在境内使用） | 免征增值税 | 财税〔2017〕29号 |

## 第三步：佐证/验证——扣缴操作流程

**增值税代扣代缴**：
1. 购买方为一般纳税人 → 代扣增值税后可凭完税凭证抵扣进项
2. 计算：代扣增值税 = 含税金额 ÷ (1+税率) × 税率
3. 附加税费：城建税7%/5%/1% + 教育费附加3% + 地方教育附加2%

**企业所得税**：
- 服务费 ≠ 特许权使用费 → 通常不构成预提税义务（除非服务方有PE）
- 服务费 = 特许权使用费 → 10%预提税（见playbook-royalty-vs-service-fee）

**对外付汇税务备案**：
- 单笔支付>5万美元等值 → 需向主管税务机关提交《服务贸易等项目对外支付税务备案表》
- 所需材料：合同/协议、发票或付款通知、完税证明（已扣缴的）

## 第四步：结论/建议

- **合同定性**：明确服务内容为"服务"还是"技术使用许可"，影响税种和税率
- **发生地文档**：保留服务在境外完成的证据（出境记录、境外工作日志、成果交付记录）
- **增值税抵扣**：一般纳税人代扣代缴后别忘记申请进项抵扣
- **免税适用**：跨境研发服务符合条件可免增值税（需提前向主管税务机关备案）
- **合同拆分**：技术支持（服务）与技术许可（特许权使用费）应在合同中合理拆分
