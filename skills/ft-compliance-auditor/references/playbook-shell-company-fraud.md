# 场景：空壳企业供应商审计程序

**政策依据**：《税收征收管理法》第六十三条；国家税务总局公告2019年第38号（异常增值税扣税凭证管理）；《企业内部控制应用指引第7号——采购业务》

> 空壳企业特征识别的完整指标清单见 `ft-invoice-manager/references/playbook-shell-company-fraud.md`。本 playbook 聚焦合规审计师视角：如何在企业内部建立供应商准入审查和定期排查机制。

## 第一步：供应商准入审查（入口控制）

新供应商首次合作前必须通过以下审查：

| 审查项 | 数据来源 | 红线标准（任一触发即拒绝） |
|--------|----------|--------------------------|
| 工商存续状态 | 国家企业信用信息公示系统 | 注销/吊销/经营异常名录 |
| 成立时间 | 工商登记 | 成立不足6个月且无行业资质 |
| 注册地址 | 实地核查或地图验证 | 虚拟地址且无实际办公场所 |
| 社保缴纳人数 | 社保公开查询或要求提供 | 0人且为生产/服务型企业 |
| 税务信用等级 | 要求供应商提供或税务公开查询 | D级纳税人 |
| 经营范围匹配度 | 营业执照 | 开票品目与经营范围完全不符 |

**决策规则**：6项审查中有1项触发红线的，禁止准入。有2项以上疑点但未触发红线的，升级至财务负责人审批并留档说明。

## 第二步：存量供应商定期排查（持续监控）

**排查频率**：每季度一次（Q1/Q4加密为每月，因征期密集）。

**排查清单**：
1. 导出本季度所有取得进项发票的供应商名单
2. 批量查验供应商当前工商状态（是否变为异常/注销）
3. 检查是否收到税务机关推送的异常凭证通知（38号公告）
4. 核对单一供应商进项金额占比（单一供应商超过总进项30%需关注）
5. 核查新增大额供应商（单笔>50万）的三流一致性（合同方=开票方=收款方）

**量化预警指标**：
- 某供应商开票金额环比增长>200% 且 合作时间<12个月 --> 高风险
- 供应商开具的发票中，服务类品目（咨询/推广/劳务）占比>80% 且 无对应服务成果 --> 高风险
- 同一法人关联的多家供应商合计开票金额>100万 --> 须穿透核查

## 第三步：异常凭证应急处理

收到税务机关异常凭证推送后，48小时内完成以下动作：

1. **确认影响范围**：涉及哪些发票、金额、已抵扣税额
2. **立即暂停付款**：对该供应商未结算款项暂停支付
3. **进项转出**：已申报抵扣的，当期做进项税额转出
4. **收集自证材料**：如交易真实，收集合同+物流单+验收单+银行付款凭证，向税务机关提出核实申请
5. **追责与整改**：调查准入环节是否存在审查疏漏，完善准入制度

**关联技能协同**：
- 发现异常发票 --> 通知 `ft-invoice-manager` 处理发票作废/红冲/进项转出
- 涉及金额重大（单笔>100万或累计>500万） --> 升级至 `ft-legal-counsel` 评估法律风险
- 涉及关联方交易 --> 通知 `ft-transfer-pricing` 检查定价公允性
- 需要资金流追踪 --> 联合 `ft-risk-assessment` 做资金流向分析

## 第四步：审计底稿与整改报告

**审计底稿应包含**：
- 供应商准入审查记录（含审查日期、审查人、结论）
- 定期排查记录（含排查日期、异常发现、处理结果）
- 异常凭证处理记录（含通知日期、影响金额、应对措施、最终结论）

**年度审计报告中应披露**：
- 本年度供应商准入拒绝率及拒绝原因分布
- 异常凭证发生次数及累计金额
- 整改完成情况及制度改进措施

**风险提示**：企业善意取得虚开发票虽不构成虚开罪，但进项税额仍须转出、成本不得税前扣除。建立有效的供应商审查机制是证明"善意取得"的关键证据。
