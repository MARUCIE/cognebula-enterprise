# 场景：发票丢失处理

**政策依据**：国家税务总局公告2020年第1号（简化发票丢失处理流程）；《增值税专用发票使用规定》

## 第一步：判断丢失发票类型

| 丢失类型 | 影响 | 处理方式 |
|----------|------|----------|
| 专票发票联+抵扣联均丢失 | 无法入账和抵扣 | 用记账联复印件+已勾选认证证明 |
| 专票抵扣联丢失 | 无法抵扣 | 用发票联认证后入账抵扣 |
| 专票发票联丢失 | 无法入账 | 用抵扣联认证后作记账凭证 |
| 普通发票丢失 | 无法入账 | 取得开票方记账联复印件+证明 |
| 空白发票丢失 | 安全风险 | 立即挂失+书面报告税务机关 |

## 第二步：处理流程（2020年简化后）

**重大变化**：自2020年1月8日起（1号公告），丢失增值税专用发票不再需要登报声明作废。

**专票丢失处理**：
1. 发现丢失后立即向主管税务机关报告
2. 填写《发票丢失/损毁报告表》
3. 联系开票方取得相应联次复印件并加盖发票专用章
4. 已认证的：凭记账联复印件+认证结果通知书作为记账和抵扣凭证
5. 未认证的：凭发票联或记账联复印件进行认证后抵扣

**普通发票丢失处理**：
1. 向主管税务机关报告
2. 联系开票方取得记账联复印件
3. 由开票方在复印件上注明"与原件一致"并加盖发票专用章
4. 以复印件作为记账凭证

## 第三步：空白发票丢失处理（最严重）

1. 发现后立即向主管税务机关报告
2. 填写《发票丢失/损毁报告表》
3. 在电子税务局进行挂失处理（发票状态变为"丢失"）
4. 税务机关核实后按规定处理

**处罚**：丢失发票可处1万元以下罚款；丢失空白发票情节严重的可处1-3万元罚款。

## 第四步：风险控制

- **发票管理制度**：指定专人保管发票，建立领用登记台账
- **电子发票优势**：全电发票和电子发票不存在物理丢失问题（可在平台重新下载），建议推动供应商开具电子发票
- **及时发现**：每月盘点发票，及时发现丢失。发现越早处理越主动
- **税前扣除**：丢失发票的费用，凭开票方记账联复印件+内部付款凭证等佐证材料，仍可税前扣除（28号公告第十四条）
- **保管期限**：已开具的发票存根联和发票登记簿应保存5年
