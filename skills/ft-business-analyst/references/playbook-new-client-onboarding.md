# 场景：新客户接入调查清单

**政策依据**：《代理记账管理办法》（财政部令第98号修订）；《税收征管法》第二十一条

## 第一步：基础信息采集

**工商信息核查**：
- 营业执照（统一社会信用代码、经营范围、注册资本、成立日期、营业期限）
- 股东结构及实缴情况
- 法定代表人身份证明
- 企业信用信息公示系统查询（经营异常/行政处罚/司法案件）
- 关联企业信息（共同股东/法人的其他公司）

**税务信息核查**：
- 纳税人资格（一般纳税人/小规模纳税人）
- 核定税种及税率
- 税务评级（A/B/C/D/M级）
- 是否享受税收优惠（高新/小微/区域优惠等）
- 发票票种及当前开票限额
- 是否存在欠税/罚款/异常发票

## 第二步：财务状况评估

**账务交接检查清单**：
| 检查项 | 关注点 |
|--------|--------|
| 近12个月报表 | 资产负债表+利润表+现金流量表 |
| 科目余额表 | 各科目期末余额是否合理 |
| 银行对账 | 未达账项/长期挂账 |
| 往来明细 | 应收应付账龄分析 |
| 固定资产台账 | 与账面是否一致 |
| 发票情况 | 进销项统计/留抵税额/发票使用情况 |

**红旗指标**（发现以下情况需警惕）：
- 银行余额与账面严重不符
- 其他应收款/其他应付款金额异常大
- 存在大量长期挂账的预收/预付款
- 成本费用占比与行业严重偏离

## 第三步：风险评级

| 风险等级 | 判定标准 | 服务策略 |
|----------|----------|----------|
| 低风险 | 经营正常/账务规范/纳税评级B以上 | 标准服务流程 |
| 中风险 | 账务有瑕疵/曾有税务异常/行业特殊 | 增加审核频次+专项辅导 |
| 高风险 | 涉及虚开/偷税前科/账务严重混乱/D级纳税人 | 评估是否接入+增加服务费+强化风控 |

## 第四步：交接与建档

- 签订代理记账委托协议（明确双方权责、服务范围、保密义务）
- 建立客户档案：基础信息+税务信息+特殊事项备忘
- 制定首月服务计划（清理历史遗留问题优先级）
- 设定关键节点提醒：申报日历+年报时间+汇算清缴+社保调基
- 首月对账：全面核对银行流水与账面记录
