# 场景：关税完税价格CIF计算

**政策依据**：《中华人民共和国海关审定进出口货物完税价格办法》（海关总署令第213号修订）

## 第一步：理解完税价格概念

进口货物的完税价格以成交价格为基础审查确定。成交价格（CIF到岸价格）= 货价 + 运费 + 保险费。关税完税价格 = CIF价格。

如果成交价格为FOB（离岸价）或CFR（成本加运费），需要调整为CIF：
- FOB → CIF：加运费和保险费
- CFR → CIF：加保险费
- CIF：直接使用

保险费计算：如果实际投保，按保险费实际金额。如果未投保或无法确定，海关按"货价加运费"总额的0.3%计算保险费。

## 第二步：应计入完税价格的费用

按审价办法第十一条，以下费用应计入完税价格：
- 由买方负担的除购货佣金以外的佣金和经纪费
- 由买方负担的包装费、容器费
- 由买方提供的物料/工具/模具/消耗材料的价值（按分摊计算）
- 与被估货物有关的特许权使用费（商标费、专利费等）
- 卖方直接或间接从买方对该货物后续转售/处置/使用中获得的收益

不计入完税价格的费用（第十二条）：购货佣金（买方代理人佣金）、进口后的安装/运输/技术指导费、进口后的利息（需单独列明且利率不超过同期合理水平）。

## 第三步：计算实例

进口一批设备，合同价格条款FOB上海 USD 500,000，海运费USD 15,000，保险费USD 1,600，汇率7.20。

完税价格（人民币）= (500,000 + 15,000 + 1,600) × 7.20 = 516,600 × 7.20 = 3,719,520元

如果合同价格为CIF但未含保险费信息：
完税价格 = (500,000 + 15,000) × (1 + 0.3%) × 7.20 = 515,000 × 1.003 × 7.20 = 3,720,636元

关税 = 完税价格 × 关税税率。例如税率10%：3,719,520 × 10% = 371,952元。

## 第四步：特殊情况与海关审价

海关有权对成交价格进行审查（审价办法第五至九条）。以下情况可能导致海关不接受申报价格：买卖双方存在关联关系且影响成交价格、存在不能确定的特殊安排、成交价格明显偏低（海关有价格数据库参考）。

海关不接受申报价格时，按以下方法依次估价：相同货物成交价格、类似货物成交价格、倒扣价格法、计算价格法、合理方法。企业应保留完整的商业单证（合同、发票、运费单、保单）以支持申报价格。

关键输出：完税价格计算表 + 费用调整明细 + CIF组成分析 + 申报价格支撑文件清单。
