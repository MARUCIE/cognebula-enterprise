# 场景八：固定资产加速折旧政策

**政策依据**：《企业所得税法》第32条；财税〔2014〕75号；财税〔2015〕106号；财税〔2018〕54号；财政部税务总局公告2023年第37号（延至2027年底）

## 第一步：初步判别——加速折旧政策层级

| 政策层级 | 适用范围 | 内容 |
|---------|---------|-----|
| 通用政策 | 所有企业 | ≤500万元设备一次性扣除（2024-2027） |
| 六大行业 | 生物/专用设备/铁路/航空/计算机/信息传输 | 所有新购设备可加速折旧 |
| 全制造业 | 制造业企业 | 所有新购设备可加速折旧 |
| 研发设备 | 所有企业 | 专门用于研发的设备可加速折旧 |
| 特定条件 | 技术进步/常年强震动高腐蚀 | 可缩短折旧年限或加速折旧 |

## 第二步：深入分析——三种加速方式

**方式一：一次性税前扣除（≤500万元）**
- 全行业适用，2024-2027年间新购设备器具
- 单位价值≤500万元一次性扣除
- 效果最直接，第一年即全额抵税

**方式二：缩短折旧年限**
- 最低不得低于法定最低年限的60%
- 例：设备法定10年 → 可缩短至6年
- 需满足技术进步、常年强震动/高腐蚀等条件

**方式三：加速折旧方法**
- 双倍余额递减法：年折旧率 = 2/预计使用年限
- 年数总和法：年折旧率 = (年限-已用年数)/年数总和

**法定最低折旧年限**：
| 资产类型 | 最低年限 | 缩短后最低 |
|---------|---------|----------|
| 房屋建筑物 | 20年 | 12年 |
| 飞机/火车/轮船/机器 | 10年 | 6年 |
| 器具工具家具 | 5年 | 3年 |
| 运输工具（除上述） | 4年 | 2.4年 |
| 电子设备 | 3年 | 1.8年 |

## 第三步：佐证/验证——选择策略与台账

**选择决策树**：
```
新购固定资产
  |
  +-- 单位价值≤500万 → 一次性扣除（默认最优）
  |     但：亏损企业/优惠期企业需评估
  |
  +-- 单位价值>500万
        +-- 属于六大行业/制造业 → 加速折旧/缩短年限
        +-- 研发专用设备 → 加速折旧/缩短年限
        +-- 其他 → 符合"技术进步"等条件 → 加速/缩短
                    不符合 → 按法定年限直线法
```

**台账要求**：
- 逐项资产记录：税务折旧 vs 会计折旧
- 年度纳税调整：税务折旧>会计折旧时调减，反之调增
- 保存期限：资产处置后再保存5年

## 第四步：结论/建议

- **500万以下标配**：2024-2027年间购置≤500万设备，默认一次性扣除
- **亏损企业谨慎**：一次性扣除可能加大亏损，超过弥补年限（5年/10年）则浪费
- **会计独立处理**：会计折旧可按正常年限，税务选择加速折旧（二者独立）
- **预缴即享**：加速折旧/一次性扣除可在季度预缴时享受，无需等到年度汇算
- **研发叠加**：研发专用设备加速折旧+研发费加计扣除100% --> 抵扣效果叠加

**风险提示**：
- 一次性扣除后资产处置时，税务上账面净值为零，处置收入全额计入应税所得
- 亏损企业选择一次性扣除可能导致亏损超过5年弥补期限而浪费扣除额度
- 加速折旧台账缺失或不完整，汇算清缴时可能被要求按法定年限重新计算
