# 场景：印花税应税合同类型

**政策依据**：《中华人民共和国印花税法》第一条、附《印花税税目税率表》

## 第一步：梳理应税合同全景图

按印花税法附表，应税合同分为以下类型：

| 合同类型 | 税率 | 纳税义务人 |
|---------|------|-----------|
| 买卖合同 | 万分之三 | 买卖双方 |
| 借款合同 | 万分之零点五 | 借贷双方 |
| 融资租赁合同 | 万分之零点五 | 出租人和承租人 |
| 租赁合同 | 千分之一 | 出租人和承租人 |
| 承揽合同 | 万分之三 | 定作方和承揽方 |
| 建设工程合同 | 万分之三 | 发包方和承包方 |
| 运输合同 | 万分之三 | 托运方和承运方 |
| 技术合同 | 万分之三 | 各方 |
| 保管合同 | 千分之一 | 寄存方和保管方 |
| 仓储合同 | 千分之一 | 存货方和保管方 |
| 财产保险合同 | 千分之一 | 投保人和保险人 |

## 第二步：常见易混淆合同判别

**不属于应税合同**（新法取消或未列入的）：
- 借用合同、赠与合同（无对价的不征印花税）
- 劳动合同/劳务合同（不在税目中）
- 咨询顾问合同（非技术合同的一般咨询）
- 代理合同、委托合同（不在列举税目中）
- 物业服务合同

**容易误判的合同**：
- 技术合同 vs 一般咨询合同：技术开发/技术转让/技术咨询/技术服务属于技术合同（应税）；管理咨询/法律咨询/财务咨询不属于（不征税）
- 买卖合同 vs 加工承揽合同：需看合同性质——定制加工属承揽，标准品购销属买卖
- 借款合同：仅限与银行及其他金融机构签订的借款合同，企业间借款合同不征印花税

## 第三步：电子合同与框架协议处理

电子合同同样缴纳印花税（印花税法第十二条明确包含电子形式的应税凭证）。通过电子商务平台签订的买卖合同、电子签章的租赁合同等，均应纳入印花税申报范围。

框架协议/年度合同处理：如果框架合同列明了金额或单价×预估数量，按合同金额缴纳。如果框架合同未列明金额，在实际结算时按结算金额缴纳。后续补充协议如增加金额，按增加部分补缴。

## 第四步：实操管理建议

建立合同台账与印花税联动机制：法务/业务部门签订合同后，同步将合同信息（类型、金额、签订日期、对方）传递给财务部门进行印花税判定和申报。

高频合同自动化：对于月度大量签订的买卖合同（如电商/贸易企业），建议按月汇总申报而非逐笔。金额核对：年末将合同台账金额与已申报印花税金额交叉核对，防止遗漏。

关键输出：合同类型与税率对照表 + 易混淆合同判定指引 + 合同台账模板 + 年度印花税核对表。
