# 场景九：优惠政策叠加规则

**政策依据**：《企业所得税法》第34条；国家税务总局公告2019年第2号；财税〔2009〕69号

## 第一步：初步判别——叠加类型分析

**税率式优惠**：降低适用税率（如高新15%、小微5%、西部15%）
**税基式优惠**：减少应纳税所得额（如加计扣除、加速折旧、免税收入）
**税额式优惠**：直接减免税额（如残疾人加计扣除的节税额、专用设备抵免）

**核心规则**：同类优惠择优，不同类优惠可叠加

## 第二步：深入分析——叠加可行性矩阵

| 优惠A | 优惠B | 是否可叠加 | 说明 |
|------|------|-----------|-----|
| 高新15%税率 | 小微5%税率 | 不可（择优） | 同为税率优惠，通常小微更优 |
| 高新15% | 西部大开发15% | 不可（择优） | 税率相同，选其一即可 |
| 高新15% | 研发加计扣除100% | **可以叠加** | 税率优惠+税基优惠 |
| 小微5% | 研发加计扣除100% | **可以叠加** | 税率优惠+税基优惠 |
| 西部15% | 研发加计扣除100% | **可以叠加** | 税率优惠+税基优惠 |
| 高新15% | 技术转让减免 | **可以叠加** | 税率优惠+税基优惠 |
| 加速折旧 | 研发加计扣除 | **可以叠加** | 两项均为税基优惠但不同维度 |
| 软件即征即退（增值税） | 高新15%（所得税） | **可以叠加** | 不同税种 |
| 专用设备抵免10% | 高新15% | **可以叠加** | 税额优惠+税率优惠 |

## 第三步：佐证/验证——最优组合测算

**案例**：某高新技术企业，西部注册，年利润500万，研发费用200万

**组合一：高新15% + 研发加计**
- 应纳税所得额 = 500 - 200（加计扣除） = 300万
- 应纳税额 = 300 × 15% = 45万

**组合二：小微5% + 研发加计**
- 应纳税所得额 = 500 - 200（加计扣除） = 300万
- 应纳税额 = 300 × 25% × 20% = 15万
- **但前提**：应纳税所得额≤300万且其他小微条件满足

**组合三：西部15% + 研发加计**
- 同组合一 = 45万（税率相同）

**最优**：如满足小微条件 → 组合二（15万）> 组合一/三（45万）

## 第四步：结论/建议

- **年初规划**：年初评估可适用的各项优惠，设计最优叠加方案
- **小微优先**：如果能满足小微条件（≤300万应纳税所得额），综合税负（5%）通常最低
- **研发加计必加**：无论适用何种税率优惠，研发加计扣除都可叠加，必须申报
- **不同税种独立**：增值税优惠和所得税优惠属于不同税种，互不影响
- **政策时效**：各项优惠到期日不同，需建立政策有效期台账，到期前评估续期或替代方案
- **审计备查**：叠加享受的各项优惠均须分别留存备查材料
