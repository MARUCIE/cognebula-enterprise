# 场景一：组织形式选择——个独 vs 有限 vs 合伙

**政策依据**：《企业所得税法》第1条、第28条；《个人所得税法》第3条、第6条；《合伙企业法》第6条；财税〔2000〕91号；财税〔2008〕65号

## 第一步：初步判别——业务性质与规模画像

1. 确认业务类型：生产制造 / 技术服务 / 咨询 / 投资控股
2. 预估年利润区间：50万以下 / 50-300万 / 300万以上
3. 股东人数与身份：自然人 / 法人 / 混合
4. 是否需要对外融资或引入投资者
5. 是否涉及资质审批（如建筑、医疗）

**判定规则**：年利润50万以下、无融资需求、单一自然人 → 优先考虑个人独资企业

## 第二步：深入分析——三种形式税负测算

| 组织形式 | 企业层面 | 股东层面 | 综合税负（100万利润） |
|---------|---------|---------|-------------------|
| 有限公司 | 25% EIT（小微5%） | 20%分红个税 | 40%（小微24%） |
| 个人独资 | 无 | 5%-35%经营所得 | 约30%（核定更低） |
| 合伙企业 | 无 | 先分后税5%-35% | 约35%（高利润段） |

**关键差异**：
- 有限公司：双重征税但可享小微优惠（应纳税所得额≤300万，税率5%）
- 个独/合伙：穿透征税，无企业层面税但最高35%
- 合伙：每个合伙人独立计税，利润分配可筹划

## 第三步：佐证/验证——非税因素交叉检验

1. **有限责任需求**：个独承担无限责任，高风险业务排除
2. **税收优惠可得性**：高新技术企业15%税率仅适用有限公司
3. **核定征收可行性**：个独核定综合税负可低至2%-3%，但2021年后趋严（财政部税务总局公告2021年第41号）
4. **利润留存需求**：有限公司可不分红延后个税
5. **转让退出路径**：有限公司股权转让成熟，合伙份额转让受限

## 第四步：结论/建议

- **年利润≤300万 + 无融资**：有限公司享小微优惠（综合税负约24%）
- **年利润≤100万 + 服务类**：个独企业（查账或核定，综合约15%-25%）
- **多人合作 + 需穿透**：合伙企业（注意合伙协议的利润分配条款）
- **需要资质/融资/上市**：必须有限公司
- **混合架构**：有限公司 + 个独（将部分服务外包至个独实现利润分流）

**风险提示**：利用个独/合伙纯粹转移利润无合理商业目的的，税务机关可依据《税收征管法》第35条核定征收或依据第36条进行关联调整。
