# 场景：社保跨省转移接续

**政策依据**：《社会保险法》第十九条、第三十二条；《城镇企业职工基本养老保险关系转移接续暂行办法》（国办发〔2009〕66号）

## 第一步：确认可转移险种

| 险种 | 是否可跨省转移 | 转移内容 |
|------|--------------|----------|
| 养老保险 | 可以 | 个人账户全额 + 统筹基金12% |
| 医疗保险 | 可以 | 个人账户余额 + 缴费年限 |
| 失业保险 | 可以 | 缴费年限转移，基金不转 |
| 工伤保险 | 不转移 | 在新参保地重新参保 |
| 生育保险 | 不转移（已并入医保） | 随医保转移 |

## 第二步：养老保险转移流程

**线上办理**（推荐）：
1. 登录国家社会保险公共服务平台（si.12333.gov.cn）或"掌上12333"APP
2. 选择"养老保险关系转移申请"
3. 填写转入地信息，提交申请
4. 系统自动联系转出地社保机构
5. 转出地15个工作日内办理转出
6. 转入地45个工作日内完成接续

**线下办理**：
1. 在原参保地社保局开具《参保缴费凭证》
2. 持凭证到新参保地社保局提交转入申请
3. 新参保地社保局向原参保地发出联系函
4. 原参保地15个工作日内办理转出并划转基金
5. 新参保地收到基金后15个工作日内完成接续

## 第三步：关键规则

**养老保险**：
- 个人账户储存额全额转移
- 统筹基金按缴费基数的12%转移（不是全部单位缴费）
- 缴费年限合并计算
- 不得退保（城镇职工养老不允许一次性退还）

**待遇领取地确定**：
- 在户籍所在地参保的 → 户籍地领取
- 非户籍地但累计缴费≥10年的 → 该地领取
- 多地累计缴费均≥10年的 → 最后一个满10年的地方领取
- 各地均不满10年的 → 汇集到户籍地领取

## 第四步：风险提示

- **缴费年限断档**：转移过程中的空档期不计入缴费年限，建议在新单位参保后立即申请转移
- **重复参保**：同一时段在两地参保的，只保留一地记录，重复部分退回个人账户
- **临近退休**：距退休不足5年的，原则上不应再转移到新参保地（避免影响待遇领取地判定）
- **公积金同步**：社保转移时不要遗忘住房公积金转移（公积金转移需单独办理，通过住房公积金异地转移平台）
