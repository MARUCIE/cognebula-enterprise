# 场景：同一控制下企业合并

**政策依据**：《企业会计准则第20号——企业合并》第五至九条；《企业会计准则第33号》第四十七条

## 第一步：判断是否构成同一控制下合并

同一控制下企业合并是指参与合并的企业在合并前后均受同一方或相同的多方最终控制，且该控制并非暂时性的。判断要素：
- 最终控制方相同（同一自然人、同一法人、同一管理层团队通过协议安排）
- 合并前控制关系已存在一定期间（通常12个月以上）
- 该控制并非为实施合并而临时安排

常见情形：集团内部重组（母公司将A子公司并入B子公司）、同一自然人控制的两家公司合并。非同一控制：从独立第三方收购。

## 第二步：账务处理——账面价值法

同一控制下合并采用"账面价值法"（而非公允价值），不确认商誉。合并方取得的被合并方资产和负债，按被合并方在最终控制方合并报表中的账面价值计量。

吸收合并分录：
- 借：被合并方各项资产（账面价值）
- 贷：被合并方各项负债（账面价值）
- 贷：股本/实收资本（发行股份面值×数量或投入资本）
- 差额调整资本公积（资本溢价）；资本公积不足冲减的，调整留存收益

合并对价与取得的净资产账面价值之差，不确认损益，全部通过权益处理。

## 第三步：合并财务报表的追溯调整

按准则第四十七条，同一控制下合并应视为合并后的主体自最终控制方开始控制时起即已存在。具体要求：
- 合并利润表：包含被合并方自当期期初至合并日的收入和费用
- 合并资产负债表：期初数追溯调整为假设合并已发生的状态
- 比较财务报表：追溯调整比较期间数据

这意味着合并当年的利润表中，被合并方"全年"的经营成果都应纳入（不仅仅是合并日之后）。

## 第四步：实操注意事项

确定"账面价值"口径：应使用被合并方在最终控制方合并报表中的账面价值，而非被合并方个别报表的账面价值（两者可能因上层合并调整而不同）。合并费用（审计费、评估费、法律费）直接计入当期损益（管理费用），不计入合并成本。被合并方在合并前分配的利润，由合并方享有的部分冲减"应收股利"或"银行存款"。

关键输出：同一控制判定分析 + 账面价值确认表 + 合并分录 + 追溯调整底稿 + 比较报表调整。
