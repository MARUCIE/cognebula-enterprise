# 场景二：混合销售业务拆分节税

**政策依据**：《增值税暂行条例实施细则》第5条；《营业税改征增值税试点实施办法》（财税〔2016〕36号）第40条；国家税务总局公告2017年第11号；国家税务总局公告2018年第42号

## 第一步：初步判别——混合销售 vs 兼营

1. **混合销售**：一项销售行为同时涉及货物和服务，且二者具有关联性（如设备销售+安装）
2. **兼营**：纳税人从事多种不同税率业务，各业务相互独立（如既卖货又提供咨询）

**判定规则**：
- 同一合同、同一客户、因果关联 → 混合销售，从主业适用税率
- 可独立成交、无因果关系 → 兼营，分别核算分别适用税率

**关键**：混合销售从主业适用单一税率（货物企业全额13%，服务企业全额6%）；兼营可分别适用低税率，但必须分别核算。

## 第二步：深入分析——拆分节税空间测算

**案例**：制造企业销售设备800万 + 安装调试200万，合计1,000万

| 方案 | 销项税 | 节税效果 |
|-----|--------|---------|
| 混合销售（全额13%） | 1,000 × 13% = 130万 | 基准 |
| 兼营拆分（设备13% + 安装9%） | 800×13% + 200×9% = 122万 | 节税8万 |
| 设立服务子公司（设备13% + 安装6%） | 800×13% + 200×6% = 116万 | 节税14万 |

**拆分前提**：
- 安装服务能独立签订合同
- 服务定价符合独立交易原则
- 分别开具发票、分别核算收入

## 第三步：佐证/验证——合规红线检查

1. **合同形式**：两份独立合同或一份合同中明确拆分货物与服务金额
2. **定价合理性**：安装费不能畸高或畸低，参照市场价格
3. **核算独立性**：会计上分别确认收入、成本
4. **发票匹配**：货物开具13%发票，服务开具对应税率发票
5. **关联交易**：如设立子公司承接，需符合《企业所得税法》第41条独立交易原则

**反面案例**：将货物价款压低、服务费畸高以适用低税率 → 税务机关可依据财税〔2016〕36号第44条核定销售额

## 第四步：结论/建议

- **短期方案**：在合同中明确拆分货物与服务金额，按兼营分别核算
- **中期方案**：设立独立服务公司，承接安装/维保/培训等低税率业务
- **关键指标**：服务金额占比超过20%时拆分效果显著
- **底线原则**：拆分必须有真实业务支撑和合理商业目的，纯粹为降税的人为拆分有被纳税调整的风险
