# 场景：按税种检索法规

**政策依据**：中国现行税种体系（18个税种）；国家税务总局法规库

## 第一步：税种分类与核心法源

| 税种 | 核心法律 | 主要法规 |
|------|----------|----------|
| 增值税 | 《增值税暂行条例》 | 财税〔2016〕36号（营改增）/实施细则 |
| 消费税 | 《消费税暂行条例》 | 财税〔2006〕33号/实施细则 |
| 企业所得税 | 《企业所得税法》 | 实施条例/各年度优惠公告 |
| 个人所得税 | 《个人所得税法》 | 实施条例/专项附加扣除办法 |
| 城建税 | 《城市维护建设税法》 | 2021年版 |
| 印花税 | 《印花税法》 | 2022年版 |
| 契税 | 《契税法》 | 2021年版 |
| 房产税 | 《房产税暂行条例》 | 各地实施细则 |
| 土地增值税 | 《土地增值税暂行条例》 | 实施细则/清算管理办法 |
| 车船税 | 《车船税法》 | 实施条例 |
| 资源税 | 《资源税法》 | 2020年版 |
| 环境保护税 | 《环境保护税法》 | 2018年版 |

## 第二步：检索路径

**按税种检索的标准流程**：
1. 确定税种 → 定位核心法律/条例
2. 查找实施条例/实施细则 → 获取操作层面规定
3. 检索财政部/税务总局公告 → 获取最新优惠政策和具体执行口径
4. 检索地方性规定 → 获取本省/本市执行差异

**权威检索渠道**：
- 国家税务总局官网法规库：www.chinatax.gov.cn
- 财政部官网税政司栏目
- 中国法律法规信息库：flk.npc.gov.cn
- 各省税务局官网地方法规栏目

## 第三步：检索关键字模板

| 查询需求 | 推荐检索词 |
|----------|-----------|
| 某税种最新税率 | "[税种名称] 税率 最新" |
| 某税种优惠政策 | "[税种名称] 减免 优惠 [年份]" |
| 某税种申报规则 | "[税种名称] 申报 纳税义务发生时间" |
| 某税种征收管理 | "[税种名称] 征收管理 发票" |
| 某税种跨区域规则 | "[税种名称] 跨区域 预缴 汇总" |

## 第四步：注意事项

- **效力层级**：法律 > 行政法规 > 部门规章 > 规范性文件 > 地方性规定
- **时效性**：注意文件是否被废止或修订（标注"全文废止""部分条款修订"的需跟踪替代文件）
- **适用范围**：部分政策有行业限制、地区限制或时间限制
- **知识图谱辅助**：在KG中按税种节点查询，可快速定位关联法规和适用条件
