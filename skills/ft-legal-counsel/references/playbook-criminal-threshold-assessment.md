# 场景五：涉税犯罪构成要件判定

**政策依据**：《刑法》第201-212条；最高人民法院 最高人民检察院《关于办理危害税收征管刑事案件适用法律若干问题的解释》（法释〔2024〕4号，2024年3月20日施行）；最高人民检察院、公安部《关于公安机关管辖的刑事案件立案追诉标准的规定（二）》（2022年修订）

## 第一步：初步判别——涉税行为是否可能触及刑事

1. **行为性质**：偷税/逃税 vs 合理避税 → 主观故意是分界线
2. **金额门槛**：是否达到刑事追诉标准
3. **比例门槛**：逃税金额占应纳税总额的比例
4. **行政前置**：逃税罪须先经税务行政处理（201条第4款特殊规定）

**刑行衔接关键**：税务机关查处 → 达到移送标准 → 移送公安机关 → 检察院批准逮捕 → 法院审判

## 第二步：深入分析——主要涉税罪名与追诉标准

| 罪名 | 刑法条文 | 追诉标准 | 法定刑 |
|-----|---------|---------|-------|
| 逃税罪 | 201条 | 逃税额≥10万且占应纳税≥10% | 3年以下+罚金；特别巨大7年以下 |
| 虚开增值税专用发票罪 | 205条 | 虚开税额≥10万或致使国家税款被骗≥5万（法释[2024]4号第11条） | 3年以下→无期，金额特别巨大 |
| 虚开发票罪（普票） | 205条之一 | 100份以上或金额≥40万 | 2年以下→7年以下 |
| 骗取出口退税罪 | 204条 | 税额≥10万 | 5年以下→无期 |
| 逃避追缴欠税罪 | 203条 | 欠税≥10万且转移/隐匿财产致无法追缴 | 3年以下→7年以下 |

## 第三步：佐证/验证——犯罪构成四要件分析

**以逃税罪为例**：

1. **客体**：国家税收征管秩序
2. **客观方面**：
   - 采取欺骗、隐瞒手段进行虚假纳税申报或不申报
   - 逃避缴纳税款数额较大（≥10万）且占应纳税额10%以上
3. **主体**：纳税人、扣缴义务人（单位犯罪：对单位判处罚金+直接责任人员）
4. **主观方面**：故意（过失不构成逃税罪）

**特殊注意**：
- "不申报"须同时具有逃税故意（不知道需要申报≠逃税）
- 会计计算错误导致少缴 → 通常认定为过失 → 不构成犯罪
- 税务代理人的过错 → 要看委托人是否明知

## 第四步：结论/建议

- **行政阶段阻断**：逃税罪有"首次免责"条款（201条第4款），经税务机关通知后补缴+罚款+滞纳金的不予追究刑责（见playbook-first-offense-exemption）
- **虚开零容忍**：虚开增值税专用发票罪无"首次免责"，任何金额虚开均有刑事风险
- **自查自纠**：发现税务问题应主动补申报补缴税，在被税务机关发现前自纠从轻处理
- **证据保存**：保留所有纳税申报的合理依据和决策过程记录（证明非故意）
- **律师介入时点**：一旦收到税务稽查通知书且涉及金额较大，立即启动刑事风险评估
