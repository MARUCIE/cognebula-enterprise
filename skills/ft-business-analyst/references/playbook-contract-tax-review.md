# 场景：合同涉税审查要点

**政策依据**：《增值税暂行条例》第十九条（纳税义务发生时间）；《印花税法》；《企业所得税法》第八条

## 第一步：合同涉税审查框架（6项必查）

| 审查项 | 关注点 | 风险 |
|--------|--------|------|
| 1.合同主体 | 签约方是否为实际交易方 | 三流不一致→虚开嫌疑 |
| 2.交易内容 | 货物/服务定性是否准确 | 影响税率适用和发票类型 |
| 3.金额与税额 | 含税/不含税是否明确 | 税负承担争议 |
| 4.发票条款 | 发票类型/税率/开具时间 | 进项抵扣风险 |
| 5.付款条款 | 付款时间/方式 | 增值税纳税义务发生时间 |
| 6.违约条款 | 违约金/赔偿金 | 价外费用→增值税 |

## 第二步：关键条款审查细则

**金额条款**：
- 必须明确是含税价还是不含税价
- 如未约定，默认为含税价（实务中常引发争议）
- 税率变动风险条款：约定"如税率调整，合同金额相应调整"

**发票条款**：
- 约定发票类型（增值税专用发票/普通发票/全电发票）
- 约定适用税率
- 约定开票时间（先开票后付款/先付款后开票/同步）
- 约定未按约定开票的违约责任

**付款条款与纳税义务时间的关系**：
| 结算方式 | 增值税纳税义务时间 |
|----------|-------------------|
| 预收款（服务） | 收到预收款当天 |
| 预收款（货物） | 发出货物当天 |
| 赊销/分期 | 合同约定收款日 |
| 委托代销 | 收到代销清单或收到代销款当天 |

## 第三步：印花税条款审查

**常见涉税合同类型及税率**：
| 合同类型 | 税率 | 备注 |
|----------|------|------|
| 买卖合同 | 万分之三 | 含购销/供应/预购/调剂等 |
| 借款合同 | 万分之零点五 | 金融机构与借款人 |
| 租赁合同 | 千分之一 | |
| 承揽合同 | 万分之三 | 加工/定制/修缮等 |
| 建设工程合同 | 万分之三 | |
| 运输合同 | 万分之三 | |
| 技术合同 | 万分之三 | 技术开发/转让/许可/咨询 |
| 产权转移书据 | 万分之五 | 股权/不动产/知识产权转让 |

## 第四步：审查输出

每份合同审查后输出涉税审查意见，包含：
- 涉及税种清单
- 税率适用是否正确
- 发票条款是否完善
- 纳税义务发生时间是否明确
- 印花税应缴税额
- 修改建议（如有）

**关联技能协同**：
- 合同涉及印花税计算 --> `ft-stamp-property-tax` 确认印花税新法税率和计税依据（playbook-stamp-tax-new-law）
- 合同付款条款影响增值税纳税义务时间 --> `ft-tax-advisor` 确认增值税处理
- 合同涉及关联方交易 --> `ft-transfer-pricing` 审查定价公允性
- 合同签约主体审查 --> `ft-compliance-auditor` 核查交易方三流一致性
- 合同发票条款设计 --> `ft-invoice-manager` 确认发票类型和开具要求
