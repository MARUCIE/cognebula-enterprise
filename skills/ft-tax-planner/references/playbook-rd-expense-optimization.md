# 场景三：研发费归集优化——最大化加计扣除

**政策依据**：《企业所得税法》第30条；财税〔2015〕119号；国家税务总局公告2017年第40号；财政部税务总局公告2023年第7号（加计扣除比例100%）

## 第一步：初步判别——是否符合加计扣除条件

1. **行业排除**：烟草制造、住宿餐饮、批发零售、房地产、娱乐、租赁和商务服务 → 不得享受
2. **活动认定**：必须具备"创造性运用科学技术新知识，或实质性改进技术、产品、工艺"特征
3. **三个排除**：日常性技术支持活动、市场调研、效率调查 → 不属于研发活动
4. **会计基础**：已建立研发费辅助账或辅助账簿

**速判**：制造业/信息技术/科学研究行业 + 有新产品/新技术/新工艺开发 → 基本符合

## 第二步：深入分析——费用归集优化策略

**八大费用项目及优化要点**：

| 费用类别 | 归集要点 | 常见遗漏 |
|---------|---------|---------|
| 人员人工 | 直接从事研发的工资奖金五险一金 | 外聘研发人员劳务费可归集 |
| 直接投入 | 材料、燃料、动力、模具、工装 | 试制品销售收入应冲减 |
| 折旧费用 | 研发专用设备折旧 | 共用设备按工时分摊可归集 |
| 无形资产摊销 | 用于研发的软件/专利摊销 | 自主研发形成的无形资产摊销可叠加 |
| 新产品设计费 | 设计、测试、验收费用 | 委托外部设计费按80%归集 |
| 其他费用 | 查新、评审、知识产权费 | 上限为其他费用总额的10% |
| 委托研发 | 委托境内按80%扣除 | 委托境外按80%×2/3扣除 |
| 装备调试 | 大型装备的装备调试与验收费 | 工业性试验验证费常被忽略 |

## 第三步：佐证/验证——归集合规检查清单

1. **研发项目立项文件**：立项报告、可行性研究、预算
2. **辅助账设置**：按项目分别归集，自主研发/委托研发/合作研发分类
3. **人员花名册**：研发人员占比（高新需≥10%）、工时记录
4. **共用资源分摊**：有合理分摊依据（工时法/面积法/头数法）
5. **负面清单排查**：行政办公费、招待费、差旅费不得全额归入"其他"

**常见风险点**：
- 研发人员同时兼管理岗但未做工时分摊
- 研发领料未做实际消耗统计，全额归集
- "其他费用"突破10%上限

## 第四步：结论/建议

- **项目立项前置**：每个研发项目年初立项编号，事后补立项举证困难
- **辅助账实时归集**：按月归集研发费用，年底集中汇总风险大
- **人员工时台账**：研发与非研发时间按月记录，兼职人员按实际工时比例分摊
- **加计比例**：2023年起所有企业统一100%加计扣除（制造业此前已是100%）
- **高新联动**：研发费归集同时满足高新认定比例要求（收入<5000万→5%，5000万-2亿→4%，>2亿→3%）
