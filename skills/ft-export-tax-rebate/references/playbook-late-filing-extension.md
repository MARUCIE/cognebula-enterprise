# 场景九：逾期申报延期审批

**政策依据**：国家税务总局公告2012年第24号第4条；国家税务总局公告2018年第16号（延长申报期限）；国家税务总局公告2022年第9号

## 第一步：初步判别——是否已超过退税申报期限

**法定申报期限**：出口之日次月起至次年4月30日前的增值税纳税申报期

**逾期情形分类**：
1. **申报期限内未申报**：仍可在下一个纳税申报期补报（次年4月30日前）
2. **超过次年4月30日**：按规定视为放弃退税，除非有正当理由可申请延期
3. **超过期限但有正当理由**：可向税务机关申请延期办理

## 第二步：深入分析——正当理由与申请条件

**可以申请延期的正当理由**：
1. 因不可抗力（自然灾害、突发公共卫生事件等）
2. 出口退税电子信息尚未传输到位（海关/外汇/税务系统间延迟）
3. 收汇未完成但有合理原因
4. 税务机关要求的补充材料尚在准备中
5. 其他经税务机关认可的正当理由

**电子信息缺失（最常见原因）**：
- 报关单电子信息未传输到退税系统
- 增值税发票认证信息未传输
- 解决路径：向出口地海关/主管税务机关查询信息传输状态

## 第三步：佐证/验证——延期申请操作

**申请流程**：
```
发现逾期风险（距截止日<30天且信息不全）
  → 向主管税务机关提交书面延期申请
    → 附正当理由说明+证明材料
      → 税务机关审核（通常15个工作日内）
        → 批准延期（明确延长期限）
          → 在延长期限内完成申报
```

**申请材料**：
1. 《出口退税延期申报申请书》
2. 逾期原因书面说明
3. 相关证明材料：
   - 不可抗力：政府公告/新闻报道
   - 信息缺失：海关查询结果/系统截图
   - 收汇问题：银行催收记录/合同条款
4. 涉及的出口货物明细清单

**时效**：应在发现逾期风险后及时（建议退税申报期截止前30日内）提出申请

## 第四步：结论/建议

- **预警机制**：建立退税申报截止日倒计时提醒（T-90/T-60/T-30天）
- **信息跟踪**：每月登录退税申报系统查询电子信息到位情况
- **提早申报**：不要等到最后期限才申报，预留缓冲时间
- **逾期补救**：一旦逾期，立即向税务机关咨询并提交延期申请
- **后果了解**：未申请延期且超过期限的 → 出口按免税处理（不退税）或视同内销征税
