# 场景：内部投资抵消

**政策依据**：《企业会计准则第33号——合并财务报表》第三十一至三十四条

## 第一步：确认长期股权投资与子公司所有者权益

内部投资抵消是合并报表的核心步骤：将母公司的"长期股权投资"与子公司"所有者权益"中母公司享有的份额相互抵消。

获取数据：母公司长期股权投资明细（账面价值、持股比例）、各子公司资产负债表中所有者权益各项目（股本/实收资本、资本公积、盈余公积、未分配利润、其他综合收益）。按准则第三十一条，在购买日需按公允价值调整子公司可辨认净资产。

## 第二步：编制抵消分录（首次合并）

假设母公司以5,000万收购子公司80%股权，购买日子公司可辨认净资产公允价值5,500万：
- 借：股本 2,000
- 借：资本公积 1,500
- 借：盈余公积 500
- 借：未分配利润 1,500
- 借：商誉 600（=5,000-5,500×80%）
- 贷：长期股权投资 5,000
- 贷：少数股东权益 1,100（=5,500×20%）

商誉 = 合并成本 - 享有子公司可辨认净资产公允价值份额。若为负数（即"负商誉"），计入当期营业外收入。

## 第三步：连续合并的调整

后续年度合并时，需要将子公司个别报表从账面价值调整为公允价值基础，再做抵消。具体步骤：
1. 公允价值调整：购买日评估增值部分（如存货、固定资产、无形资产）按后续处理调整（存货当期转销、固定资产/无形资产分期摊销）
2. 将子公司净利润调整为公允价值基础的净利润
3. 按调整后净利润计算母公司应享有的投资收益
4. 抵消分录中"未分配利润"需从购买日累计调整至当期末

连续编制时，上年度合并调整需通过"未分配利润—年初"带入本年。

## 第四步：商誉减值测试与特殊情况

每年年末对商誉进行减值测试（《企业会计准则第8号——资产减值》第二十三条），将商誉分摊至资产组，比较账面价值与可收回金额。商誉减值一经确认，后续期间不得转回。

特殊情况：分步取得控制权的，需将之前持有的股权在合并日按公允价值重新计量，差额计入投资收益。处置部分股权但不丧失控制权的，作为权益性交易处理。

关键输出：公允价值调整表 + 商誉计算底稿 + 抵消分录明细（含连续调整）+ 商誉减值测试底稿。

**风险提示**：
- 公允价值评估偏差直接影响商誉金额，审计机构重点关注评估假设的合理性
- 商誉减值测试需每年进行，不得延迟；减值一经确认不得转回
- 分步取得控制权时，之前持有的股权重新计量可能产生重大损益波动
