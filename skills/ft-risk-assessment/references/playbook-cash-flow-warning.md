# 场景六：现金流异常预警

**政策依据**：《税收征收管理法》第35条（核定征收条件）；国税发〔2005〕43号（纳税评估指标）

## 第一步：初步判别——现金流异常的税务含义

1. **经营现金流与利润严重背离**：有利润但无经营现金流入
2. **银行流水远大于申报收入**：可能存在体外循环收入
3. **大额现金交易频繁**：规避银行监控和税务追踪
4. **资金在关联方间频繁流转**：可能存在资金挪用或利润转移

**税务关注点**：金税四期加强了银行数据与税务申报数据的交叉比对

## 第二步：深入分析——关键预警指标

| 指标 | 计算公式 | 预警阈值 |
|-----|---------|---------|
| 经营现金流/净利润 | 经营活动现金流量净额÷净利润 | <0.5或>3（异常） |
| 销售收现率 | 销售商品现金÷主营业务收入 | <0.8（大量赊销或收入未入账） |
| 银行存款变动率 | (期末-期初)÷期初 | 与收入变动方向严重不一致 |
| 现金支出异常率 | 非经营性现金支出÷总支出 | >30%（资金用途可疑） |
| 大额现金占比 | 现金交易÷总交易 | >20%（现金交易占比过高） |

**异常组合**：
- 利润高+经营现金流为负+应收账款激增 → 收入可能不真实（虚构交易）
- 利润低+银行流水大+个人账户资金多 → 可能存在账外收入
- 利润高+经营现金流高+但纳税很少 → 可能享受了不恰当的优惠

## 第三步：佐证/验证——现金流分析框架

**三类现金流交叉验证**：
```
经营活动现金流 > 0：正常经营产生现金
投资活动现金流 < 0：正常扩张投入
筹资活动现金流：视阶段（扩张期>0，成熟期<0）

异常组合：
经营<0 + 投资<0 + 筹资>0 = 靠借钱维持运营（财务困难信号）
经营>0 + 投资>0 + 筹资<0 = 出售资产还债（可能隐藏经营问题）
```

**个人账户排查**：
- 法定代表人个人银行账户是否有大额经营性资金往来
- 财务人员个人账户是否有异常资金进出
- 关联方个人账户是否有与企业相关的资金流

## 第四步：结论/建议

- **月度监控**：现金流三大指标每月计算，异常立即排查
- **银行对账**：企业银行流水与账面收入/支出定期核对
- **个人账户隔离**：严禁使用个人账户收取经营性款项
- **大额现金管控**：单笔现金交易超过5万元须有合理说明和审批
- **资金流向图**：绘制资金流向图，标识关联方资金往来（预防转让定价风险）
