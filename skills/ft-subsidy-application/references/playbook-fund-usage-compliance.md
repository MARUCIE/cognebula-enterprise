# 场景：补贴资金使用合规

**政策依据**：《财政违法行为处罚处分条例》（国务院令第427号）；各项专项资金管理办法

## 第一步：明确资金使用范围

政府补贴资金通常在项目批复或资金下达文件中明确规定使用范围。常见可用范围：设备购置、材料采购、外协加工费、检测费、差旅费、专家咨询费、劳务费（部分项目限定比例）。

常见禁止用途：基本建设支出（除非项目明确允许）、偿还债务、日常经营性支出（水电物业等）、对外投资或理财、罚款/赔偿/捐赠、人员工资福利（除科研人员劳务费外）。

核心原则：专款专用，按预算执行。预算调整超过一定比例（通常单项调整超过20%或总额调整超过10%）需报主管部门批准。

## 第二步：建立专项资金管理台账

会计核算要求：
- 单独设账核算：在"银行存款"下设辅助项或单独设户管理补贴资金
- 支出与项目预算科目一一对应
- 每笔支出附原始凭证（发票、合同、验收单）
- 建立资金使用台账：日期、金额、用途、对应预算科目、审批人

内控要求：资金支出需经项目负责人和财务负责人双重审批。大额支出（如设备采购超过50万）需按政府采购程序或企业内部招标程序执行。

## 第三步：常见违规行为与后果

**常见违规行为**：
- 挪用资金用于非项目支出（最严重）
- 虚列支出套取资金（如虚开发票）
- 超范围支出（如用研发补贴买办公家具）
- 未按规定执行政府采购程序
- 资金闲置超过一年未使用（部分项目有使用进度要求）

**法律后果**（《处罚处分条例》第六条）：责令改正、追回资金、警告或通报、对主管人员给予行政处分。情节严重的，3年内不得申请各类财政资金。涉嫌犯罪的移送司法机关（挪用公款/贪污）。

## 第四步：迎检与自查清单

主管部门常规检查内容：资金到位和使用情况、项目执行进度、财务凭证抽查、资产购置核实、配套资金落实情况。

自查清单：
- 资金使用率是否达到要求（如80%以上）
- 每笔支出是否有合规发票和审批
- 是否存在资金在个人账户或非项目账户停留
- 设备购置是否有合同、发票、验收单、资产标签
- 预算执行偏差是否在允许范围内
- 配套资金是否按比例到位
- 项目成果（专利、论文、产品）是否按计划产出

关键输出：资金使用合规自查表 + 专项台账模板 + 预算执行对照 + 违规风险提示 + 迎检准备清单。
