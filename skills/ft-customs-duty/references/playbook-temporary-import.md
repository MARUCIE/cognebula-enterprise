# 场景：暂时进出口税务处理

**政策依据**：《海关法》第三十三条；《中华人民共和国海关暂时进出境货物管理办法》（海关总署令第233号）

## 第一步：理解暂时进出口制度

暂时进出境货物是指经海关批准，在一定期限内进境或出境，并在规定期限内复运出境或进境的货物。核心特点：货物不发生所有权转移，进出境仅为临时使用目的。

常见暂时进出口货物：展览会展品、国际会议设备、影视拍摄器材、体育赛事器材、工程施工机械（跨国项目临时使用）、科研仪器（临时测试）、样品（非销售用途）、专业工具（维修/安装使用后带回）。

暂时进出境期限：自进境或出境之日起不超过6个月。确需延期的，可向海关申请延期，延期最多3次，每次不超过6个月，最长不超过2年（233号令第五条）。

## 第二步：免税vs征税的区分

**完全免税**（无需缴纳关税和进口环节税）：
- 适用ATA单证册的暂时进境货物（中国已加入《关于暂准进口的伊斯坦布尔公约》）
- 在展览会、交易会、会议上展示/使用且不在境内销售的展品
- 文化/体育交流用品
- 各类科学研究/技术交流用仪器设备

**按月征收关税**（233号令第八条）：不属于完全免税范围的暂时进境货物，海关按月征收关税。月征税额 = 完税价格 × 关税税率 × 1/60（即5年折算）。暂时进境货物累计征收的关税不超过按一般进口全额计征的关税。

## 第三步：ATA单证册操作

ATA单证册是国际通用的暂时进出口通关文件，由中国国际贸易促进委员会（贸促会）签发。

办理流程：向当地贸促会提交申请 → 提供货物清单（品名、数量、价值、产地）→ 缴纳担保金（通常为货值的110%）→ 签发ATA单证册 → 持单证册在各国海关办理暂时进出口（免税免担保）。

优势：一份单证册可在所有缔约国使用，免除各国逐一办理暂时进口手续和缴纳保证金。有效期1年（展览品可延至6个月）。

注意：ATA单证册项下货物必须在有效期内复运出境。逾期未出境的，视为正式进口，海关要求补缴全部关税和进口环节税。

## 第四步：期满处置与会计处理

期满处置方式：
1. **复运出境（最常见）**：按原暂时进境申报向海关办理复运出境手续，退还保证金/终止按月征税
2. **转为正式进口**：办理正式进口手续，缴纳关税和进口环节税（已按月征收的关税可抵扣）
3. **留购/捐赠**：按正式进口办理

会计处理：暂时进口的设备不确认为企业资产（所有权未转移）。按月缴纳的关税计入当期费用（管理费用或营业成本）。保证金计入"其他应收款"，复运出境退还时冲回。如果转为正式进口，按进口资产全额确认。

关键输出：暂时进出口适用性分析 + 免税/征税判定 + ATA单证册办理指引 + 按月征税计算 + 期满处置方案。
