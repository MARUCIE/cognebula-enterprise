# 场景三：研发费加计扣除范围界定

**政策依据**：《企业所得税法》第30条；财税〔2015〕119号；国家税务总局公告2017年第40号；财政部税务总局公告2023年第7号（比例统一为100%）

## 第一步：初步判别——加计扣除的基本概念

**加计扣除**：企业实际发生的研发费用，在按规定据实扣除的基础上，再按100%加计扣除

**效果**：研发费用100万 → 税前扣除200万 → 减少应纳税所得额100万 → 节税25万（25%税率）

**负面清单行业（不得享受）**：烟草制造、住宿餐饮、批发零售、房地产、娱乐业、租赁和商务服务业

## 第二步：深入分析——可加计 vs 不可加计的界定

**可加计扣除的费用（正面清单）**：

| 费用类别 | 可加计项目 | 不可加计项目 |
|---------|----------|------------|
| 人员人工 | 研发人员工资薪金/五险一金/外聘劳务 | 福利费/工会经费（但可税前扣除） |
| 直接投入 | 材料/燃料/模具/样品购买 | 共用材料未做分摊的不可归集 |
| 折旧费用 | 研发专用/共用设备折旧 | 房屋折旧（除专门用于研发） |
| 无形资产摊销 | 用于研发的软件/专利摊销 | 与研发无关的无形资产摊销 |
| 新产品设计费 | 配方/工艺设计/测试/检验 | 常规测试/质检（非研发目的） |
| 其他费用 | 查新/评审/知识产权申请费 | 上限为可加计费用总额的10% |
| 委托研发 | 境内80%、境外80%×2/3 | 超出比例限制部分 |

## 第三步：佐证/验证——七大灰色地带

1. **研发 vs 日常改进**：
   - 可加计：为开发新产品进行的系统性研究
   - 不可加计：对现有产品的日常测试和质量检验

2. **共用人员/设备**：
   - 可加计：按合理标准（工时/面积/头数）分摊后的研发部分
   - 不可加计：未做分摊的全额归集

3. **"其他费用"10%上限**：
   - 计算基准：其他费用 ≤ 前五项费用总和 × 10% / (1-10%)
   - 超出部分仍可税前扣除，但不能加计

4. **委托研发vs合作研发**：
   - 委托研发：按合同金额80%加计（委托方）
   - 合作研发：双方按各自承担的费用分别加计

5. **政府补助冲减**：
   - 与研发相关的政府补助冲减研发费用后再加计
   - 但企业可选择作为不征税收入（不冲减但对应费用不可扣除）

6. **研发失败的费用**：
   - 可加计：研发项目失败产生的费用同样可以加计扣除
   - 关键：有完整的研发立项和费用记录

7. **试制品收入冲减**：
   - 研发过程中的试制品对外销售取得的收入，应冲减研发费用

## 第四步：结论/建议

- **辅助账必须建**：按项目设立研发费用辅助账，作为加计扣除的核心依据
- **立项先行**：研发项目须有书面立项报告，明确研发目标/内容/预算
- **分摊有据**：共用人员和设备须有合理分摊标准并保留分摊计算底稿
- **10%上限注意**："其他费用"超限是最常见的审计调整项
- **季度预缴可享**：自2023年起，加计扣除可在季度预缴时申报享受
