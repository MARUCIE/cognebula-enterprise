# 场景：坏账准备税会差异

**政策依据**：《企业所得税法》第十条第七项；国家税务总局公告2011年第25号（资产损失税前扣除管理办法）

## 第一步：核心差异对照

| 维度 | 会计处理 | 税务处理 |
|------|----------|----------|
| 计提时 | 按预期信用损失模型计提坏账准备 | 计提的坏账准备一律不得税前扣除 |
| 实际发生时 | 冲减坏账准备 | 实际损失可税前扣除（需满足条件） |
| 转回时 | 冲回坏账准备计入信用减值损失 | 无税务意义 |

**核心规则**：会计上的坏账准备（无论计提、转回还是核销）在税务上完全不认可。税法只认"实际发生的、确实无法收回的坏账损失"。

## 第二步：纳税调整方向

| 期间事项 | 纳税调整 |
|----------|----------|
| 当年计提坏账准备 | 纳税调增（计提额全额调增） |
| 当年转回坏账准备 | 纳税调减（转回额全额调减） |
| 实际核销坏账（税法认可） | 纳税调减（核销金额） |
| 已核销坏账收回 | 纳税调增（收回金额） |

**综合调整公式**：
纳税调增额 = 当年计提额 - 当年转回额 - 当年税法认可的核销额 + 当年已核销坏账收回额

简化版：纳税调增额 = 期末坏账准备余额 - 期初坏账准备余额 + 当年税法认可的核销额

## 第三步：税法认可的坏账损失条件

企业实际发生的坏账损失税前扣除须满足（25号公告）：

**留存备查资料**：
1. 应收账款相关合同/协议/收款凭证
2. 债务人破产清算的法院裁定书
3. 债务人被注销/吊销营业执照的工商信息
4. 诉讼判决书/裁定书（确认无法执行）
5. 逾期3年以上且有催收记录证明确无法收回的说明

**申报要求**：
- 在年度企业所得税汇算清缴时作为资产损失扣除
- 填报A105090《资产损失税前扣除及纳税调整明细表》
- 不再需要报税务机关审批，但资料须留存备查

## 第四步：递延所得税处理

坏账准备产生可抵扣暂时性差异，需确认递延所得税资产：

```
递延所得税资产 = 坏账准备余额 × 企业所得税税率
```

分录：
```
计提时：借：递延所得税资产  贷：所得税费用
转回时：借：所得税费用      贷：递延所得税资产
```

**风险提示**：确认递延所得税资产的前提是"未来期间很可能取得足够的应纳税所得额"。持续亏损企业不应确认。
