# 场景：红字发票开具流程

**政策依据**：《增值税专用发票使用规定》（国税发〔2006〕156号修订）；国家税务总局公告2016年第47号；全电发票红字处理流程

## 第一步：判断红字发票开具情形

| 情形 | 申请方 | 处理方式 |
|------|--------|----------|
| 销售退回 | 购方或销方 | 开具红字信息表+红字发票 |
| 开票有误（未交付） | 销方 | 销方直接申请 |
| 开票有误（已交付已抵扣） | 购方 | 购方申请，做进项转出 |
| 开票有误（已交付未抵扣） | 购方或销方 | 退回原发票+销方申请 |
| 部分退货 | 购方 | 按退货金额开具红字 |
| 折让 | 购方 | 按折让金额开具红字 |

## 第二步：纸质专票红字流程

**购方已抵扣的**：
1. 购方在增值税发票管理系统中填开《红字增值税专用发票信息表》
2. 信息表上传后经税务系统校验通过
3. 购方做进项税额转出
4. 销方根据校验通过的信息表开具红字专票

**购方未抵扣的（发票退回）**：
1. 购方退回原发票
2. 销方在系统中申请《红字信息表》
3. 校验通过后开具红字专票

## 第三步：全电发票红字流程（简化）

**全电发票红冲**：
1. 销方或购方登录电子发票服务平台
2. 发起红字发票确认单
3. 对方在平台上确认（未抵扣的销方可直接发起）
4. 确认后系统自动生成红字发票
5. 购方已抵扣的，确认后系统自动做进项转出

**全电发票优势**：无需纸质信息表，线上一键操作，双方确认即可。

## 第四步：风险与注意事项

- **时限要求**：红字发票应在业务发生后及时开具。长期不处理的退货/折让，税务机关可能质疑真实性
- **金额限制**：红字发票金额不得超过原蓝字发票金额
- **部分红冲**：允许对同一张蓝字发票开具多张红字发票（部分退货场景），但红字金额合计不得超过蓝字金额
- **增值税处理**：红字发票开具后，销方冲减销项税额，购方冲减进项税额（或做进项转出）
- **企业所得税**：退货/折让同步调整当期收入和成本
- **常见错误**：仅开具红字发票但未同步做账务调整；信息表未校验通过就开具红字发票
