# 场景七：外贸企业合规检查

**政策依据**：财税〔2012〕39号；《海关法》；《外汇管理条例》

## 第一步：初步判别——外贸企业特殊合规领域

外贸企业同时涉及**海关、税务、外汇管理**三个监管体系，合规检查须三方联动。

## 第二步：深入分析——检查清单

```
═══ 外贸企业合规检查表 ═══
检查期间：____年____月/季度
检查人：________  日期：________

一、出口退税合规
  □ 1.1 退税申报按期完成（次年4月30日前）
  □ 1.2 报关单与采购发票品名/数量匹配
  □ 1.3 退税率查询确认（商品HS编码正确）
  □ 1.4 征退差额已做进项转出/成本处理
  □ 1.5 单证备案材料齐全

二、进口关税与增值税
  □ 2.1 进口关税完税凭证已入账
  □ 2.2 进口增值税已做进项抵扣
  □ 2.3 海关缴款书稽核比对通过
  □ 2.4 保税货物台账管理合规

三、外汇合规
  □ 3.1 出口收汇按期到账
  □ 3.2 收汇与报关单金额匹配
  □ 3.3 进口付汇单据齐全
  □ 3.4 外汇收支申报（银行端）完成
  □ 3.5 无异常资金跨境流动

四、海关合规
  □ 4.1 报关数据与实际货物一致
  □ 4.2 归类/价格/原产地申报准确
  □ 4.3 加工贸易手册（如有）核销按期
  □ 4.4 减免税设备监管期内合规使用
  □ 4.5 海关AEO认证维护（如适用）

五、增值税专项检查
  □ 5.1 内销与出口分别核算
  □ 5.2 不得退税的商品按内销征税或免税处理
  □ 5.3 代理出口的增值税处理正确
  □ 5.4 进料加工的免税进口料件核算准确

六、关联交易与转让定价
  □ 6.1 进出口关联交易定价合理
  □ 6.2 海关估价与税务定价一致
  □ 6.3 特殊关系申报（海关要求）
```

## 第三步：佐证/验证——外贸高频风险

1. **报关单与发票不匹配**：品名/数量/金额三者不一致 → 退税被拒
2. **收汇逾期**：未按期收汇又无合理说明 → 不得退税
3. **海关与税务口径差**：关联交易海关审价与税务转让定价标准不同
4. **归类错误**：HS编码归类错误 → 退税率差异/关税差异

## 第四步：结论/建议

- **三方对账**：海关/税务/外汇数据每月交叉核对
- **退税时效**：建立退税申报日历，不错过申报期限
- **AEO认证**：有条件的企业申请AEO高级认证（通关便利+退税加速）
- **归类预裁定**：大额出口商品提前做海关预裁定，锁定HS编码
- **外汇管控**：建立收汇催收机制，与退税申报联动管理
