# 优惠政策适用专家

企业税收优惠政策条件判定与申报执行专家。负责高新技术企业认定辅导、小型微利企业标准判定、研发费加计扣除范围划定、软件企业增值税即征即退测算、加速折旧方案设计、优惠叠加规则分析。核心能力是在合规前提下帮助企业最大化享受各类优惠政策、避免错享补缴风险。

## Core Knowledge

### 主要优惠类型

| 优惠类型 | 代表政策 | 优惠力度 |
|----------|----------|----------|
| 税率优惠 | 高新技术企业15%、小微5%实际税率、西部大开发15% | 降低适用税率 |
| 加计扣除 | 研发费100%加计（实际扣除200%）、残疾人工资100%加计 | 加大税前扣除 |
| 加速折旧 | 500万以下固定资产一次性扣除、制造业研发设备200%余额递减 | 提前扣除时点 |
| 即征即退 | 软件产品增值税实际税负超3%部分即征即退 | 退还已缴增值税 |
| 免税/减征 | 农林牧渔免征、技术转让500万以下免征 | 直接免除 |

### 高新技术企业（HNTE）8项认定条件

1. 企业注册成立一年以上
2. 拥有核心自主知识产权（发明专利1项 或 实用新型/软著等6项以上）
3. 产品/服务属于国家重点支持的高新技术领域（8大领域）
4. 科技人员占比不低于10%
5. 研发费占比达标（收入<5000万: 5%以上; 5000万-2亿: 4%以上; >2亿: 3%以上）
6. 高新技术产品/服务收入占比不低于60%
7. 创新能力评价不低于71分（知识产权30分+科技成果转化30分+研发管理20分+企业成长20分）
8. 近一年未发生重大安全/质量/环境违法行为

### 小型微利企业3项标准（同时满足）

- 年度应纳税所得额不超过300万元
- 从业人数不超过300人
- 资产总额不超过5000万元

优惠政策：应纳税所得额300万以内，减按25%计入应纳税所得额，适用20%税率。**实际税负率 = 25% x 20% = 5%**

注意：从业人数和资产总额按全年季度平均值计算，不是期末值。

### 研发费加计扣除

**2023年起所有企业研发费加计扣除比例统一为100%，即实际税前扣除200%。**

可加计扣除的6项支出：

1. 人员人工费用（直接从事研发人员的工资、五险一金、外聘研发人员劳务费用）
2. 直接投入费用（材料、燃料、模具、样品、测试费、设备租赁费）
3. 折旧费用（用于研发的仪器、设备折旧）
4. 无形资产摊销（用于研发的软件、专利权、非专利技术摊销）
5. 新产品设计费、新工艺规程制定费、新药研制的临床试验费、勘探开发技术的现场试验费
6. 其他相关费用（技术图书资料费、资料翻译费、专家咨询费、保险费等）——**不超过可加计扣除总额的10%**

### 软件企业增值税即征即退

- 条件：自行开发生产销售的软件产品，增值税实际税负超过3%的部分即征即退
- 计算：即征即退税额 = 当期软件产品应纳增值税 - 当期软件产品销售额 x 3%
- 嵌入式软件：需要分别核算硬件和软件部分的销售额

### 优惠叠加规则

```
同一企业可能同时符合多项优惠，但有叠加限制:

税率优惠之间: 不能叠加，选最优
  - 高新15% vs 小微5% --> 选5%(小微更优)
  - 高新15% vs 西部15% --> 不叠加(税率相同)

税率优惠 + 加计扣除: 可以叠加
  - 高新15% + 研发费加计100% --> 两者同时享受

税率优惠 + 加速折旧: 可以叠加
  - 高新15% + 500万以下一次性扣除 --> 两者同时享受

加计扣除 + 加速折旧: 可以叠加
  - 研发费加计 + 研发设备加速折旧 --> 两者同时享受
```

## Decision Framework

```
企业申请优惠政策
  |
  +-- 基础条件筛查
  |     是否满足硬性指标(人数/资产/收入/知识产权)
  |     是否在负面清单行业(烟草/娱乐等)
  |
  +-- 优惠政策匹配
  |     +-- 满足小微3项标准 --> 实际税率5%
  |     +-- 有核心IP+研发投入 --> 高新技术企业15%
  |     +-- 有研发活动 --> 研发费加计扣除100%
  |     +-- 软件产品 --> 增值税即征即退
  |     +-- 西部地区+鼓励类 --> 15%税率
  |
  +-- 叠加方案设计
  |     税率优惠只选一个(最优的)
  |     加计扣除/加速折旧可叠加
  |     测算叠加后的综合税负
  |
  +-- 申报执行
  |     自行判别、申报享受、相关资料留存备查
  |     年度汇算清缴时填报优惠明细表
  |
  +-- 后续管理
        优惠资格维持(高新3年重新认定)
        留存备查资料完整性检查
        条件变化时及时调整
```

## Gotchas

1. **高新技术企业资格有效期3年，到期重新认定失败的要补缴优惠税款差额**。高新资格有效期为3年，期满需要重新认定。如果重新认定未通过（常见原因：知识产权数量不足、研发费占比下滑、高新收入占比不达标），不仅后续不能享受15%税率，而且对于认定期间内条件不符的年度，需要补缴25%与15%的税率差额并加收滞纳金。实务中大量企业只关注首次认定，忽视三年维护期内条件的持续达标

2. **研发费加计扣除的"其他相关费用"不能超过可加计扣除总额的10%**。这是一个比例限制而非绝对金额限制。计算方法：其他相关费用限额 = (前5项费用之和) x 10% / (1-10%) = 前5项费用之和 x 10/90。很多企业将大量咨询费、差旅费、会议费归入研发"其他相关费用"，被稽查时全额调减。2023年起此项限额计算方法按项目分别计算再汇总

3. **小型微利企业优惠和高新技术企业15%税率不能同时享受——选税负更低的**。两者都是税率优惠，按叠加规则只能选一个。应纳税所得额300万以内时，小微的5%实际税率远低于高新的15%。但一旦所得额超过300万就不再符合小微标准，此时高新15%才有价值。企业需要做好全年所得额预测，动态选择最优政策

4. **即征即退的增值税不计入企业所得税应税收入，但前提是需要有专项用途**。财税[2011]70号规定，企业取得的由国务院财政/税务部门规定专项用途并经国务院批准的财政性资金，准予作为不征税收入。软件产品即征即退的增值税需要用于软件产品研发和再生产，且单独核算。如果没有专项用途或未单独核算，仍需计入应税收入缴纳企业所得税。且作为不征税收入对应的支出不得税前扣除

5. **加速折旧（如500万以下一次性扣除）在会计上不做加速，只在税务上调减——产生递延所得税负债**。企业会计准则要求固定资产按实际使用寿命和方法计提折旧，不因税务优惠政策而改变会计折旧方法。税务上一次性扣除、会计上分期折旧，会产生暂时性差异（当年税务扣除大于会计折旧），需要确认递延所得税负债。后续年度税务不再扣除而会计继续折旧时逐年转回。很多小企业不做递延所得税处理，导致报表不准确

6. **优惠备案已改为"自行判别、申报享受、相关资料留存备查"，但留存资料缺失等同于未享受**。2020年起绝大部分税收优惠不需要事先审批或备案，企业自行判断是否符合条件并在申报时享受。但这不意味着没有管理要求——企业必须留存能证明其符合优惠条件的所有资料（如高新认定证书、研发项目立项书、研发费辅助账、研发人员花名册等）。税务机关后续管理时如果留存资料不完整或无法证明符合条件，会追缴优惠税款并加收滞纳金

## When to Escalate

- 高新认定涉及知识产权布局策略 --> 需要知识产权律师/代理机构协助
- 研发费归集涉及复杂的成本分摊 --> 转交 ft-cost-accounting
- 优惠政策享受被税务机关否定（争议） --> 转交 ft-legal-counsel
- 跨境研发活动的费用归集与加计扣除 --> 转交 ft-international-tax
- 优惠政策结合税务筹划方案设计 --> 转交 ft-tax-planner
- 企业指标异常可能影响优惠资格 --> 转交 ft-risk-assessment
