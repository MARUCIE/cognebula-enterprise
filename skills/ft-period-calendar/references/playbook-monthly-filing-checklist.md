# 场景一：月度申报清单

**政策依据**：《税收征收管理法》第25条；《增值税暂行条例》第23条；各税种申报期限规定

## 第一步：初步判别——月度申报涉及的税种

| 税种 | 申报期限 | 适用纳税人 |
|-----|---------|----------|
| 增值税 | 次月15日前 | 一般纳税人（月报） |
| 消费税 | 次月15日前 | 消费税纳税义务人 |
| 个人所得税（代扣代缴） | 次月15日前 | 所有扣缴义务人 |
| 城建税 | 随增值税/消费税 | 增值税/消费税纳税人 |
| 教育费附加 | 随增值税/消费税 | 增值税/消费税纳税人 |
| 地方教育附加 | 随增值税/消费税 | 增值税/消费税纳税人 |
| 印花税（按次） | 纳税义务发生时 | 有应税凭证的纳税人 |

**小规模纳税人**：增值税按季申报（非月度），但个税仍须月度申报

## 第二步：深入分析——月度申报操作日历

```
每月1日-5日：
  □ 上月财务账务结转/关账
  □ 增值税进项发票确认（勾选确认平台）
  □ 工资薪金计算+个税计算

每月6日-10日：
  □ 增值税申报表填写（主表+附表）
  □ 附加税申报表填写
  □ 个人所得税扣缴申报
  □ 消费税申报（如涉及）

每月11日-15日：
  □ 申报数据复核+提交
  □ 税款缴纳（三方协议自动扣款或手动缴纳）
  □ 申报回执保存
  □ 特殊事项处理（更正申报/延期申报）

每月16日-月末：
  □ 上月申报完成情况检查
  □ 本月税务台账更新
  □ 预计下月申报数据（大额进项/收入预判）
```

## 第三步：佐证/验证——月度申报检查清单

**增值税申报检查**：
- [ ] 销项税额与开票数据一致
- [ ] 进项税额与勾选确认数据一致
- [ ] 进项转出是否已正确计算
- [ ] 免税收入/不征税收入是否正确填列
- [ ] 附表一与附表二数据逻辑一致

**个税申报检查**：
- [ ] 员工花名册与申报人数一致
- [ ] 工资薪金总额与账面数据一致
- [ ] 专项附加扣除是否已更新
- [ ] 离职员工当月是否正确处理
- [ ] 年终奖是否选择正确计税方式

## 第四步：结论/建议

- **关账时间**：每月5日前完成上月关账，预留10天申报时间
- **自动扣款**：签订三方协议实现税款自动划缴，避免遗漏
- **申报回执**：每月申报完成后保存回执截图/PDF
- **异常处理**：申报系统异常时及时联系税务机关，保留沟通记录
- **提前预判**：月末预估下月税款，做好资金准备
