# 场景：关联方识别方法

**政策依据**：《企业所得税法实施条例》第一百零九条；《企业会计准则第36号——关联方披露》；国家税务总局公告2016年第42号

## 第一步：关联方识别框架

**税法口径**（比会计口径更宽）：

| 关联类型 | 认定标准 |
|----------|----------|
| 股权控制 | 一方直接/间接持股≥25% |
| 共同控制 | 双方被同一第三方持股≥25% |
| 人员控制 | 半数以上董事/高管由对方委派或兼任 |
| 经营依赖 | 一方特许权/技术控制另一方生产经营 |
| 交易控制 | 购销/劳务/资金达年度总额50%以上由另一方控制 |
| 人员兼任 | 双方高管互为兼职 |
| 亲属关系 | 自然人通过配偶/直系亲属/兄弟姐妹控制的企业之间 |
| 实质控制 | 其他在利益上有关联的关系 |

## 第二步：识别操作步骤

**股权穿透查询**：
1. 通过国家企业信用信息公示系统查询目标企业股东结构
2. 对每个股东进行穿透查询（层层追溯至实际控制人）
3. 绘制股权关系图，标注各层持股比例
4. 识别25%以上持股关系及共同控制关系

**人员交叉排查**：
1. 导出企业董事/监事/高管名单
2. 交叉比对所有关联企业的人员名单
3. 标记兼任/委派关系

**交易依赖分析**：
1. 统计企业前5大客户和前5大供应商
2. 计算单一客户/供应商交易额占总额的比例
3. 超过50%的重点核查是否存在控制关系

## 第三步：常见隐性关联关系

| 表面关系 | 实质可能 |
|----------|----------|
| 老板配偶名下公司 | 亲属关联 |
| 老板朋友代持的公司 | 实质控制 |
| 员工注册的个体户 | 可能构成经营依赖 |
| 同一办公地址不同公司 | 高度可疑关联 |
| 同一会计/同一银行开户行 | 辅助判断线索 |

## 第四步：管理建议

- 新客户接入时必须完成关联方识别
- 建立关联方台账并每年更新
- 关联交易发生时逐笔记录金额和定价依据
- 达到42号公告同期资料准备门槛的，提前准备本地文档
- 关联方之间的交易价格必须有可比非关联交易支撑（可比非受控价格法/再销售价格法/成本加成法等）
