# 场景三：成本异常指标检测

**政策依据**：国税发〔2005〕43号；《企业所得税法》第8条（成本费用扣除真实性）；《发票管理办法》第22条

## 第一步：初步判别——成本异常的信号

1. **毛利率异常偏低**：显著低于同行业水平
2. **成本增长与收入脱节**：收入平稳但成本大幅增长
3. **成本结构突变**：材料/人工/制造费用比例突然变化
4. **存货周转异常**：存货增加但成本也在增加（可能虚增采购）

## 第二步：深入分析——五大成本异常模式

| 异常模式 | 表现特征 | 检测方法 |
|---------|---------|---------|
| 虚增材料成本 | 采购发票金额>实际采购量 | BOM用量×产量 vs 实际领料对比 |
| 虚列人工成本 | 工资表人数>实际在职人数 | 社保缴纳人数 vs 工资表人数 |
| 成本提前列支 | 跨期费用未摊销 | 大额成本发生月份分布是否均匀 |
| 个人消费入成本 | 非经营性支出列为成本 | 检查大额费用明细的业务相关性 |
| 关联采购高价 | 向关联方采购价>市场价 | 采购单价 vs 市场询价对比 |

**成本配比异常检测公式**：
- 投入产出比 = 主要材料投入量 ÷ 产品产出量
- 异常信号：投入产出比逐期上升（材料浪费或虚增投入）
- 单位产品成本 vs 行业均值偏离>30% → 需深入分析

## 第三步：佐证/验证——成本真实性检查清单

1. **采购环节**：采购合同→发票→入库单→付款凭证四流一致
2. **领用环节**：领料单→生产工单→产出记录→成品入库逻辑闭环
3. **人工成本**：劳动合同→考勤记录→工资表→社保/个税申报一致
4. **折旧摊销**：资产台账→折旧计提→实物盘点三者匹配
5. **期末存货**：账面数→盘点数→差异说明

**高风险信号组合**：
- 毛利率低于行业20%以上 + 存货周转慢 + 应付账款增大 → 可能虚增成本
- 成本急增但产量未增 + 废品率未变 → 可能虚列成本

## 第四步：结论/建议

- **BOM对照**：制造企业按BOM标准用量核查实际领料，差异超5%须说明
- **供应商核实**：大额供应商是否真实存在（工商登记/经营状态/实地核查）
- **成本趋势**：月度成本趋势图监控，突变月份重点核查
- **关联采购**：关联方采购须有市场比价资料（转让定价文档）
- **自查频率**：每季度做一次成本异常扫描，年末全面自查
