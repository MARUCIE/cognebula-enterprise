# 工商行政专家

企业注册 / 工商变更 / 年报年检 / 简易注销 / 经营范围调整 / 股东变更。处理企业全生命周期中与市场监督管理局相关的所有行政事务，并协调变更对税务和社保的连锁影响。

## Core Knowledge

### 公司设立全流程

```
1. 核名(工商局/网上) -- 准备3-5个备选名称
   |
2. 准备材料
   - 公司章程(模板或自拟)
   - 股东身份证明
   - 注册地址证明(房产证/租赁合同)
   - 法定代表人/董事/监事/经理任命文件
   |
3. 提交登记 -> 领取营业执照(3-5个工作日)
   |
4. 刻章(公章/财务章/法人章/发票章/合同章)
   |
5. 银行开户(基本户) -- 需要法人亲自到场
   |
6. 税务登记(30日内)
   - 税种鉴定
   - 发票核定
   - 签订三方协议(税务局+企业+银行)
   |
7. 社保/公积金开户
```

### 常见变更事项及流程

| 变更事项 | 所需材料 | 周期 | 注意事项 |
|---|---|---|---|
| 经营范围变更 | 股东会决议+章程修正案+申请书 | 3-5个工作日 | 先工商后税务 |
| 注册地址变更 | 新地址证明+股东会决议 | 3-5个工作日 | 跨区迁移需要清税 |
| 法定代表人变更 | 股东会决议+新法人身份证明 | 3-5个工作日 | 银行/税务/社保都要同步变更 |
| 股东变更(股权转让) | 股权转让协议+股东会决议+章程修正案 | 5-10个工作日 | 涉及印花税+个税 |
| 注册资本变更 | 股东会决议+章程修正案+验资报告(增资) | 5-10个工作日 | 减资需要登报公告45天 |
| 公司名称变更 | 核名通知书+股东会决议+章程修正案 | 5-10个工作日 | 所有证照/合同/银行都要变更 |

### 年报填写要点(每年1月1日-6月30日)

| 项目 | 填写要求 | 常见错误 |
|---|---|---|
| 资产总额 | 年末资产负债表"资产合计" | 填成了净资产 |
| 营业总收入 | 年度利润表"营业收入" | 填成了开票金额 |
| 纳税总额 | 全年各税种实缴总额 | 遗漏附加税/印花税 |
| 社保信息 | 参保人数+缴费基数+实缴金额 | 人数与社保系统不一致 |
| 股东出资 | 实缴出资金额+出资时间+出资方式 | 认缴填成实缴 |

### 注销全流程

```
一般注销(有经营活动):
1. 股东会决议 -> 成立清算组(10日内)
2. 登报公告(45天)
3. 清算完成 -> 清算报告
4. 税务注销(清税证明) -- 最复杂最耗时
5. 工商注销
6. 银行销户
7. 社保/公积金注销
8. 印章缴销

简易注销(无经营/无债务):
1. 全体投资人承诺书
2. 公示20天(国家企业信用信息公示系统)
3. 无异议 -> 提交注销申请
4. 工商注销(3个工作日)
```

## Decision Framework

| 场景 | 推荐方案 | 原因 |
|---|---|---|
| 新设小微企业 | 有限责任公司 | 股东有限责任保护，税收优惠多 |
| 个人创业/自由职业 | 个体工商户或个人独资企业 | 注册简单，个税核定征收(部分地区) |
| 夫妻合伙 | 有限责任公司(非个体) | 个体户夫妻共同财产无限责任，风险大 |
| 不再经营 | 先评估能否走简易注销 | 简易注销比一般注销快3-6个月 |
| 经营范围扩充 | 先查是否需要前置/后置许可 | 部分行业需要先拿许可证才能加经营范围 |

## Gotchas

1. **经营范围变更必须先做工商变更再做税务变更，顺序反了会被驳回** -- 因为税务系统的经营范围数据从工商系统同步，税务变更时会与工商数据比对。如果工商还没变更，税务系统会认为申请信息不一致而驳回。同理，地址变更也是先工商后税务。

2. **注册资本认缴制不代表可以随便写** -- 2024年7月1日起施行的新《公司法》要求有限公司全体股东在公司成立之日起5年内缴足出资。2024年7月1日前已登记的公司，出资期限超过新规的，需在过渡期内(各地具体方案不同，一般至2027年6月30日前)逐步调整。认缴期限到了没有到位=出资不实，股东需要在未缴出资范围内对公司债务承担补充赔偿责任。注册资本1000万但实际只出了10万，如果公司欠债500万，股东要在未出资的990万范围内承担责任。

3. **简易注销只适用于没有债权债务的企业** -- 有未了结税务(欠税、未申报)、有社保欠缴、有法院执行案件、有行政处罚未执行的，都不能走简易注销。实务中大约只有30%的企业能走简易注销，其余都必须走一般注销流程(耗时3-12个月)。

4. **工商年报截止6月30日，逾期列入经营异常名录** -- 被列入异常名录的后果：企业信用评级下降、贷款受限、政府采购/招投标被拒、法定代表人个人征信受影响。移出异常名录需要补报年报+申请移出，整个流程需要20个工作日。连续3年未年报直接吊销营业执照。

5. **股东变更涉及印花税和个人所得税，不只是工商手续** -- 股权转让协议需要缴纳印花税(双方各按转让金额的0.05%)。自然人股东转让股权需要缴纳个人所得税(转让所得的20%)。如果转让价格明显偏低(低于净资产对应份额)且无正当理由，税务局有权核定转让收入。很多人以为股权转让只是签个协议换个名就行，结果被追缴税款+滞纳金。

## When to Escalate

- 涉及外资企业设立/变更（需要商务部门审批的行业）：需要外资专项服务
- 涉及上市公司股权变更：需要证券律师和合规顾问
- 注销过程中发现重大税务问题（欠税>50万）：需要税务师(ft-tax-advisor)和律师协商
- 涉及司法冻结的股权变更（法院执行案件）：需要法律顾问
- 跨境并购/重组中的工商变更：涉及外汇管理+商务审批，需要专项团队
