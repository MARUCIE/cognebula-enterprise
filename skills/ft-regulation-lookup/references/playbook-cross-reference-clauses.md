# 场景：条款交叉引用查询

**政策依据**：《立法法》第九十二条（法律效力层级）；《税收规范性文件制定管理办法》（国家税务总局令第41号）第五条至第七条

## 第一步：理解法规引用结构

税收法规存在大量交叉引用关系：
```
法律（人大）
  ├── 行政法规/实施条例（国务院）
  │     ├── 部门规章（财政部/税务总局）
  │     │     ├── 规范性文件/公告
  │     │     └── 操作指引/解读
  │     └── 地方性法规
  └── 司法解释（最高法/最高检）
```

**常见引用模式**：
- "根据《XX法》第X条的规定" → 引用上位法
- "按照《XX实施条例》第X条执行" → 引用同级配套法规
- "本公告所称XX，适用《XX》第X条的定义" → 术语定义引用
- "参照国税发〔[年份]〕[编号]号执行" → 引用其他相关文件

## 第二步：交叉引用查询方法

**从条款查引用**（找到一个条款后查找其引用的其他条款）：
1. 阅读目标条款全文，标记所有引用的其他文件/条款
2. 逐一查阅被引用的条款内容
3. 检查被引用条款是否仍然有效

**从条款查被引用**（查找哪些其他法规引用了特定条款）：
1. 在法规库中搜索"[目标条款的文件号]第X条"
2. 查看搜索结果中引用该条款的所有文件
3. 按时间排序获取最新的引用和解读

**案例**：查询"企业所得税法第四十一条"的交叉引用
- 被引用于：《企业所得税法实施条例》第109-123条（关联交易细则）
- 被引用于：国税发〔2009〕2号（特别纳税调整办法）
- 被引用于：国家税务总局公告2016年第42号（关联申报）
- 被引用于：国家税务总局公告2017年第6号（特别纳税调查调整）

## 第三步：KG中的引用关系建模

```
节点：法规条款
属性：文号、条款号、内容摘要、有效状态
关系类型：
  - 引用(cites) → 该条款引用的其他条款
  - 被引用(cited_by) → 引用该条款的其他法规
  - 替代(supersedes) → 该条款替代的旧条款
  - 配套(supplements) → 该条款的配套实施细则
```

## 第四步：实用技巧

- **逐层展开**：从顶层法律条款开始，逐层向下展开到实施细则→公告→操作指引
- **重点标记**：对高频引用的核心条款（如企业所得税法第8条"相关性/合理性"、增值税条例第10条"不得抵扣"）做重点标注
- **引用图谱**：为复杂场景（如关联交易）绘制条款引用关系图，一目了然看到完整的法规依据链
- **版本管理**：同一条款可能经历多次修订，须确认引用的是最新版本的条款内容
