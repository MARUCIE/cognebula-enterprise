# 场景三：先缴税后复议的操作

**政策依据**：《税收征收管理法》第88条第1款；《税务行政复议规则》第33条；《纳税担保试行办法》（国家税务总局令第11号）

## 第一步：初步判别——是否必须先缴税

1. **征税争议**（税款+滞纳金）：必须先缴清税款和滞纳金，或提供相应担保 → 方可申请复议
2. **处罚争议**（罚款）：无需先缴，可直接申请复议或提起诉讼
3. **混合决定书**：同一决定书既有补税又有罚款 → 补税部分须先缴/担保，罚款部分可直接复议

**实务困境**：补税金额往往很大，企业可能无力一次性缴清 → 纳税担保成为关键替代方案

## 第二步：深入分析——三种解决路径

**路径一：直接缴清税款和滞纳金**
- 缴清后60日内申请复议
- 复议撤销原决定后可申请退税+利息
- 适用于金额较小或企业现金流充裕的情况

**路径二：提供纳税担保**
- 担保方式：保证、抵押、质押
- 保证人条件：有担保能力的法人/自然人（国有企业/事业单位一般不得为保证人）
- 抵押物：不动产、机器设备、知识产权等
- 质押物：存款、国债、保险单、股权等
- 担保金额：覆盖应纳税款+滞纳金+实现担保权的费用

**路径三：协商分期缴纳（征管法第31条）**
- 特殊困难（不可抗力/大额亏损）→ 可申请延期缴纳≤3个月
- 但延期缴纳不等于可以提起复议（仍须全额缴清/担保后才能复议）

## 第三步：佐证/验证——担保操作流程

```
收到税务处理决定书
  → 15日内（建议）启动担保准备
    → 向税务机关提出书面担保申请
      → 税务机关审查担保方式和价值
        → 签订担保合同/出具担保书
          → 税务机关确认担保有效
            → 60日内申请行政复议
```

**注意事项**：
1. 担保申请须在缴税期限届满前提出
2. 税务机关有权拒绝不充分的担保
3. 担保期间滞纳金继续计算（至实际缴清之日）
4. 复议期间担保物不得处分
5. 败诉后税务机关可处置担保物抵缴税款

## 第四步：结论/建议

- **时间节点**：收到决定书后立即评估是缴税还是担保（切勿拖延至缴税期限届满）
- **担保优先**：金额较大时优先准备不动产抵押担保（税务机关接受度最高）
- **银行保函**：条件允许时可开具银行保函替代实物担保（成本约1%-3%/年）
- **复议期间不停止执行**：即使提起复议，原行政行为不停止执行（除非法定例外）
- **宪法性挑战**：该制度被学界长期质疑"以缴税为获得救济的前提"有违宪性，但目前仍为现行有效规定
