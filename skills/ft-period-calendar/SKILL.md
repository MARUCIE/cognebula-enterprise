# 征期日历工具

申报截止日计算/节假日延期规则/提醒生成。代理记账公司最基础也最容易出错的工具——逾期申报的滞纳金和罚款是完全可以避免的损失。

## Core Knowledge

### 标准征期规则

| 税种 | 申报频率 | 标准截止日 | 备注 |
|---|---|---|---|
| 增值税(一般纳税人) | 月度 | 次月15日 | 最常见 |
| 增值税(小规模纳税人) | 季度 | 次季首月15日 | 1/4/7/10月 |
| 企业所得税预缴 | 季度 | 次季首月15日 | 同增值税季度 |
| 企业所得税汇算清缴 | 年度 | 次年5月31日 | 硬性截止 |
| 个人所得税代扣代缴 | 月度 | 次月15日 | 工资薪金/劳务报酬 |
| 个人所得税汇算清缴 | 年度 | 次年6月30日 | 个人自行申报 |
| 城建税/教育费附加 | 随增值税 | 同增值税 | 随主税种申报 |
| 印花税 | 按次/按季 | 各地不同 | 上海按季/部分地区按次 |
| 房产税 | 按半年/按年 | 各地不同 | 一般4月和10月 |
| 城镇土地使用税 | 按半年/按年 | 各地不同 | 一般同房产税 |
| 残疾人就业保障金 | 年度 | 各地不同 | 一般8-9月 |
| 工会经费 | 月度/季度 | 各地不同 | 随个税或单独申报 |
| 社保 | 月度 | 当月或次月 | 各地扣款日不同 |
| 公积金 | 月度 | 当月或次月 | 各地扣款日不同 |

**核心规则：** 申报截止日遇到法定节假日(含调休)或公休日(周六周日)，顺延到下一个工作日。

### 提醒生成规则

- T-7：提前7天生成预警提醒(黄色)
- T-3：提前3天生成紧急提醒(橙色)
- T-1：截止日前1天最终提醒(红色)
- T+1：逾期第1天生成逾期预警(红色+处罚预估)
- 批量提醒：代账公司同时管理N家客户，按截止日排序批量提醒

## Decision Framework

| 查询类型 | 处理方式 | 输出格式 |
|---|---|---|
| "这个月什么时候截止" | 查当月所有税种截止日 | 日历视图(按日期排序) |
| "增值税什么时候报" | 根据纳税人类型返回 | 单条截止日+延期说明 |
| "下个季度要报什么" | 列出季度申报事项 | 清单(含月度+季度事项) |
| "春节征期延到什么时候" | 查具体月份延期规则 | 延期前→延期后+依据 |
| "帮我生成全年纳税日历" | 生成12个月完整日历 | 表格(含延期标注) |
| "逾期了怎么办" | 计算逾期天数+罚则 | 滞纳金金额+补救步骤 |

### 逾期后果计算

- 滞纳金(针对逾期缴纳税款)：应纳税额 x 0.05%/日 x 逾期天数(从缴纳期限届满次日起算)，征管法第32条
- 罚款(针对逾期申报)：责令限期改正，可处2000元以下罚款；情节严重的，处2000元以上10000元以下罚款(征管法第62条，上限10000元)
- 注意：逾期申报(罚款)和逾期缴税(滞纳金)是两件事，可能同时发生
- 信用影响：逾期申报纳入纳税信用评价，影响A/B/C/D级评定

## Gotchas

1. **15日是截止日不是开始日** -- 很多人以为每月15日才"开始可以申报"，实际上15日(或顺延后的日期)是最后一天。电子税务局一般每月1日就可以开始申报。建议客户月初就完成申报，不要等到截止日当天。

2. **节假日顺延只顺延工作日，不是自然日** -- "顺延"的含义是跳过法定假日和公休日，找到下一个工作日。国庆7天假后，15日在假期内，顺延到假期后第一个工作日。不是简单地加7天。

3. **汇算清缴5月31日截止是硬性的，但遇周末/节假日同样顺延** -- 2025年5月31日是周六，按标准顺延规则实际截止日为6月2日(周一)。但注意：此顺延是否适用于汇算清缴截止日，历年做法不完全一致，部分年份STA会单独发文明确。建议按5月31日前完成为安全底线，6月2日作为最后容错线。

4. **新设企业首次申报期从税务登记日开始算** -- 不是从工商注册日开始。如果1月15日注册公司，2月1日才做税务登记，那首次申报期从2月算起。很多新客户搞混注册日和税务登记日，以为自己已经逾期。

5. **跨地区经营的预缴期限和总部不同** -- 建筑企业在项目所在地预缴增值税的期限是取得预收款的次月申报期内。这个期限跟总部的正常申报期不一定一致，特别是项目所在地和总部在不同省份时，节假日安排可能不同。

6. **出口退税申报期限已放宽但仍有限制** -- 2022年起(财政部 税务总局公告2022年第16号)，出口退税申报不再受"次年4月15日"硬性截止限制，但仍需在收齐退税凭证的次月增值税申报期内申报。逾期未收齐单证的，可在收齐后申报。但超过规定时限(出口之日起次年4月增值税纳税申报期)未申报的，需提供相关举证材料。具体以当年退税管理办法为准，不能照搬旧规则。

7. **社保和税务的征期不同步** -- 社保扣款日各地差异很大(有的当月扣、有的次月扣)，而且银行托收和税务申报是两套系统。不能假设"税报完了社保也报完了"。需要分别确认。

8. **电子税务局申报通道关闭时间** -- 部分地区的电子税务局在征期截止日当天24:00自动关闭申报通道。如果当天最后1小时系统拥堵导致提交失败，技术上已经逾期。建议至少提前1天完成申报。

## When to Escalate

- 客户已经逾期超过30天：需要评估信用等级影响和罚款减免可能，交 ft-compliance-auditor
- 涉及延期申报审批(因不可抗力/特殊困难)：需要向主管税局申请，超出日历计算范围
- 客户有跨地区经营(5+项目)：预缴日历复杂度超出单一日历工具能力，需要 ft-tax-advisor 统筹
- 税局调整征期(临时通知)：需要人工确认官方公告，日历规则需要更新
