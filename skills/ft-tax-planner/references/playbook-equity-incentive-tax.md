# 场景八：股权激励税务方案

**政策依据**：财税〔2005〕35号；国税函〔2009〕461号；财税〔2016〕101号（递延纳税）；财政部税务总局公告2019年第35号；《个人所得税法》第2条

## 第一步：初步判别——激励类型与企业阶段

1. **企业类型**：上市公司 / 非上市公司 / 拟上市公司
2. **激励工具**：股票期权 / 限制性股票 / 股票增值权 / 股权奖励
3. **激励对象**：高管 / 核心技术人员 / 全员
4. **激励股权来源**：增发新股 / 大股东转让 / 回购

**关键区分**：上市与非上市的税务处理规则完全不同，必须先确认。

## 第二步：深入分析——各环节税务处理

**上市公司股票期权**：
| 环节 | 税务处理 | 税率 |
|-----|---------|-----|
| 授予日 | 不征税 | — |
| 行权日 | (行权价-授予价)×股数 → 工资薪金 | 并入综合所得3%-45% |
| 持有期分红 | 持股>1年免税；1月-1年减半；<1月全额 | 20%差别化 |
| 转让日 | 暂免征收个人所得税 | 免税（上市公司） |

**非上市公司股权激励（符合财税〔2016〕101号）**：
- 递延至转让环节按"财产转让所得"20%征税（非工资薪金45%）
- **条件**：境内居民企业、股权激励计划经董事会/股东会审议、激励标的为本公司股权、激励对象限技术骨干/管理人员且≤30%员工、持有≥3年

**税负差异测算**（激励收益100万）：
- 行权时按工资薪金（45%边际）：约38万税
- 递延至转让按财产转让（20%）：约20万税
- **节税约18万**

## 第三步：佐证/验证——递延纳税合规检查

1. **101号文七项条件**逐一核对：
   - 属于境内居民企业的股权激励计划 ✓/✗
   - 经公司董事会、股东会审议通过 ✓/✗
   - 激励标的应为境内居民企业的本公司股权 ✓/✗
   - 激励对象应为技术骨干和高级管理人员 ✓/✗
   - 激励对象人数累计不得超过本公司最近6个月在职职工平均人数的30% ✓/✗
   - 股票/期权自授予日起持有满3年 ✓/✗
   - 实施递延纳税的股票/期权转让须满足持有满12个月 ✓/✗
2. **备案要求**：递延纳税须在相关股票/期权激励计划实施的次月15日内报主管税务机关备案
3. **关联定价**：股权转让价格须公允，低价转让将面临核定

## 第四步：结论/建议

- **非上市公司首选**：满足101号文条件 → 递延纳税，20%替代45%，节税显著
- **上市公司**：行权时点选择低股价期（降低行权收益）+ 持股超1年享分红免税
- **备案时效**：激励计划实施次月15日内完成备案，逾期不得享受递延
- **持有期限**：提醒员工3年持有期+12个月锁定期的刚性要求
- **退出机制税务**：离职回购/对赌条款触发的股权变动可能产生新的纳税义务，须提前约定
