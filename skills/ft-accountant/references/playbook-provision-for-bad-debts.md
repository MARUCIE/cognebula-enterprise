# 场景：坏账准备计提

**政策依据**：《企业会计准则第22号——金融工具确认和计量》（财会〔2017〕7号）；《企业所得税法》第十条第七项

## 第一步：判别应收款项分类

按预期信用损失模型（ECL），将应收款项分为三阶段：
- **第一阶段**：信用风险自初始确认后未显著增加 → 按未来12个月预期信用损失计提
- **第二阶段**：信用风险显著增加但未发生信用减值 → 按整个存续期预期信用损失计提
- **第三阶段**：已发生信用减值 → 按整个存续期预期信用损失计提，利息按摊余成本净额计算

**简化处理**：对不含重大融资成分的应收账款和合同资产，准则允许直接按整个存续期预期信用损失计提（无需判断阶段），这是代账实务中最常用的方法。

## 第二步：计提方法选择

实务中常用账龄分析法（组合评估）：

| 账龄 | 参考计提比例 |
|------|-------------|
| 1年以内 | 5% |
| 1-2年 | 10% |
| 2-3年 | 30% |
| 3-4年 | 50% |
| 4-5年 | 80% |
| 5年以上 | 100% |

**注意**：上述比例仅为参考，企业应根据自身历史损失率、前瞻性信息合理确定。比例一经确定不得随意变更，变更须作为会计估计变更处理。

## 第三步：实际核销条件

应收款项满足以下条件可核销：
1. 债务人破产清算后确实无法收回
2. 债务人死亡且无遗产可供清偿
3. 债务人逾期3年以上且有确凿证据证明无法收回
4. 与债务人达成债务重组协议且确认无法收回部分

核销需提供：催收记录、法院判决书/裁定书、债务人破产清算公告、工商注销证明等。

## 第四步：税务风险提示

**关键差异**：会计上计提的坏账准备在税务上不得税前扣除（《企业所得税法》第十条）。只有实际发生的、符合条件的坏账损失才能税前扣除，且必须按《企业资产损失所得税税前扣除管理办法》（国家税务总局公告2011年第25号）的要求留存备查资料。

每年汇算清缴时须做纳税调增处理（调增当年计提数，调减当年转回/核销数）。
