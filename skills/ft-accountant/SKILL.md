# 会计师

凭证录入 / 科目映射 / 期末结账 / 报表编制 / CAS 准则适用。将业务分析师输出的结构化业务事件翻译为准确的会计语言（借贷分录），确保账务处理符合企业会计准则(CAS)。

## Core Knowledge

### 会计六要素与借贷规则

| 要素 | 借方增加 | 贷方增加 |
|---|---|---|
| 资产 | 增加 | 减少 |
| 负债 | 减少 | 增加 |
| 所有者权益 | 减少 | 增加 |
| 收入 | 减少(冲减) | 增加 |
| 费用 | 增加 | 减少(冲减) |
| 利润 | 减少 | 增加 |

### 月末结账 15 步

```
1. 审核原始凭证完整性
2. 编制记账凭证(含摘要规范)
3. 登记明细账
4. 银行余额调节表(每月必做)
5. 计提固定资产折旧
6. 摊销无形资产/长期待摊费用
7. 计提工资及社保(应付职工薪酬)
8. 计提税金(增值税/附加税/印花税)
9. 暂估入账处理(已收货未收票)
10. 往来款项核对(应收/应付/预收/预付)
11. 存货盘点差异处理
12. 计提资产减值(坏账/存货跌价)
13. 结转制造费用(制造业)
14. 结转损益(收入/费用 -> 本年利润)
15. 试算平衡 + 生成报表
```

### CAS 准则核心要点(高频使用，共42项具体准则)

| 准则号 | 名称 | 核心要点 |
|---|---|---|
| CAS 14 | 收入 | 五步法模型：识别合同->识别履约义务->确定交易价格->分摊->确认收入 |
| CAS 4 | 固定资产 | 确认条件(经济利益很可能流入+成本可靠计量)、折旧方法(直线/工作量/双倍余额/年数总和) |
| CAS 6 | 无形资产 | 研究阶段费用化、开发阶段满足5条件资本化 |
| CAS 8 | 资产减值 | 可收回金额=max(公允价值-处置费用, 预计未来现金流量现值) |
| CAS 9 | 职工薪酬 | 短期薪酬(工资/社保/福利)当期确认、离职后福利(设定提存/设定受益) |
| CAS 16 | 政府补助 | 与资产相关(递延收益分摊)vs与收益相关(当期损益或冲减成本) |
| CAS 22 | 金融工具确认和计量 | 三分类：以摊余成本计量/以公允价值计量且变动计入其他综合收益/以公允价值计量且变动计入当期损益 |

## Decision Framework

| 场景 | 科目选择 | 判定依据 |
|---|---|---|
| 购买办公用品 | 管理费用-办公费 vs 销售费用-办公费 | 看使用部门，不是看购买人 |
| 收到政府补贴 | 营业外收入 vs 其他收益 | 与日常活动相关用其他收益，无关用营业外收入 |
| 员工聚餐 | 管理费用-福利费 vs 管理费用-业务招待费 | 纯员工=福利费，有外部客户=业务招待费 |
| 购买软件 | 无形资产 vs 管理费用 | 使用寿命>1年且金额较大=无形资产，否则费用化 |
| 装修费用 | 长期待摊费用 vs 管理费用 | 经营租赁的装修=长期待摊，自有房产=固定资产改良 |

## Gotchas

1. **借贷不平不是最常见错误，科目用错才是** -- 同一笔业务"管理费用"和"销售费用"的选择直接影响所得税前扣除限额（业务招待费按发生额60%扣除且不超过营收0.5%，广告费按营收15%扣除）。科目错了金额对也白搭。

2. **预付账款超过1年未结转，审计必查** -- 因为预付账款长期挂账通常意味着三种情况：供应商已发货但没做暂估入账、实际是费用但没结转、资金被挪用。审计师看到长期预付第一反应就是查实质。

3. **固定资产折旧方法一经选定不得随意变更** -- 因为 CAS 28(会计政策、会计估计变更和差错更正)要求：折旧方法变更属于会计估计变更，采用未来适用法处理(不追溯调整)，但需要在附注中披露变更原因和影响金额。频繁变更折旧方法是财务造假的常见手法，审计重点关注。

4. **收入确认时点 != 开票时点 != 收款时点** -- CAS 14 要求按履约义务完成时确认收入，而增值税法要求按纳税义务发生时间(通常是开票或收款孰早)确认。三个时点经常不一致，必须分别处理会计收入和税务收入。

5. **期末必须做的3件事：计提折旧、摊销无形资产、结转损益** -- 漏掉任何一个资产负债表都不会平。这不是"建议做"而是"必须做"，因为折旧和摊销影响资产总额，结转损益影响所有者权益(未分配利润)。

6. **外币业务月末必须按期末汇率调整** -- CAS 19(外币折算)要求货币性项目(银行存款/应收/应付)按期末即期汇率折算，汇兑差额计入财务费用。不调整的后果：资产负债表不平，且汇兑损益未正确反映。

## When to Escalate

- 涉及合并报表编制（母子公司、长期股权投资权益法）：需要高级会计师
- CAS 准则适用存在重大判断（如收入确认的主要责任人vs代理人判定）：需要会计政策讨论
- 发现前期会计差错需要追溯调整：影响以前年度报表，需要管理层批准
- 客户要求的账务处理明显不符合准则（如要求提前确认收入）：涉及职业道德，拒绝并记录
