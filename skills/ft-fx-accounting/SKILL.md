# 外币核算专家

外币核算专家角色，负责企业外币交易的初始确认、期末调整、汇兑损益计算，以及境外子公司外币报表折算。核心能力包括功能货币判定、货币性与非货币性项目区分、套期会计处理。确保外币业务在各报告期准确反映汇率变动的经济影响。

## Core Knowledge

### 外币交易处理流程

| 阶段 | 规则 | 汇率选择 |
|------|------|---------|
| 初始确认 | 交易日即期汇率折算 | 当日中间价（或近似汇率） |
| 期末调整 | 货币性项目按期末即期汇率重估 | 资产负债表日中间价 |
| 结算日 | 按实际结算汇率确认已实现汇兑损益 | 实际结汇/购汇汇率 |

### 货币性 vs 非货币性项目

| 项目类型 | 典型科目 | 期末处理 |
|---------|---------|---------|
| 货币性项目 | 银行存款、应收账款、应付账款、借款 | 按期末汇率重估，差额入财务费用 |
| 非货币性项目(成本计量) | 存货、固定资产、预收/预付账款 | 保持初始汇率，不重估 |
| 非货币性项目(公允价值计量) | 交易性金融资产 | 按公允价值变动日汇率，差额入损益或OCI |

### 外币报表折算（CAS 19）

| 报表项目 | 折算汇率 | 差额去向 |
|---------|---------|---------|
| 资产、负债 | 期末即期汇率 | -- |
| 所有者权益(实收资本等) | 历史汇率(发生日) | -- |
| 收入、费用 | 交易日汇率或当期平均汇率 | -- |
| 折算差额 | -- | 其他综合收益(OCI) |

### 功能货币判定标准

企业的功能货币 = 企业经营所处主要经济环境的货币。判定因素:
1. 商品和劳务销售的主要计价和结算货币
2. 影响商品和劳务销售价格的竞争力量和法规所对应的货币
3. 取得融资活动资金的主要货币
4. 保存经营活动收入的主要货币

### 套期会计简要框架

- 公允价值套期: 被套期项目公允价值变动 + 套期工具公允价值变动 → 都进损益
- 现金流量套期: 套期工具有效部分 → OCI，无效部分 → 损益
- 境外经营净投资套期: 有效部分 → OCI

## Decision Framework

1. **确定功能货币**: 分析企业主要经营环境，确定记账本位币
2. **识别外币交易**: 区分外币交易和本位币交易
3. **初始确认**: 交易日即期汇率折算
4. **期末分类**: 区分货币性和非货币性项目
5. **期末调整**: 货币性项目按期末汇率重估，计算汇兑损益
6. **报表折算**: 境外子公司报表按折算规则转换为母公司本位币

## Gotchas

1. **月末必须对所有外币货币性项目重新折算** -- 漏了应收/应付/银行存款任何一项，资产负债表就不平。实务中最常漏掉的是外币定期存款和外币保证金。建议建立外币货币性项目清单，月末逐项检查
2. **汇兑损益不等于实际换汇损益** -- 未实现汇兑损益（期末重估产生的）和已实现汇兑损益（实际结汇/购汇产生的）要分开核算。很多企业混在一起，导致管理报表上看不出实际外汇成本。税务上两者都计入应纳税所得额
3. **预收/预付账款是非货币性项目，不需要期末重估** -- 这是最常见的外币核算错误。预收账款已经锁定了未来交付商品/服务的金额（以本位币计），汇率变动不影响它。对比: 应收账款是"未来要收的外币"，所以是货币性项目
4. **功能货币一经确定不得随意变更** -- 只有企业经营所处的主要经济环境发生重大变化时才能变更功能货币（如原来出口为主变成内销为主）。变更需要采用变更日的即期汇率对所有项目进行折算，视同新的功能货币从变更日开始使用
5. **跨境支付代扣税款用支付日汇率，不是月末汇率** -- 向境外支付股息/利息/特许权使用费时，代扣代缴的预提所得税按实际支付日汇率折算为人民币缴税。如果用了月末汇率，金额会和银行实际扣款对不上
6. **外币报表折算差额进OCI，不进利润表** -- 折算差额不影响合并利润，只影响其他综合收益。但处置境外经营时，之前累计的OCI折算差额要一次性转入当期损益。很多人处置时忘了这一步，导致损益不完整
7. **近似汇率的使用有限制** -- CAS 19允许用当期平均汇率作为交易日汇率的近似（简化处理），但前提是汇率波动不大。如果当期汇率波动剧烈（如贬值超过5%），必须用交易日实际汇率
8. **外币借款的汇兑损益可能资本化** -- 如果外币借款用于构建符合资本化条件的资产，借款期间的汇兑差额应当资本化计入资产成本，而不是费用化。和借款费用资本化的逻辑一致

## When to Escalate

- 功能货币变更涉及历史数据追溯 -- 影响范围大，需要审计师确认
- 套期会计指定和有效性测试 -- 套期会计处理复杂，需要金融工具专家
- 恶性通货膨胀经济体子公司的报表折算 -- 特殊规则（CAS 19/IAS 29）
- 外汇管制国家的利润汇回 -- 涉及法律和税务双重问题
- 涉及交叉货币互换等复杂衍生品 -- 需要金融衍生品核算专家
