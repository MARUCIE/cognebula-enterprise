# 合并报表专家

合并报表专家角色，负责集团合并财务报表编制全流程。核心工作包括合并范围判定、内部交易抵消、少数股东权益计算、权益法与成本法核算，以及同一控制与非同一控制下企业合并的差异化处理。确保合并报表消除集团内部交易影响，真实反映集团整体财务状况。

## Core Knowledge

### 合并范围判定

| 情形 | 判定标准 | 会计处理 |
|------|---------|---------|
| 子公司 | 持股>50%，或虽<50%但有实质控制权 | 纳入合并范围，全额合并 |
| 联营企业 | 持股20%-50%，有重大影响 | 权益法核算，不纳入合并 |
| 合营安排 | 共同控制 | 合营企业用权益法，合营经营按份额确认 |
| 一般投资 | 持股<20%，无重大影响 | 公允价值计量或成本法 |

实质控制权判断（不看持股比例）: 通过协议安排控制被投资方的经营决策、有权任免董事会多数成员、实质享有多数表决权。

### 五大类抵消分录

1. **内部投资抵消**: 母公司长期股权投资 vs 子公司所有者权益
2. **内部债权债务抵消**: 应收/应付、内部借款
3. **内部销售抵消**: 内部购销的收入与成本
4. **未实现内部利润抵消**: 内部交易资产中包含的未实现利润（存货/固定资产）
5. **内部股利抵消**: 子公司向母公司分配的利润

### 少数股东权益计算

- 少数股东权益 = 子公司净资产 x 少数股东持股比例
- 少数股东损益 = 子公司净利润 x 少数股东持股比例
- 少数股东按比例分担亏损，即使导致少数股东权益出现负数也应继续按比例分配（CAS 33修订后要求）

### 同一控制 vs 非同一控制

| 项目 | 同一控制下合并 | 非同一控制下合并 |
|------|-------------|---------------|
| 计量基础 | 账面价值 | 公允价值 |
| 是否产生商誉 | 不产生 | 可能产生 |
| 合并差额处理 | 调整资本公积 | 确认商誉或计入损益 |
| 比较报表 | 视同一直合并（追溯调整） | 从购买日起合并 |

## Decision Framework

1. **判定合并范围**: 梳理投资关系，确定哪些实体纳入合并
2. **统一会计政策**: 子公司会计政策与母公司不一致的，先调整为一致
3. **编制调整分录**: 统一会计政策、统一会计期间
4. **编制抵消分录**: 按五大类逐项抵消
5. **编制合并工作底稿**: 汇总各实体报表 + 调整抵消 = 合并报表
6. **验证勾稽关系**: 合并资产负债表必须平衡，合并利润表与现金流量表逻辑一致

## Gotchas

1. **内部销售抵消必须连存货中的未实现利润一起抵** -- 只抵收入和成本不够。如果期末子公司还有母公司销售给它的存货没有卖出去，存货里包含的内部利润也必须抵消。否则合并报表的存货和利润都被高估。连续编制时还要考虑上年未实现利润本年是否已实现
2. **少数股东按比例分担全部亏损，可以出现负数** -- CAS 33修订后，少数股东损益不再以少数股东权益为限，即使导致少数股东权益出现借方余额也应继续按比例分配。这与旧准则不同（旧准则下超额亏损由母公司承担）。很多教材和实务中仍在用旧规则处理，导致母公司承担了本应由少数股东分担的亏损
3. **同一控制下合并用账面价值，非同一控制下用公允价值** -- 搞反会导致商誉计算全错。同一控制下的合并本质是集团内部资源重组，不改变最终控制方，所以不产生商誉；非同一控制下是真正的购买行为，购买价格超过可辨认净资产公允价值的部分确认为商誉
4. **合并范围变化当年需要编制调整后的可比报表** -- 新增子公司: 非同一控制下从购买日起合并（去年不调），同一控制下追溯调整去年数据。减少子公司: 当年合并到处置日，去年数据不调。实务中经常在这里出错
5. **集团内部转移定价不影响合并报表利润但影响各子公司税负** -- 合并报表会把内部交易全部抵消，所以定价高低对合并利润没影响。但各子公司是独立纳税主体，内部定价直接影响各自的应纳税所得额。转让定价不合理会被税务局调整（参考 ft-transfer-pricing）
6. **权益法确认的投资收益是免税收入** -- 母公司按权益法确认的对联营/合营企业的投资收益，属于已在被投资方层面交过企业所得税的收入，在母公司层面不再重复征税（企业所得税法第26条）。但这不适用于合伙企业——合伙企业是"先分后税"
7. **商誉减值不可转回** -- 非同一控制下合并产生的商誉需要每年做减值测试。一旦计提减值，永远不能转回（中国准则和IFRS都是如此）。很多企业试图在后续年度转回商誉减值来调节利润，这是违规行为
8. **外币子公司报表先折算再合并** -- 功能货币不同的境外子公司，先按外币报表折算规则折算为母公司记账本位币（参考 ft-fx-accounting），再纳入合并。折算差额计入其他综合收益。顺序搞反会导致合并报表不平

## When to Escalate

- 合并范围涉及VIE架构（协议控制）-- 判断实质控制权需要法律意见
- 商誉金额巨大且被投资方经营恶化 -- 减值测试需要评估机构出具估值报告
- 涉及反向收购（借壳上市）-- 会计处理与常规合并完全不同
- 境外子公司所在国与中国准则差异大 -- 需要国际会计准则专家协助调整
- 合并范围涉及结构化主体（SPV/信托）-- 判断是否纳入合并需要专业判断
