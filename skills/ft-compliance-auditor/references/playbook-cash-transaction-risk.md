# 场景：大额现金交易预警

**政策依据**：中国人民银行令〔2016〕第3号（大额交易和可疑交易报告管理办法）；《现金管理暂行条例》第五条；《税收征管法》第十七条

## 第一步：了解银行自动报告阈值

金融机构必须向中国反洗钱监测分析中心报告的大额交易：

| 交易类型 | 自然人 | 法人/其他组织 |
|----------|--------|-------------|
| 单日现金收付 | ≥ 5万元 | ≥ 20万元 |
| 境内转账 | ≥ 50万元 | ≥ 200万元 |
| 跨境转账 | ≥ 1万美元等值 | ≥ 20万元等值 |

**关键**：银行自动报告，企业/个人无法阻止。金税四期与人民银行反洗钱系统已实现数据共享。

## 第二步：企业现金交易常见风险点

| 风险行为 | 税务后果 |
|----------|----------|
| 大额现金收入不入账 | 隐匿收入，偷税 |
| 大额现金支付无发票 | 无法税前扣除（单次>500元需发票） |
| 公转私后现金取出 | 涉嫌套取资金，股东借款超期视同分红 |
| 拆分交易规避报告 | 可疑交易报告，加重监管 |
| 频繁存取整数金额 | 触发银行可疑交易监控模型 |

**决策规则**：单笔对公现金支出超过1000元应优先使用转账支付。企业间交易超过1000元建议全部对公转账（参照《现金管理暂行条例》限制范围）。

## 第三步：自查清单

1. 调取近12个月银行现金收支明细
2. 逐笔核对现金收入是否全部入账并申报
3. 核查现金支出是否取得合规发票
4. 检查公转私记录：是否有借款超过一年未归还的（视同分红计征20%个税）
5. 检查是否存在化整为零的拆分交易行为

## 第四步：整改规范

- **制度建设**：制定《现金管理制度》，明确现金使用范围、审批权限、单笔限额
- **库存现金控制**：设定库存现金限额（一般为3-5天日常零星支出需要），超额及时送存银行
- **大额交易登记**：所有超过5000元的现金交易需部门负责人审批并登记用途
- **收款方式优化**：推广对公转账、POS收款、企业二维码收款，减少现金交易占比
- **税务自查周期**：每季度进行一次现金交易与账面收入的交叉核对
