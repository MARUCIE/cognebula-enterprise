# 场景五：标准成本差异分析

**政策依据**：《企业产品成本核算制度（试行）》（财会〔2013〕17号）第18条；管理会计应用指引第301号——标准成本法

## 第一步：初步判别——标准成本法适用性

**适用条件**：
1. 产品生产工艺稳定，可制定标准
2. 需要成本控制和绩效考核
3. 有历史数据或工程标准作为基准

**三类标准**：
- 理想标准：理论最优（几乎不可能达到）
- 正常标准：正常效率下可达到（最常用）
- 现行标准：基于最近实际情况制定

## 第二步：深入分析——三大差异分解

**总差异 = 实际成本 - 标准成本**

**1. 直接材料差异**：
- 价格差异 = (实际价格 - 标准价格) × 实际用量
- 用量差异 = (实际用量 - 标准用量) × 标准价格
- 责任：价格差异→采购部门；用量差异→生产部门

**2. 直接人工差异**：
- 工资率差异 = (实际工资率 - 标准工资率) × 实际工时
- 效率差异 = (实际工时 - 标准工时) × 标准工资率
- 责任：工资率差异→人力资源部门；效率差异→生产部门

**3. 制造费用差异**（二因素分析法）：
- 预算差异 = 实际制造费用 - 按实际产量的弹性预算
- 产量差异 = 弹性预算 - 标准制造费用

**数字案例**：
| 项目 | 标准 | 实际 | 差异 |
|-----|-----|-----|-----|
| 材料：10kg×5元 = 50元 | 50元 | 11kg×5.5元 = 60.5元 | 不利10.5元 |
| → 价格差异 | | (5.5-5)×11 = 5.5元不利 | |
| → 用量差异 | | (11-10)×5 = 5元不利 | |
| 人工：2h×20元 = 40元 | 40元 | 2.5h×18元 = 45元 | 不利5元 |
| → 工资率差异 | | (18-20)×2.5 = 5元有利 | |
| → 效率差异 | | (2.5-2)×20 = 10元不利 | |

  差异金额小（<预算5%）→ 计入当期损益

## 第四步：结论/建议

- **标准定期更新**：至少每年更新标准成本（原材料价格/工艺变化/工资调整）
- **月度分析**：每月出具差异分析报告，异常差异及时追踪
- **责任明确**：价格差异归采购，用量/效率差异归生产，据此考核
- **差异处理**：差异金额在实际成本5%以内可直接计入损益，超过须分摊
- **持续改善**：不利差异连续出现 → 检查标准是否过时或流程是否退化
