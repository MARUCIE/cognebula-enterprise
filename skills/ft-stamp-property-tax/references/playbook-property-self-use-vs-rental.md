# 场景：自用+出租房产税分别计算

**政策依据**：《房产税暂行条例》第三条、第四条；国税发〔2005〕173号

## 第一步：确认房产使用状态分类

同一房产可能存在以下使用状态组合：
- **全部自用**：从价计征（余值×1.2%）
- **全部出租**：从租计征（租金×12%）
- **部分自用+部分出租**：需按面积分别计算
- **部分出租+部分闲置**：闲置部分按自用从价计征
- **全部免租期**：按自用从价计征（财税〔2010〕121号）

关键原则：出租部分从租计征，非出租部分（含自用和闲置）从价计征。年度内使用状态发生变化的，按变化当月起调整计征方式。

## 第二步：面积分摊与原值计算

**面积分摊**：按建筑面积比例分摊房产原值。

示例：企业拥有办公楼总面积5,000平方米（原值含地价3,000万），其中出租2,000平方米、自用3,000平方米。

自用部分房产原值 = 3,000万 × (3,000÷5,000) = 1,800万
自用部分应纳税额 = 1,800万 ×（1-30%）× 1.2% = 15.12万/年

出租部分年租金收入120万（不含增值税）
出租部分应纳税额 = 120万 × 12% = 14.4万/年

合计年度房产税 = 15.12 + 14.4 = 29.52万

## 第三步：年度内用途变化处理

情形一：年中新增出租。假设7月起将原自用的1,000平方米出租，1-6月按自用从价计征，7-12月按出租从租计征。

- 1-6月（自用4,000㎡）：3,000万×(4,000/5,000)×(1-30%)×1.2%÷2 = 10.08万
- 7-12月（自用3,000㎡）：3,000万×(3,000/5,000)×(1-30%)×1.2%÷2 = 7.56万
- 7-12月（出租1,000㎡+原出租2,000㎡合计3,000㎡租金假设90万）：90万×12% = 10.8万

情形二：年中收回出租改自用。同理按月分段计算。

实操建议：在房产税台账中按月记录每栋/每层房产的使用状态，便于准确计算。

## 第四步：申报操作与风险控制

房产税税源采集时，需按房产逐栋填报：房产证号、坐落地址、原值、面积、从价/从租标识、租金收入。对于部分出租的房产，建议在税源信息中将同一栋房产按自用和出租分拆为两条记录。

风险控制要点：
- 出租面积与租赁合同约定面积一致性核对
- 租金收入与增值税申报收入一致性核对
- 年中变化是否及时更新税源信息
- 关联方出租是否按市场价格确认租金（防止低价出租少缴房产税）
- 免租期是否从价计征（常见遗漏）

关键输出：房产使用状态台账 + 面积分摊计算表 + 分段计税底稿 + 年度房产税汇总 + 税源采集表。
