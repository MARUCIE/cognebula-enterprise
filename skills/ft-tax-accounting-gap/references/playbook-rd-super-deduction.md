# 场景：研发费加计扣除归集

**政策依据**：《企业所得税法》第三十条第一项；财税〔2015〕119号；国家税务总局公告2017年第40号；财政部 税务总局公告2023年第7号（加计扣除比例提至100%）

## 第一步：判断适用条件

**适用企业**：除烟草制造业、住宿和餐饮业、批发和零售业、房地产业、租赁和商务服务业、娱乐业以外的所有企业。

**加计扣除比例**（2023年起）：
- 所有适用企业：研发费用 × 100% 加计扣除
- 集成电路/工业母机企业：研发费用 × 120% 加计扣除

**研发活动界定**：企业为获得科学与技术新知识、创造性运用科学技术新知识、实质性改进技术/产品/服务而持续进行的具有明确目标的系统性活动。

**不适用**：
- 产品的常规性升级
- 市场调查/推广
- 管理方面的研究
- 质量控制/测试
- 社会科学/人文科学研究

## 第二步：费用归集范围（正面清单）

| 费用类别 | 具体内容 |
|----------|----------|
| 人员人工费用 | 研发人员工资薪金+五险一金+外聘研发人员劳务费 |
| 直接投入费用 | 材料/燃料/动力+中间试制品费用+研发仪器/设备维修费 |
| 折旧费用 | 研发专用设备折旧 |
| 无形资产摊销 | 研发用软件/专利/非专利技术摊销 |
| 新产品设计费 | 设计/制定/试验/检验费 |
| 其他相关费用 | 专家咨询费/研发成果鉴定费等（不超过可加计总额的10%） |

## 第三步：关键操作要求

**研发项目管理**：
1. 建立研发项目立项制度（每个项目须有立项报告）
2. 设置研发费用辅助账（国家税务总局公告2021年第28号简化格式）
3. 研发人员名单及工时分配记录
4. 研发成果报告或阶段性成果文档

**委托研发处理**：
- 委托境内机构/个人研发：按实际发生费用的80%计入委托方加计扣除基数
- 委托境外研发：按实际发生额的80%计入，但不得超过境内研发费的2/3

## 第四步：风险提示

- **归集口径**：研发人员兼做其他工作的，工资须按工时合理分摊，不得将非研发人员工资计入
- **辅助账必备**：未按要求设置辅助账的，不得享受加计扣除
- **预缴享受**：7月和10月预缴时可选择就上半年/前三季度研发费享受加计扣除
- **与高新技术企业重叠**：同时享受高新15%税率和加计扣除是允许的（叠加享受）
- **失败的研发**：研发活动未取得预期成果，其费用仍可加计扣除（鼓励创新试错）
