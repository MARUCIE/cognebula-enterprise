# 场景十：年终结账关键日期

**政策依据**：《企业会计准则——基本准则》第39条（会计期间）；《企业所得税法》第54条（汇算清缴）

## 第一步：初步判别——年终结账的时间窗口

**12月份关键操作日历**：
```
12月1日-10日：
  ● 11月份月度申报+税款缴纳
  ● 年终预估：全年利润预测+税负测算
  ● 存货盘点计划制定

12月11日-20日：
  ● 年终存货实地盘点
  ● 固定资产盘点
  ● 往来款对账确认函发出
  ● 股东借款清理（12月31日前归还）

12月21日-31日：
  ● 年终费用归集截止
  ● 资产减值/跌价准备评估
  ● 坏账损失确认
  ● 税收优惠条件最终确认
  ● 年终奖计提（须在次年5月31日前发放）
  ● 关账准备
```

## 第二步：深入分析——年终必做的税务动作

| 事项 | 截止日 | 影响 |
|-----|-------|-----|
| 股东借款清理 | 12月31日 | 超期视同分红缴20%个税 |
| 小微企业判定 | 12月31日 | 确认年度应纳税所得额≤300万 |
| 高新收入占比 | 12月31日 | 确认≥60%，否则失去15%优惠 |
| 研发费归集 | 12月31日 | 确认辅助账完整，加计扣除最大化 |
| 暂估入库 | 12月31日 | 暂估金额影响全年成本 |
| 资产处置 | 12月31日 | 处置时点影响折旧和损失扣除 |
| 捐赠支出 | 12月31日 | 确认不超过利润12%的限额 |

## 第三步：佐证/验证——结账检查清单

**收入检查**：
- [ ] 12月发出商品是否已确认收入（权责发生制）
- [ ] 预收款是否应转收入
- [ ] 视同销售事项是否已确认

**成本费用检查**：
- [ ] 12月采购是否已暂估入库
- [ ] 年终奖是否已计提
- [ ] 折旧/摊销是否计提至12月
- [ ] 限额费用（招待费/广告费/福利费）是否超标

**资产负债检查**：
- [ ] 存货盘点差异处理
- [ ] 固定资产盘点差异处理
- [ ] 坏账准备/跌价准备计提
- [ ] 往来款对账差异处理

**税务检查**：
- [ ] 增值税进项勾选确认（12月31日前完成当年发票确认）
- [ ] 各税种已足额申报
- [ ] 股东借款已清理
- [ ] 优惠条件已确认

## 第四步：结论/建议

- **12月中旬启动**：年终结账工作从12月15日开始，不要拖到最后几天
- **盘点优先**：存货和固定资产盘点安排在12月20日前完成
- **预估利润**：12月初预估全年利润，评估是否需要调整费用安排
- **发票催收**：12月31日前催收全年暂估对应的发票（虽然可延至次年5月31日）
- **团队协作**：年终结账涉及会计/税务/业务/仓库多部门，须提前召开协调会
