# 场景：批量法规合规审查

**政策依据**：《税收征收管理法》第四条（纳税人依法纳税义务）；第五十四条（税务检查权限）；《企业所得税法》第五条至第二十一条（应税收入与扣除项目）

## 第一步：确定审查范围

**按审查目的选择范围**：
| 审查类型 | 范围 | 触发条件 |
|----------|------|----------|
| 全面审查 | 所有涉税事项 | 新客户接入/年度审计 |
| 专项审查 | 特定税种或业务 | 稽查通知/政策变动 |
| 更新审查 | 近期变动的法规 | 季度/半年度例行 |
| 应急审查 | 特定风险点 | 收到税务预警/媒体曝光 |

## 第二步：批量审查流程

**Phase 1：法规清单生成**
1. 根据客户行业+业务类型，自动生成应适用的法规清单
2. 按税种分组：增值税法规组/所得税法规组/其他税种组
3. 标记每条法规的核心合规要求

**Phase 2：合规状态比对**
```
法规要求            | 企业实际           | 合规状态
--------------------|--------------------|---------
小规模年销售额≤500万 | 实际销售额480万    | 合规
发票按月认证抵扣     | 按月认证           | 合规
业务招待费≤收入5‰    | 实际超出           | 不合规
研发费辅助账         | 未建立             | 不合规
```

**Phase 3：不合规项处理**
1. 按风险等级排序（高→低）
2. 逐项列出不合规原因和整改建议
3. 标注整改时限和责任人

## 第三步：批量审查自动化

**KG辅助审查**：
```
输入: 客户基本信息(行业/规模/纳税人资格/享受优惠)
自动匹配: KG中该行业+规模的全部适用法规
输出: 合规检查清单(待人工确认合规状态)
```

**检查清单模板**（按税种分组）：

```
## 增值税合规检查
□ 纳税人资格与实际销售额匹配
□ 进项认证/勾选按期完成
□ 不得抵扣进项已做转出
□ 发票开具品名与实际业务一致
□ 简易计税备案完整（如适用）

## 企业所得税合规检查
□ 季度预缴按时足额
□ 限额扣除项目未超标（招待费/广告费/福利费/教育经费）
□ 资产折旧方法与备案一致
□ 关联交易定价合理
□ 优惠政策资格条件持续满足

## 个人所得税合规检查
□ 全员全额扣缴申报
□ 专项附加扣除信息真实
□ 劳务报酬代扣义务履行
□ 年终奖计税方式选择合理
```

## 第四步：审查报告输出

**标准输出格式**：
```
审查日期: YYYY-MM-DD
客户名称: XX公司
审查范围: [全面/专项: XX]
审查法规数: XX条
合规项: XX项
不合规项: XX项
高风险项: XX项

不合规项清单:
1. [法规引用] - [不合规描述] - [风险等级] - [整改建议] - [整改时限]
2. ...

总体合规评级: [优/良/中/差/严重不合规]
```
