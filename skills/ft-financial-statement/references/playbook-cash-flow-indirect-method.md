# 场景：现金流量表间接法编制

**政策依据**：《企业会计准则第31号——现金流量表》

## 第一步：间接法原理

间接法从净利润出发，通过调整不涉及现金的项目，倒推出经营活动产生的现金流量净额。

**核心公式**：
```
经营活动现金流量净额 = 净利润
  + 不影响现金的费用（折旧/摊销/减值/坏账）
  + 不属于经营活动的损益（处置资产损益/财务费用）
  + 经营性资产负债变动
```

## 第二步：调整项目明细

**加回不涉及现金流出的费用**：
| 调整项 | 方向 | 说明 |
|--------|------|------|
| 资产减值准备 | +（加回） | 计提减值不影响现金 |
| 固定资产折旧 | +（加回） | 折旧不是现金支出 |
| 无形资产摊销 | +（加回） | 摊销不是现金支出 |
| 长期待摊费用摊销 | +（加回） | 同上 |
| 递延所得税资产减少/负债增加 | +/+（加回） | 非现金的税务调整 |

**剔除非经营活动损益**：
| 调整项 | 方向 | 说明 |
|--------|------|------|
| 处置固定资产损失 | +（加回） | 属于投资活动 |
| 处置固定资产收益 | -（扣除） | 属于投资活动 |
| 财务费用 | +（加回） | 属于筹资活动 |
| 投资收益 | -（扣除） | 属于投资活动 |

**经营性资产负债变动**：
| 变动项 | 资产增加 | 资产减少 |
|--------|----------|----------|
| 存货 | -（占用现金） | +（释放现金） |
| 经营性应收 | -（占用现金） | +（回收现金） |
| 经营性应付 | +（延迟付款） | -（支付现金） |

## 第三步：简化编制模板

```
净利润                          [净利润金额]
加：资产减值准备                +[金额]
    固定资产折旧                +[金额]
    无形资产摊销                +[金额]
    长期待摊费用摊销            +[金额]
    处置固定资产损失（减收益）    +/-[金额]
    财务费用                    +[金额]
    投资损失（减收益）           +/-[金额]
    递延所得税资产减少（增加负数）+/-[金额]
    递延所得税负债增加（减少负数）+/-[金额]
    存货的减少（增加负数）       +/-[金额]
    经营性应收的减少（增加负数）  +/-[金额]
    经营性应付的增加（减少负数）  +/-[金额]
─────────────────────────────────
经营活动产生的现金流量净额        [合计金额]
```

## 第四步：验证与风险提示

- **交叉验证**：间接法计算的经营活动现金流净额应等于直接法的结果
- **现金流与利润背离**：净利润为正但经营现金流为负，说明盈利质量差（利润未转化为现金）
- **应收膨胀**：经营性应收大幅增加导致现金流恶化，需关注收入真实性和回款能力
- **代账简化**：小微企业可不编制现金流量表（执行小企业会计准则的可选编）
