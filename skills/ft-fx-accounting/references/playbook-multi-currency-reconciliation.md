# 场景：多币种银行对账

**政策依据**：《企业内部控制应用指引第6号——资金活动》第八条；《企业会计准则第19号》

## 第一步：建立多币种对账框架

对于持有多个外币银行账户的企业，应按币种和银行账户逐一编制银行余额调节表。对账频率：至少每月一次。对账范围：所有外币银行账户（包括境内外币账户和境外银行账户）。

对账基础数据：银行对账单（Bank Statement，按原币列示）+ 企业银行日记账（外币辅助账，按原币和本位币双重记录）。对账时以外币原币金额为对账基础，因为汇率折算差异不影响实际资金余额。

## 第二步：编制银行余额调节表（外币版）

对账公式保持与人民币相同：
- 银行对账单余额 + 企业已收银行未收 - 企业已付银行未付 = 企业账面余额（原币）
- 企业账面余额 + 银行已收企业未收 - 银行已付企业未付 = 银行对账单余额（原币）

调节后余额应在原币口径一致。常见未达账项：
- 跨境汇款在途（通常T+1至T+3）
- 银行手续费/电报费（银行已扣企业未记）
- 利息收入（银行已计企业未记）
- 跨时区交易日差异（如纽约时间 vs 北京时间）

## 第三步：汇率差异处理

对账完成后，核对企业账面的人民币折算余额：企业外币账面余额（原币）× 期末汇率 = 应有的人民币金额。如果与账面人民币余额不一致，差额即为需要调整的汇兑损益。

常见差异原因：
- 入账时使用了不同的汇率口径（中间价 vs 买入价 vs 银行实际结汇汇率）
- 上月重估后余额与本月交易入账汇率之间的差异
- 系统自动重估参数设置错误

建议：每月对账时同步完成外币重估，确保银行存款—外币户的人民币余额与"原币×期末汇率"一致。

## 第四步：特殊情况与内控建议

境外银行账户对账注意事项：时差导致截止日不同（如美国银行Statement日为月底美东时间，可能与北京时间有一天差异）。银行手续费可能以当地货币计价而非账户币种。需关注外管局规定的境外银行账户余额上限及定期报告义务。

内控建议（应用指引第6号第八条）：银行对账由非出纳人员执行（职责分离）、大额未达账项应限时清理（通常超过3个月的未达账项必须查明原因）、所有外币账户纳入资金日报监控。

关键输出：多币种银行余额调节表 + 未达账项明细 + 汇率差异调整分录 + 外币账户月度对账报告。
