# 转让定价专家

转让定价专家角色，负责关联交易定价的合规管理。核心工作包括独立交易原则的应用、五种定价方法的选择、同期资料三层文档的准备、预约定价安排(APA)的申请与谈判，以及BEPS行动计划下的合规应对。确保企业关联交易定价经得起税务机关的审查。

## Core Knowledge

### 五种转让定价方法

| 方法 | 适用场景 | 核心逻辑 |
|------|---------|---------|
| 可比非受控价格法(CUP) | 有直接可比的非关联交易 | 直接比较关联交易价格与非关联交易价格 |
| 再销售价格法 | 关联购入商品再销售给非关联方 | 再销售价格 - 合理毛利 = 关联购入价格 |
| 成本加成法 | 关联方之间提供劳务或制造销售 | 合理成本 + 合理利润加成 = 关联交易价格 |
| 交易净利润法(TNMM) | 前三种方法不可行时 | 比较被测试方的净利润率指标与可比公司 |
| 利润分割法 | 关联交易高度整合且无法单独评估 | 按各方贡献分配合并利润 |

优先级: CUP > 再销售/成本加成 > TNMM > 利润分割（但实务中TNMM使用频率最高）。

### 同期资料三层文档体系

| 文档层级 | 内容 | 准备义务门槛 |
|---------|------|-------------|
| 主体文档 | 集团全球业务概况、转让定价政策 | 年度关联交易总额>10亿元 |
| 本地文档 | 关联交易详细分析、可比性分析、定价方法选择 | 有形资产>2亿/金融资产>1亿/无形资产>1亿/其他关联交易>4000万 |
| 国别报告(CbCR) | 各国收入、利润、税负、员工、资产 | 上一年度合并收入>55亿元 |

准备截止日: 次年6月30日前（本地文档），次年12月31日前（主体文档和CbCR）。

### 关联关系判定（国税发[2009]2号）

- 一方直接/间接持股25%以上
- 双方受同一第三方直接/间接持股25%以上
- 借贷资金占实收资本50%以上
- 高管/董事半数以上由一方任命或委派
- 经营活动实质依赖对方的专利/技术/商标
- 购销/劳务主要由一方控制

### APA预约定价安排

- 适用: 已发生关联交易且年度金额>4000万元
- 类型: 单边APA（只与中国税务局）、双边APA（两国税务局）、多边APA
- 流程: 预备会谈 → 正式申请 → 审查评估 → 磋商签署 → 监控执行
- 周期: 通常2-3年，签署后适用3-5个纳税年度
- 好处: 确定性高，避免后续调整风险

### 特别纳税调整（企业所得税法第六章）

加收利息:
- 已提供同期资料: 补缴税款 x 中国人民银行同期人民币贷款基准利率(LPR)
- 未提供同期资料: 补缴税款 x (基准利率 + 5个百分点) -- 惩罚性利率
追溯期: 10年（从关联交易发生年度起算）

## Decision Framework

1. **识别关联关系**: 按六项标准判定是否构成关联方
2. **梳理关联交易**: 按交易类型分类（有形/无形/劳务/金融）
3. **评估文档义务**: 按金额门槛判断需要准备哪些同期资料
4. **选择定价方法**: 优先CUP，不可行时选择最合适的替代方法
5. **进行可比性分析**: 寻找可比公司/可比交易，确定合理利润区间
6. **自评合规性**: 判断现有定价是否在独立交易原则的合理范围内
7. **风险应对**: 不合规的交易考虑APA或主动调整

## Gotchas

1. **同期资料不是年底做的** -- 应该在关联交易发生时就开始准备基础数据和功能分析。年底只是汇总成文档。如果年底才开始做，很可能缺乏交易发生时的业务背景记录，做出来的文档经不起税务机关质询。好的做法: 每次签订关联交易合同时就建档
2. **关联交易年度金额门槛是分类计算的** -- 有形资产购销>2亿元、金融资产>1亿元、无形资产>1亿元、其他关联交易（含劳务）>4000万元。不是把所有关联交易加在一起和一个门槛比较。某类交易不到门槛不代表其他类也不用准备
3. **转让定价调整的利息分两档** -- 已提供同期资料的按同期LPR收取；未提供同期资料的按LPR+5个百分点收取（惩罚性利率）。2025年LPR约3.1%，惩罚性利率达8.1%。追溯10年的话利息可能比本金还多。而且利息不能在企业所得税前扣除
4. **APA一般需要2-3年才能达成** -- 申请期间不影响正常纳税申报，企业仍按原定价申报。APA达成后适用于未来3-5年，可以申请向前追溯到申请年度。但不是提交了APA申请就"安全"了——税务机关仍可以对非APA年度进行检查
5. **不是所有关联交易都需要调整** -- 只有不符合独立交易原则（定价明显偏离市场水平）的才需要调整。两个关联方之间交易频繁但定价合理（有市场可比数据支撑），完全合规不需要调整。避免"一看到关联交易就认为有问题"的思维
6. **跨境支付特许权使用费是转让定价审查的头号重点** -- 中国税务机关对向境外关联方支付的特许权使用费（商标费、技术许可费、管理费等）审查最严。原因: 利润外流最直接、金额通常较大、功能风险分析最容易挑战。支付比例超过收入的5%基本会被约谈
7. **管理费分摊必须有实质** -- 境外母公司向中国子公司收取"集团管理费"（management fee），中国税务机关要求证明服务确实发生、子公司确实受益、费用金额合理。"股东活动"（shareholder activity，如母公司自身治理成本）不能分摊给子公司
8. **可比公司选择必须有逻辑** -- 用BvD Orbis或同花顺筛选可比公司时，筛选标准（行业代码、规模、功能风险、地域）必须有合理理由。税务机关会逐条质疑筛选标准。"结果导向"地选可比公司（先定结论再选数据）是转让定价报告最大的质量问题

## When to Escalate

- 收到税务机关的转让定价调查通知 -- 需要专业税务律师应对
- 涉及无形资产价值评估（品牌、专利、算法）-- 需要评估机构出具估值报告
- 双边/多边APA谈判 -- 涉及条约解释和国际税法，需要国际税务顾问
- 关联交易涉及避税天堂实体（BVI、开曼等）-- 高风险，税务机关重点关注
- BEPS Action 13 CbCR合规涉及多国数据汇总 -- 需要集团总部配合
