# 场景：保税区/综保区政策

**政策依据**：《中华人民共和国海关保税港区管理暂行办法》；《海关特殊监管区域管理办法》

## 第一步：理解保税区域类型

中国海关特殊监管区域主要类型：
- **综合保税区**（综保区，当前主力形态）：整合了原保税区、出口加工区、保税物流园区的功能，是最高开放层次的海关特殊监管区域
- **保税物流中心（A/B型）**：A型为单企业型，B型为多企业园区型
- **自由贸易试验区（自贸区）**：更高层面的制度创新试验，包含但不限于海关监管

核心政策优势：境内关外——货物进入保税区视为出口（暂不征关税和进口环节税），从保税区进入国内市场时再征税。

## 第二步：综保区核心税收政策

**进口环节**：
- 从境外进入综保区：免征关税和进口环节增值税/消费税
- 从综保区进入境内（内销）：按进口货物征收关税和进口环节税
- 综保区内企业之间交易：免征增值税和消费税

**出口环节**：
- 从境内进入综保区：视同出口，可申请出口退税
- 从综保区出境：免征出口关税

**增值税政策**：区内生产企业选择内销时，按进口货物完税价格（含原料成本+加工增值+利润）计征进口增值税。区内贸易企业内销时，按报关价格计征。

## 第三步：保税加工业务操作

保税加工是综保区的核心功能之一。操作模式：进口原材料免税入区→区内加工→成品出口（免税）或内销（补税）。

加工贸易手册管理：区内企业需办理加工贸易手册，记录进出料件和成品情况。海关定期核销：料件进口数量 × 单耗 = 应产出成品量，与实际出口对照。

内销补税计算：内销时按进口料件原产地和进口时的完税价格计征关税，或按实际内销价格征收（取高者）。部分综保区实行"选择性征收关税"政策——对区内加工产品按成品或料件税率中较低者征收。

## 第四步：企业入区评估与合规管理

适合入区的企业类型：加工贸易企业（料件频繁进出口）、跨境电商仓储企业、国际物流分拨中心、保税展示交易企业、保税研发测试企业。

入区成本考量：区内租金/地价通常高于区外、海关监管合规成本（系统对接、人员培训）、物流成本（区内外运输需通关手续）。

合规管理要点：账册管理（海关联网监管系统报送）、区内货物不得出区（除正常出口或内销补税）、废料/次品处理需向海关报备、定期盘库接受海关核查。

关键输出：保税区政策适用分析 + 入区成本效益测算 + 加工贸易核销管理 + 内销补税计算方案。

**风险提示**：
- 保税货物未经海关许可擅自出区，按走私行为处理，最高可追究刑事责任
- 加工贸易手册核销差异（料件短少）可能被认定为擅自内销，补征关税+增值税+滞纳金
- 综保区内企业之间交易免征增值税，但从区内到区外内销时须按进口征税，税负可能高于预期

**关联技能协同**：
- 出口加工后内销涉及退税冲回 --> `ft-export-tax-rebate` 评估退税影响
- 进口环节增值税抵扣 --> `ft-invoice-manager` 确认海关缴款书抵扣操作
- 保税区企业转让定价 --> `ft-transfer-pricing` 审查关联交易定价合规性
