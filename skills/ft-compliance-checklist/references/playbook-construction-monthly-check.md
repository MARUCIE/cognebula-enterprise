# 场景五：建筑企业月度专项检查

**政策依据**：财税〔2017〕58号（建筑服务预缴）；国家税务总局公告2017年第11号；国家税务总局公告2016年第17号

## 第一步：初步判别——建筑企业税务特殊性

建筑业涉及**跨区预缴、差额计税、甲供材、简易计税**等特殊规则，月度检查须专项关注。

## 第二步：深入分析——月度检查清单

```
═══ 建筑企业月度专项检查表 ═══
检查月份：____年____月
检查人：________  日期：________

一、增值税处理
  □ 1.1 一般计税项目（9%）与简易计税项目（3%）分别核算
  □ 1.2 甲供工程/清包工是否选择简易计税
  □ 1.3 差额计税（分包扣除）计算正确
  □ 1.4 预收工程款增值税纳税义务判定
  □ 1.5 跨区预缴增值税已按期完成（2%预征率，简易3%预征率）

二、跨区经营预缴
  □ 2.1 各项目跨区预缴台账更新
  □ 2.2 预缴税额 = (全部价款-分包款) ÷ (1+税率) × 2%（一般计税）
  □ 2.3 项目所在地已按期预缴
  □ 2.4 预缴完税凭证已取得并保存
  □ 2.5 机构所在地申报时已抵减预缴税额

三、企业所得税跨区预缴
  □ 3.1 项目部是否在当地预缴所得税（0.2%预征率）
  □ 3.2 总机构汇总纳税处理正确

四、收入确认
  □ 4.1 工程进度确认收入（完工百分比法）
  □ 4.2 工程结算与收入确认匹配
  □ 4.3 质保金的增值税处理正确

五、成本管理
  □ 5.1 分包发票取得及时（分包扣除的前提）
  □ 5.2 材料采购发票品目与工程BOM匹配
  □ 5.3 人工成本（劳务分包/自有工人）处理合规
  □ 5.4 项目间费用分摊合理

六、发票管理
  □ 6.1 开具发票与项目合同匹配
  □ 6.2 预缴税款与开票地一致
  □ 6.3 分包发票已及时取得
```

## 第三步：佐证/验证——建筑业高频风险

1. **跨区预缴遗漏**：项目所在地未预缴 → 被项目地税务机关处罚
2. **简易计税混核**：一般计税项目误用简易计税进项抵扣 → 补税
3. **分包扣除无票**：差额计税但未取得分包方发票 → 不得扣除
4. **挂靠经营**：以他人名义承接工程 → 发票风险+法律风险

## 第四步：结论/建议

- **项目台账**：每个项目独立建立税务台账（计税方式/预缴记录/发票收支）
- **跨区管理**：设专人管理跨区预缴，每月核对各项目预缴情况
- **分包管控**：分包合同签订时即约定发票类型和开票时限
- **简易/一般区分**：一旦选择简易计税，36个月内不得变更
- **质保金处理**：质保金到期前不开票不纳税，到期后须开票纳税
