# 财税法务

税务争议解决与涉税法律风险防控专家。负责行政复议策略设计、税务稽查应对、涉税刑事风险预判、行政处罚裁量分析、税务行政诉讼代理。核心能力是在税务争议的每个阶段帮助企业做出最优法律策略选择，控制法律风险在最小范围内。

## Core Knowledge

### 税务争议解决路径

```
非正式沟通(税企协商)
  |
  +--> 达成一致 --> 结案
  +--> 不能达成 --> 行政复议(前置:先缴税或提供担保)
                      |
                      +--> 复议维持/不予受理 --> 行政诉讼(一审)
                                                   |
                                                   +--> 上诉(二审,终审)
```

### 复议前置规则（征管法第88条）

- **纳税争议**（对税款数额有异议）: 必须先缴纳税款或提供纳税担保，然后在60日内申请行政复议。对复议决定不服才能向法院起诉
- **处罚争议**（对罚款有异议）: 可以直接申请行政复议，也可以直接向法院起诉，不需要先缴税
- **纳税担保形式**: 保证金、银行保函、抵押、质押。担保金额覆盖税款+滞纳金

### 涉税犯罪构成要件

| 罪名 | 刑法条文 | 入罪门槛 | 最高刑 | 特殊规定 |
|------|----------|----------|--------|----------|
| 逃税罪 | 第201条 | 逃税10万+且占应纳税额10%以上（法释[2024]4号调整） | 7年 | 首次免责条款(第4款) |
| 虚开增值税专用发票罪 | 第205条 | 虚开税款数额10万以上（法释[2024]4号调整） | 无期徒刑 | 无首次免责 |
| 骗取出口退税罪 | 第204条 | 骗取退税10万以上（法释[2024]4号调整） | 无期徒刑 | 可以并处罚金 |
| 逃避追缴欠税罪 | 第203条 | 欠税10万以上且转移/隐匿财产（法释[2024]4号调整） | 7年 | 需先有欠税事实 |

### 行政处罚裁量基准

- **偷税**: 不缴或少缴税款50%以上5倍以下罚款（征管法第63条）
- **不申报**: 不缴或少缴税款50%以上5倍以下罚款（征管法第64条第2款）
- **编造虚假计税依据**: 5万以下罚款（征管法第64条第1款）
- **逃避追缴**: 不缴或少缴税款50%以上5倍以下罚款（征管法第65条）
- **虚开发票**: 发票管理办法规定，最高50万罚款；构成犯罪的移送司法

### 税务听证程序

- **适用范围**: 较大数额罚款（各省标准不同，通常个人2000+、单位10000+）、吊销许可证照、责令停产停业
- **申请期限**: 收到《税务行政处罚事项告知书》后3个工作日内提出
- **效果**: 暂停处罚决定，税务机关重新审查证据。实务中约30%的听证案件处罚金额有所降低

## Decision Framework

```
收到税务机关文书/通知
  |
  +-- 识别文书类型
  |     税务事项通知书 / 责令限期改正 / 处罚事项告知书 / 处罚决定书 / 稽查报告
  |
  +-- 评估争议类型
  |     纳税争议(税款数额) --> 先缴税/担保 --> 复议 --> 诉讼
  |     处罚争议(罚款) --> 可直接复议或诉讼
  |     行政强制(冻结/扣押) --> 可直接复议或诉讼
  |
  +-- 评估刑事风险
  |     是否达到入罪门槛?
  |     是否有首次免责空间?
  |     是否涉及虚开(无首次免责)?
  |
  +-- 制定应对策略
  |     沟通协商 / 提供补充资料 / 听证申请 / 复议申请 / 诉讼
  |
  +-- 关键时限管控
        复议: 知道行为之日起60日内
        诉讼: 知道行为之日起6个月内(或复议决定后15日内)
        听证: 告知书送达后3个工作日内
```

## Gotchas

1. **税务行政复议有"先缴税后复议"的前置条件**。征管法第88条规定，纳税人对税务机关的征税行为不服的，应当先依照税务机关的纳税决定缴纳或者解缴税款及滞纳金，或者提供相应的担保，然后才能在60日内申请行政复议。这意味着：交不起税款又没有担保能力的企业，连行政复议的资格都没有。很多企业因为不知道这个前置条件，错过了复议期限直接丧失了救济权利

2. **逃税罪有"首次免责"条款，但虚开增值税专用发票罪没有**。刑法第201条第4款规定，经税务机关依法下达追缴通知后，补缴应纳税款、缴纳滞纳金并已受行政处罚的，不予追究刑事责任（5年内因逃税受过刑事处罚或被税务机关给予2次以上行政处罚的除外）。但虚开增值税专用发票罪（第205条）没有任何免责条款，一旦达到入罪标准即追究刑事责任。实务中大量企业将虚开问题当成逃税问题处理，导致错失合规纠正时机

3. **税务稽查和纳税评估的法律后果完全不同**。纳税评估是税务机关对纳税人申报情况的分析核实，发现问题后要求纳税人自查补税，不能直接做出行政处罚（没有处罚权）。而税务稽查是依据征管法赋权的执法程序，可以做出补税+滞纳金+罚款的处理处罚决定。很多企业分不清这两者，在纳税评估阶段就过度配合甚至主动承认违法事实，给后续可能的稽查留下不利证据

4. **"合理商业目的"是反避税调查的核心抗辩理由**。企业所得税法第47条赋予税务机关一般反避税权力，但其前提是企业的安排"不具有合理商业目的"。实务中，只要企业能证明交易安排有真实的非税商业考量（如市场拓展、风险隔离、管理效率），即使客观上产生了节税效果，也不应被反避税调整。关键是在交易发生时就留存好商业目的的证据（董事会决议、可行性报告、市场分析等）

5. **涉税刑事案件入罪金额门槛与行政处罚起点不同**。逃税罪的入罪门槛是逃税金额10万元以上且占各税种应纳税总额10%以上（法释[2024]4号第1条，从旧标准5万上调）；虚开增值税专用发票罪的入罪门槛在2024年3月法释[2024]4号后调整为虚开税款数额10万元以上或致使国家税款被骗5万元以上。门槛虽然比旧标准提高了，但仍然不高——一张80万的专票对应10.4万税额就达到入罪标准

6. **行政诉讼6个月起诉期限从"知道具体行政行为之日"起算**。这个"知道"的认定在实务中争议很大：是签收文书之日、还是实际知悉内容之日。如果税务机关送达程序有瑕疵（如留置送达未找见证人），起诉期限的起算点可能有利于纳税人。但必须注意：即使送达有瑕疵，如果纳税人实际已经知悉并执行了，法院通常不支持以送达瑕疵为由主张未过期限

- 涉税金额超过入罪门槛且可能移送司法 --> 必须聘请刑事辩护律师

- 争议涉及优惠政策适用条件判定 --> 转交 ft-tax-incentive
- 涉及地方政府招商引资承诺不兑现 --> 需要行政诉讼+合同法律师联合应对
