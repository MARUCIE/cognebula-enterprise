# 场景四：软件企业增值税即征即退

**政策依据**：财税〔2011〕100号；财政部税务总局公告2019年第68号（延续至2027年底）

## 第一步：初步判别——即征即退条件

1. **企业资质**：增值税一般纳税人
2. **产品认定**：销售自行开发生产的软件产品（含嵌入式软件）
3. **退税标准**：实际增值税税负超过3%的部分即征即退

**计算公式**：
即征即退税额 = 当期软件产品增值税应纳税额 - 当期软件产品销售额 × 3%

**效果**：软件产品实际税负锁定在3%（标准13%，退还10%）

## 第二步：深入分析——退税计算详解

**案例**：软件企业当月软件产品销售额1,000万（不含税），进项税额30万

1. 销项税额 = 1,000 × 13% = 130万
2. 软件产品应纳税额 = 130 - 30 = 100万
3. 3%税负基准 = 1,000 × 3% = 30万
4. 即征即退税额 = 100 - 30 = **70万**

**嵌入式软件特殊计算**：
- 嵌入式软件产品 = 硬件设备 + 嵌入的软件
- 软件部分销售额 = 产品总销售额 - 硬件成本 × (1 + 成本利润率10%)
- 仅软件部分适用即征即退

**硬件成本确认**：按实际生产成本（BOM材料+加工费），须有完整的成本核算资料

## 第三步：佐证/验证——申请与合规要求

**首次申请须备材料**：
1. 软件产品登记证书或软件著作权登记证书
2. 软件产品检测报告（省级软件检测机构出具）
3. 企业开发销售的软件产品列表
4. 纳税人销售软件产品的增值税申报资料

**持续合规要求**：
- 软件产品须分别核算销售额（与硬件/其他产品分开）
- 不能分别核算的不得享受即征即退
- 退税款的用途：用于软件产品研发和扩大再生产，不征企业所得税（财税〔2012〕27号）

**常见问题**：
- 混合销售未拆分软件和硬件销售额 → 无法计算退税
- 嵌入式软件硬件成本确认缺乏依据 → 退税申请被退回
- 退税款挪用于非研发用途 → 追缴已退税款

## 第四步：结论/建议

- **产品认定先行**：先取得软件产品登记证书/著作权证书，再申请退税
- **分别核算**：在会计科目中单独设置软件产品收入和成本科目
- **嵌入式BOM**：嵌入式软件产品须建立硬件BOM清单，准确核算硬件成本
- **退税款管理**：退税款专项用于研发（保留使用记录），享受免征企业所得税
- **月度申报**：即征即退按月申报退税，注意申报期限
