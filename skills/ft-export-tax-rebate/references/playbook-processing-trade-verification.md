# 场景五：进料加工手册核销

**政策依据**：《海关进出口货物征税管理办法》；海关总署公告2014年第63号；国家税务总局公告2013年第12号

## 第一步：初步判别——进料加工手册类型

1. **电子账册（联网监管）**：适用于加工贸易量大的企业，海关联网监管
2. **纸质手册**：传统方式，逐本核销
3. **区别于来料加工**：进料加工企业自负盈亏，来料加工由外方承担

**核销含义**：手册有效期内或到期后，向海关报核进口料件使用情况和出口成品情况，确认保税料件已按规定使用。

## 第二步：深入分析——核销计算逻辑

**核销公式**：

实际使用进口料件 = 出口成品量 × 单耗标准

剩余料件 = 进口料件总量 - 实际使用量 - 损耗

**核销结果处理**：
| 情形 | 海关处理 | 税务处理 |
|-----|---------|---------|
| 料件全部用于出口 | 正常核销 | 免税进口料件抵减退税计税依据 |
| 剩余料件 | 退运出境或内销补税 | 内销部分补缴关税+增值税 |
| 实际消耗>备案单耗 | 海关核查 | 超耗部分可能不予抵扣 |
| 短少（去向不明） | 作为内销补税+可能处罚 | 同海关处理 |

**退税联动**（生产企业免抵退）：

免抵退税中"不得免征和抵扣税额"的调整：
- 不得免征和抵扣税额 = (出口FOB - 免税进口料件) × (征税率 - 退税率)
- 免抵退税额 = (出口FOB - 免税进口料件) × 退税率

免税进口料件越多 → 免抵退税额越小 → 退税越少

## 第三步：佐证/验证——核销操作流程

```
手册有效期到期前（或出口完毕后30日内）
  → 企业整理进口/出口数据
    → 录入海关加工贸易核销系统
      → 提交核销申请
        → 海关审核（书面/实地）
          → 核销通知书
            → 税务机关据此调整退税
```

**关键时间节点**：
- 手册有效期：一般1年（可申请延期）
- 核销申请：手册到期后30日内
- 逾期核销：海关可按内销处理全部保税料件

**常见核销问题**：
- 单耗备案不准确，实际消耗与备案差异大
- 串料（A手册进口料件用于B手册产品）
- 边角料/副产品未按规定处理

## 第四步：结论/建议

- **单耗管理**：如实备案单耗，变化时及时向海关申请变更
- **台账精准**：建立进口料件使用台账，做到手册级/批次级追踪
- **及时核销**：不要拖延核销，逾期后果严重
- **边角料处理**：有价值的边角料须向海关申报内销或退运
- **税务协同**：核销结果及时同步到退税申报，调整免抵退计算中的免税进口料件金额
