# 场景：预约定价安排（APA）申请

**政策依据**：国家税务总局公告2016年第64号《关于完善预约定价安排管理有关事项的公告》

## 第一步：评估是否适合申请APA

预约定价安排（APA）是税务机关与纳税人之间关于关联交易定价原则和方法的事先约定。申请APA的前提条件（64号公告第四条）：
- 已依法履行关联申报义务
- 年度关联交易金额4,000万元以上
- 已按规定准备、保存和提供同期资料

APA类型：单边（仅与中国税务机关签订）、双边（中国+对方国家税务机关，通过税收协定互相协商）、多边。双边/多边APA能同时消除双重征税风险，优于单边APA。

适合申请的情形：关联交易金额大且持续、转让定价方法有争议风险、希望获得税务确定性。

## 第二步：预备会谈与意向书

正式申请前应先与主管税务机关进行预备会谈（64号公告第六至八条）。预备会谈内容：
- 企业关联交易的基本情况
- 拟采用的转让定价方法和理由
- 可比性分析的初步结论
- 拟覆盖的关联交易年度（通常3-5年）

预备会谈不具有法律约束力，但可以评估税务机关的态度。如果税务机关认为APA可行，企业提交正式的《预约定价安排谈签意向书》。

## 第三步：正式申请与谈签

提交正式申请（64号公告第十一条），材料包括：
- 《预约定价安排正式申请书》
- 转让定价分析报告（详细的功能风险分析、可比性分析、方法选用论证）
- 关联交易预测数据（未来3-5年的交易量和利润率预测）
- 集团架构和业务描述
- 以前年度的关联申报和同期资料

谈签过程：税务机关审核材料 → 现场调查 → 双方协商（如为双边APA，还需启动相互协商程序MAP） → 达成一致 → 签署APA文本。整个过程通常耗时1-3年（单边较快，双边较慢）。

## 第四步：执行监控与续签

APA签署后，企业应严格按照约定的转让定价方法和条件执行。每年度向税务机关报送《预约定价安排执行报告》（64号公告第三十条），内容包括：当年实际关联交易情况、是否符合APA约定的条件和方法、实际利润率与约定区间的对比。

如果企业不符合APA约定的条件（如功能/风险发生重大变化、关联交易结构变化），应及时通知税务机关协商修订或终止APA。APA到期前可以申请续签（第三十七条），续签程序简化。

关键输出：APA可行性评估 + 预备会谈纪要 + 正式申请材料清单 + 转让定价分析报告 + 年度执行报告。

**风险提示**：
- APA谈签过程中企业提交的数据和分析将被税务机关掌握，谈判失败后可能成为稽查线索
- 双边APA涉及MAP互相协商，耗时可能超过3年，期间关联交易定价仍须合规
- APA约定的条件发生重大变化未及时通知税务机关，可能导致APA被撤销并追溯调整
