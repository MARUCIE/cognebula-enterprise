# 场景七：视同出口退税处理

**政策依据**：财税〔2012〕39号第2条；国家税务总局公告2012年第24号第6条

## 第一步：初步判别——哪些业务视同出口

**视同出口货物**（可享受退免税）：

1. **出口企业对外援助/对外承包/境外投资**的出口货物
2. **出口企业经海关报关进入特殊区域并销售给特殊区域内单位**的货物
3. **免税品经营企业销售**的货物（海关隔离区内）
4. **出口企业或其他单位销售给用于国际金融组织或外国政府贷款国际招标建设项目**的中标机电产品
5. **生产企业向海上石油天然气开采企业销售**的自产货物
6. **其他视同出口货物**

**特殊区域**包括：保税区、出口加工区、综合保税区、珠澳跨境工业区、保税港区等

## 第二步：深入分析——各类视同出口的退税方式

| 视同出口类型 | 退税方式 | 计税依据 |
|------------|---------|---------|
| 进入特殊区域销售 | 免抵退/免退 | 销售金额 |
| 对外承包工程出口 | 免抵退/免退 | 报关单金额 |
| 国际招标中标机电 | 免退税 | 购进金额 |
| 海上石油平台供货 | 免抵退 | 销售金额 |

**与一般出口的差异**：
- 视同出口可能没有传统的出口报关单（如进入特殊区域）
- 需要替代性单证（如海关保税核注清单、中标通知书等）
- 退税率与一般出口相同（按HS编码查询）

## 第三步：佐证/验证——申报特殊要求

1. **进入特殊区域**：
   - 提供海关保税核注清单替代报关单
   - 区内购买方须为在特殊区域注册的企业
   - 货物实际进入特殊区域（海关电子围网监控）

2. **对外承包工程**：
   - 须持有商务部门颁发的对外承包工程资质
   - 出口报关单上注明工程合同号
   - 提供工程合同和出口合同

3. **国际招标中标**：
   - 须提供中标通知书
   - 项目使用的贷款合同证明
   - 供货合同须明确为国际招标项目

**常见风险**：
- 特殊区域内转卖未实际出口 → 海关追缴
- 对外工程物资在境外挪用 → 视同内销补税

## 第四步：结论/建议

- **业务识别**：销售环节判断是否属于视同出口，避免遗漏退税申报
- **单证替代**：提前了解替代性单证要求，确保齐全
- **系统申报**：退税申报系统中选择正确的业务类型代码
- **时效管理**：视同出口的退税申报期限与一般出口相同
- **海关协同**：涉及特殊区域的须与海关确认电子数据已传输
