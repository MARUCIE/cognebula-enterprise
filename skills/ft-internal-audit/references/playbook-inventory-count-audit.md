# 场景：存货盘点审计

**政策依据**：CAS 1311《存货监盘》；《企业会计准则第1号——存货》第二十一条

## 第一步：盘点前准备

获取企业存货盘点计划，评估盘点安排的充分性：盘点时间（是否接近资产负债表日）、盘点范围（是否覆盖所有仓库/存放地点）、盘点人员（是否实行交叉盘点、保管人回避）、盘点方法（全面盘点还是抽盘）。

按CAS 1311第六条，审计师应当实施以下程序：评价管理层盘点存货的指令和程序、观察管理层盘点程序的执行、检查存货、实施抽盘。预先获取存货明细账，识别高价值、高风险品种（如易变质、易过期、难计量的存货），作为监盘重点。

## 第二步：现场监盘执行

监盘当日重点观察：
- 盘点是否停止收发货（如未停止，记录截止时间和最后收发凭证号）
- 盘点人员是否按"一人数、一人记、一人复核"操作
- 盘点标签/盘点表是否连续编号、是否双联
- 残次冷背存货是否单独标记
- 委托加工/寄存/在途存货是否通过函证确认

审计师独立抽盘：选取高价值、数量大、上年差异大的品种。从账到实（测试存在性）和从实到账（测试完整性）双向抽盘。记录抽盘结果，差异立即追查。

## 第三步：盘点差异分析

获取企业盘点差异汇总表，分析差异原因：计量误差、收发错误、记账滞后、毁损/被盗、串规格/串批号。按金额和数量计算差异率（通常容忍率为账面金额的1%-3%）。重大差异需要逐项查明原因并调账。评估差异是否表明内部控制存在缺陷（如保管不善、出入库管理松散）。

## 第四步：截止测试与跌价评估

执行存货截止测试：检查盘点日前后各5天的入库单和出库单，确认记账期间正确。核查盘点日后的退货/报废，评估是否影响期末存货余额。评估存货跌价准备：获取库龄分析，对超过正常周转期（通常1年以上）的存货评估可变现净值，确认是否已足额计提跌价准备（《存货准则》第十五条）。

关键输出：监盘计划 + 抽盘记录底稿 + 盘点差异分析 + 截止测试底稿 + 跌价评估底稿。
