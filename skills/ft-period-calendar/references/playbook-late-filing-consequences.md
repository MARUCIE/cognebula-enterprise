# 场景八：逾期申报后果及补救

**政策依据**：《税收征收管理法》第62条（逾期申报罚款）、第32条（滞纳金）；《行政处罚法》第33条（首违不罚）

## 第一步：初步判别——逾期的后果分级

| 逾期类型 | 后果 |
|---------|-----|
| 逾期申报（已缴税） | 罚款2,000以下（情节严重2,000-10,000） |
| 逾期申报（未缴税） | 罚款 + 滞纳金（0.05%/天） |
| 长期不申报 | 罚款 + 滞纳金 + 可能被认定为非正常户 |
| 连续3月不申报 | 税务登记证可能被注销 |

**滞纳金计算**：
滞纳金 = 应纳税额 × 0.05% × 滞纳天数
- 例：10万税款逾期30天 → 滞纳金 = 100,000 × 0.05% × 30 = 1,500元

## 第二步：深入分析——不同情形的补救措施

**情形一：逾期但未超过30天**
- 立即补申报+补缴税款+滞纳金
- 可能适用"首违不罚"（各地有首违不罚清单）
- 主动补报态度好 → 从轻处罚

**情形二：逾期超过30天但未超过6个月**
- 补申报+补缴+滞纳金+罚款
- 罚款通常2,000元以下
- 需到办税服务厅处理（电子税务局可能已锁定）

**情形三：逾期超过6个月或被列为非正常户**
- 须到税务机关办理非正常户解除
- 补齐所有逾期的申报表
- 缴清全部税款+滞纳金+罚款
- 法人/财务负责人信用受影响

**情形四：零申报逾期**
- 无税款 → 无滞纳金
- 但仍有逾期申报的行政处罚风险
- 首违不罚可能适用

## 第三步：佐证/验证——补救操作步骤

```
发现逾期
  → 立即登录电子税务局查看逾期状态
    → 如可在线补报 → 立即补报+缴款
    → 如系统锁定 → 携带材料到办税厅
      → 填写《逾期申报处理申请》
        → 补报申报表+缴纳税款和滞纳金
          → 接受行政处罚（如有）
            → 取得处理完毕证明
```

**所需材料**：
- 营业执照副本
- 经办人身份证
- 逾期期间的申报表
- 缴款凭证（如已补缴）
- 逾期原因说明（争取从轻）

## 第四步：结论/建议

- **立即补报**：发现逾期的第一时间补报，每多一天滞纳金多一天
- **首违不罚**：首次逾期且主动补报的，积极主张适用首违不罚
- **预防机制**：设置多重提醒（手机日历+财务软件+短信提醒）
- **代理记账监督**：委托代理记账的企业也要自行监督申报是否按时完成
- **信用修复**：因逾期产生的纳税信用扣分，可在后续年度通过合规纳税逐步修复
