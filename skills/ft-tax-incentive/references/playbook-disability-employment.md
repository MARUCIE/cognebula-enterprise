# 场景七：残疾人就业税收优惠

**政策依据**：《企业所得税法》第30条第2款；《残疾人就业条例》；财税〔2016〕52号；财税〔2007〕92号（增值税即征即退）

## 第一步：初步判别——两大税种优惠

**企业所得税**：支付给残疾职工的工资，按实际支付金额的100%加计扣除

**增值税**：安置残疾人的单位，按月退还安置的每位残疾人本省最低工资标准的4倍即征即退

**适用主体**：安置残疾人就业的企业/单位（无行业限制）

## 第二步：深入分析——享受条件与计算

**企业所得税加计扣除条件**：
1. 与残疾人签订1年以上劳动合同或服务协议
2. 为残疾人缴纳社会保险
3. 通过银行等金融机构向残疾人支付工资
4. 工资不低于当地最低工资标准

**计算**：残疾职工年薪6万 → 税前扣除6万 + 加计扣除6万 = 共扣除12万

**增值税即征即退条件**（财税〔2016〕52号）：
1. 月安置残疾人≥10人且占在职职工25%以上
2. 残疾人须持有《残疾人证》或《残疾军人证》
3. 与残疾人签订1年以上劳动合同
4. 足额缴纳社保和公积金
5. 通过银行支付工资且不低于最低工资

**退税计算**：每位残疾人每月退还增值税 = 当地月最低工资 × 4
- 如月最低工资2,200元 → 每人每月退还8,800元 → 每人每年退还105,600元

## 第三步：佐证/验证——操作与合规要点

1. **残疾人认定**：须持有残疾联合会颁发的《残疾人证》（1-4级均可）或退役军人事务部门颁发的《残疾军人证》（1-8级）
2. **在职认证**：残疾人须实际在岗工作，不能"挂靠"（税务机关会现场核查）
3. **比例计算**：安置残疾人占比 = 残疾职工人数 ÷ 在职职工总人数（含劳务派遣）
4. **退税申报**：增值税即征即退需按月向主管税务机关申请
5. **所得税申报**：在年度汇算清缴时填报加计扣除明细

**风险警示**：
- 虚假安置（残疾人未实际到岗）→ 骗取退税，面临处罚
- 残疾人到岗率不足 → 退税申请被拒
- 社保/工资发放不连续 → 不符合享受条件

## 第四步：结论/建议

- **真实安置**：必须是真实的劳动关系和实际工作岗位，虚假安置后果严重
- **岗位匹配**：根据残疾类型安排合适岗位（如肢体残疾→客服/数据录入；听力残疾→加工操作）
- **双重优惠**：企业所得税加计扣除 + 增值税即征即退可同时享受
- **残保金减免**：安置残疾人达到规定比例还可减免残疾人就业保障金
- **社会责任**：残疾人就业优惠是政策鼓励方向，合规享受兼顾企业效益和社会责任
