# 场景九：加速折旧/一次性扣除选择

**政策依据**：《企业所得税法》第32条；财税〔2014〕75号；财税〔2018〕54号（500万以下一次性扣除）；财政部税务总局公告2023年第37号（延续至2027年）

## 第一步：初步判别——适用哪种折旧政策

1. **设备金额**：单位价值≤500万 → 可一次性税前扣除（2024-2027年）
2. **行业限制**：六大行业（生物药品/专用设备/铁路船舶/航空航天/计算机/信息传输）+ 制造业 → 更宽松
3. **资产类型**：研发专用设备、新购设备、已使用设备（加速折旧仅限新购）
4. **企业盈亏状态**：当前盈利/亏损对一次性扣除的影响

**速判**：2024-2027年间新购≤500万设备 → 全行业均可一次性扣除

## 第二步：深入分析——三种折旧方法对比

**案例**：新购设备480万，预计使用10年，残值率5%

| 方法 | 第1年扣除 | 前5年累计 | 资金时间价值（4%折现） |
|-----|----------|----------|-------------------|
| 直线法 | 45.6万 | 228万 | 基准 |
| 双倍余额递减 | 96万 | 322.7万 | 优于直线 |
| 一次性扣除 | 480万 | 480万 | 最优 |

**一次性扣除节税效果**（25%税率）：
- 第1年多扣除434.4万 → 多抵税108.6万（相当于无息贷款）
- 后续年度纳税调增（折旧已提完），总税额不变但资金时间价值收益显著

## 第三步：佐证/验证——决策关键因素

**一次性扣除不一定最优的场景**：

1. **亏损企业**：当年亏损 + 一次性扣除 → 加大亏损额 → 超过5年弥补期限则浪费
   - 判定：预计未来5年（高新10年）能否消化扩大的亏损
2. **享受优惠税率过渡期**：如小微企业即将超标，优惠期最后1年一次性扣除效果更好
3. **税会差异管理**：一次性扣除产生大额税会差异，需做纳税调整台账
4. **已有大额折旧**：当年已有较多折旧扣除，一次性扣除可能导致预缴退税程序复杂

**检查清单**：
- [ ] 资产确认为"新购"（含以货币形式购进的已使用固定资产）
- [ ] 单位价值≤500万元
- [ ] 会计折旧政策独立于税务选择（会计可照常分期折旧）
- [ ] 已建立固定资产台账区分税务折旧与会计折旧
- [ ] 享受当年在企业所得税预缴申报时即可扣除

## 第四步：结论/建议

- **盈利企业+≤500万设备**：一次性扣除（默认最优选择）
- **亏损企业**：测算亏损弥补空间后决定，可能选择加速折旧而非一次性
- **大型设备>500万**：双倍余额递减法或年数总和法（需满足特定行业或研发用途条件）
- **研发设备**：一次性扣除 + 加计扣除100% → 实际抵扣率达200%
- **台账要求**：无论选择何种方法，必须建立固定资产税务折旧台账（税务检查必查项）
