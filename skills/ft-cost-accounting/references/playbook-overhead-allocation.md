# 场景三：制造费用分配方法选择

**政策依据**：《企业产品成本核算制度（试行）》（财会〔2013〕17号）第14条；《企业会计准则第1号——存货》第9条

## 第一步：初步判别——选择分配基准

**制造费用**：间接生产费用，无法直接归属于某一产品/批次，需按一定标准分配

**五种常用分配方法**：
1. 直接人工工时法
2. 直接人工成本法
3. 机器工时法
4. 直接材料成本法
5. 作业成本法（ABC）

**选择原则**：分配基准应与制造费用的发生有因果关系

## 第二步：深入分析——各方法对比

| 方法 | 分配公式 | 最佳适用 | 局限 |
|-----|---------|---------|-----|
| 人工工时 | 费用总额÷总工时×各产品工时 | 劳动密集型 | 自动化程度高时失真 |
| 人工成本 | 费用总额÷总人工成本×各产品人工成本 | 工资差异大的行业 | 不反映实际资源消耗 |
| 机器工时 | 费用总额÷总机时×各产品机时 | 设备密集型 | 需准确计量机时 |
| 材料成本 | 费用总额÷总材料成本×各产品材料成本 | 材料主导型 | 忽略加工复杂度 |
| ABC作业成本 | 按多个成本动因分别分配 | 产品多样化/间接费用占比高 | 实施成本高 |

**案例**：制造费用100万，产品A用机时600h，产品B用机时400h

- 机器工时法：A分配 = 100 × 600/1000 = 60万；B = 40万
- 若改用人工工时（A=200h, B=800h）：A=20万, B=80万
- **同一成本，不同基准 → 分配结果差异巨大 → 影响产品定价和盈利分析**

## 第三步：佐证/验证——选择决策树

```
制造费用的主要驱动因素是什么？
  |
  +-- 人力驱动（手工装配/检验/包装为主）
  |     → 直接人工工时法
  |
  +-- 设备驱动（数控/自动化/大型设备为主）
  |     → 机器工时法
  |
  +-- 多因素驱动（产品品种多/复杂度差异大）
  |     → ABC作业成本法
  |
  +-- 简单生产（单品种/品种差异小）
        → 任何方法差异不大，选择最简单的
```

**ABC法何时值得实施**：
- 间接费用占总成本>40%
- 产品品种>10种且复杂度差异大
- 传统方法下出现"低价高销量产品亏损、高价低销量产品暴利"的异常
- 管理层需要精细化成本信息支持决策

## 第四步：结论/建议

- **一致性**：选定方法后保持年度内一致，不得随意变更（会计准则要求）
- **年初预定分配率**：年初确定预定费率，年末调差异，避免月度波动失真
- **数据基础**：选择机时法须有可靠的机时记录系统（MES/PLC自动采集最优）
- **成本验证**：定期比较分配结果与实际资源消耗的合理性
- **渐进升级**：中小企业可先用传统单一基准法，数据成熟后升级到ABC
