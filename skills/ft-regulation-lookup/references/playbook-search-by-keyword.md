# 场景：关键词语义搜索技巧

**政策依据**：《税收征收管理法》第七条（税法宣传与查询权利）；《政府信息公开条例》第二十条（主动公开的税收法规信息）

## 第一步：关键词构造原则

**精准关键词优于宽泛关键词**：
| 查询意图 | 差的关键词 | 好的关键词 |
|----------|-----------|-----------|
| 小微企业所得税优惠 | "小微 优惠" | "小型微利企业 应纳税所得额 300万 税率" |
| 研发费加计扣除 | "研发 扣除" | "研发费用 加计扣除 100% 归集 辅助账" |
| 增值税留抵退税 | "留抵 退税" | "增值税 增量留抵 存量留抵 进项构成比例" |

**组合策略**：
- 税种 + 业务场景 + 关键规则词
- 文号（如已知） + 条款号
- 行业 + 税种 + 特殊规则

## 第二步：语义搜索vs精确搜索

**精确搜索**（适合已知文号/条款）：
- 直接输入文号："财税〔2018〕54号"
- 直接输入条款："企业所得税法实施条例第四十三条"
- 输入法规全称关键字段

**语义搜索**（适合不确定文号的场景查询）：
- 描述业务场景："企业购买500万以下设备可以一次性扣除吗"
- 描述问题："关联企业之间借款利息能不能税前扣除"
- 描述条件："高新技术企业亏损可以弥补几年"

**KG图谱查询**（关系型检索）：
- 从实体出发查询关联法规："小型微利企业 → 适用法规 → ?"
- 从法规出发查询适用条件："财税〔2023〕12号 → 适用条件 → ?"
- 从税种出发查询全部优惠："企业所得税 → 优惠政策 → ?"

## 第三步：搜索结果验证

**三步验证法**：
1. **时效验证**：检查法规是否仍然有效（未被废止/修订）
2. **适用验证**：检查客户是否满足法规规定的适用条件
3. **交叉验证**：用不同关键词再搜索一次，确认无遗漏相关法规

## 第四步：常见搜索陷阱

- **同名不同文件**：财政部/税务总局历年公告编号可能重复（如2023年第1号 vs 2022年第1号），务必核对年份
- **废止文件仍被引用**：部分培训资料和网络文章引用已废止的法规，须以官方最新版为准
- **地方口径差异**：同一全国性法规在不同省市的执行口径可能不同，须查找对应地方文件
- **过渡期政策**：注意政策是否有执行期限（如"2023年1月1日至2027年12月31日"）
