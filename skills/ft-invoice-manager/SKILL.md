# 发票管理专家

进项认证/勾选抵扣/数电票(全电发票)/异常发票排查/进项转出判定/留抵退税申报/发票真伪验证。代理记账公司日常接触最多的税务凭证环节，直接影响客户增值税税负和现金流。

## Core Knowledge

### 增值税发票分类

| 发票类型 | 是否可抵扣 | 认证方式 | 备注 |
|---|---|---|---|
| 增值税专用发票(纸质) | 可抵扣 | 勾选平台认证 | 逐步淘汰中 |
| 增值税电子专用发票 | 可抵扣 | 勾选平台认证 | 与纸质专票同等效力 |
| 数电票(全电发票) | 可抵扣 | 自动归集，需勾选确认 | 2024年起全面推广 |
| 增值税普通发票 | 一般不可抵扣 | 不需认证 | 农产品收购发票例外 |
| 机动车销售统一发票 | 可抵扣 | 勾选平台认证 | 仅首次购置可抵扣 |
| 通行费电子发票 | 可抵扣 | 勾选平台认证 | ETC充值发票不可抵扣 |
| 旅客运输凭证 | 可计算抵扣 | 无需认证，手工计算 | 火车票9%/飞机票9%/其他3% |

### 进项抵扣规则

- 认证期限：2020年3月1日起取消360天认证期限限制，可在任意时间勾选认证
- 不得抵扣进项的6种情形：
  1. 用于简易计税/免税项目
  2. 用于集体福利/个人消费
  3. 非正常损失的购进货物及相关应税劳务
  4. 非正常损失的在产品/产成品所耗用的购进货物
  5. 购进的贷款服务、餐饮服务、居民日常服务、娱乐服务
  6. 财政部和国家税务总局规定的其他情形

### 进项转出计算

- 直接计算法：进项税额 x 不得抵扣比例
- 还原计算法：不含税价格 x 适用税率(用于无法准确区分的情况)
- 常见转出场景：兼营免税+应税项目、存货非正常损失、改变用途(如从生产转为员工福利)

### 留抵退税条件

- 增量留抵(一般行业)：连续6个月增量留抵 > 0，且第6个月增量留抵 >= 50万元
- 存量留抵：2022年大规模留抵退税政策已一次性退还大部分行业存量留抵(以2019年3月31日为基准)
- 前提条件：纳税信用等级A/B级、36个月内无骗取留抵退税/出口退税/虚开发票、36个月内无因偷税被处罚2次及以上
- 退税公式：允许退还的增量留抵税额 = 增量留抵税额 x 进项构成比例 x 100%(2022年起，此前为60%)

### 发票验证平台

- 全国增值税发票查验平台：https://inv-veri.chinatax.gov.cn
- 电子税务局-发票勾选确认：增值税发票综合服务平台
- 数电票查验：电子发票服务平台自动归集

## Decision Framework

| 场景 | 处理方式 | 关键判断 |
|---|---|---|
| 收到专票 | 勾选平台认证 -> 抵扣 | 先判断是否属于6种不得抵扣情形 |
| 收到数电票 | 电子税务局确认 -> 抵扣 | 无需认证但必须勾选确认 |
| 需要红冲 | 先开红字信息表 -> 再开红字发票 | 买方/卖方发起流程不同 |
| 失控/异常发票 | 暂停抵扣 -> 查明原因 -> 按税局通知处理 | 不要自行转出，等税局正式通知 |
| 出差票据 | 按运输类型计算抵扣(9%/3%) | 需要是国内运输，国际段不可抵扣 |
| 留抵退税 | 判断条件 -> 电子税务局申请 | 不能和出口退税同一笔进项重复享受 |
| 农产品收购 | 计算抵扣9% | 不需要取得专票，凭收购发票即可 |

## Gotchas

1. **数电票不需要认证但必须勾选确认** -- 数电票(全电发票)会自动归集到电子税务局，很多会计以为"自动归集=自动抵扣"，实际仍需在增值税发票综合服务平台勾选确认后才能抵扣。不确认的后果是申报表自动带出的进项税额为0。

2. **农产品收购发票可以计算抵扣9%** -- 从农业生产者手中购入自产农产品，凭收购发票直接按买价x9%计算进项税额。不需要取得专票。如果用于生产13%税率产品(如肉类加工)，还可以加计抵扣1%，即按10%计算。很多代账公司漏算这笔进项。

3. **火车票/飞机票可以计算抵扣进项税** -- 2019年4月1日起，员工出差的国内旅客运输凭证可以计算抵扣进项税。火车票按票面金额/(1+9%)x9%计算，飞机票按(票价+燃油附加费)/(1+9%)x9%计算。很多中小企业完全不知道这个政策，每年少抵扣数千到数万元。

4. **失控发票的后果不只是税款** -- 一张失控发票被查补，需要补缴已抵扣的进项税额 + 滞纳金(每日万分之五，按天计算) + 罚款(偷税认定的话是不缴或少缴税款50%以上5倍以下，即0.5x-5x)。滞纳金从发票认证当月起算，拖得越久越贵。发现异常发票必须第一时间处理。

5. **留抵退税和出口退税不能同时享受同一笔进项** -- 如果企业既有内销留抵又有出口退税，同一笔进项税额只能选择一种方式退还。实务中需要分别计算内销留抵退税额和出口免抵退税额，注意申报顺序：先办理免抵退税，再办理留抵退税。

6. **红字发票必须先开红字信息表** -- 不能直接作废重开。流程：申请红字信息表(买方or卖方发起) -> 税局审核通过 -> 开具红字发票 -> 买方做进项转出/卖方冲减销项。数电票的红冲流程简化了，但仍需通过电子税务局发起。

7. **ETC充值发票vs通行费发票混淆** -- ETC充值时取得的是"不征税"发票(税率栏为"不征税")，不可抵扣进项。只有实际通行后取得的通行费电子发票(左上角标注"通行费")才能抵扣。很多企业把充值发票当通行费发票抵扣，被查补时才发现。

8. **跨年发票的进项抵扣时效** -- 虽然2020年起取消了360天认证期限，但实务中跨年度的进项发票可能触发税务风险预警(特别是金额较大的)。建议当月认证当月抵扣，避免不必要的税务关注。

## When to Escalate

- 收到税局发票协查函：需要人工配合提供完整的业务链证据(合同+付款+物流+入库)
- 虚开发票嫌疑：涉及刑事风险，必须停止操作并建议客户咨询律师
- 跨境服务发票：境外供应商的发票涉及代扣代缴增值税，需要税务师专项处理
- 大额留抵退税(>100万)：税局会实地核查，需要提前准备完整的进项凭证链
- 发票系统技术故障：税控设备/电子税务局系统问题，联系税控服务商或12366
