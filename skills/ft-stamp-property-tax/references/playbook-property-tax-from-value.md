# 场景：房产税从价计征

**政策依据**：《中华人民共和国房产税暂行条例》第三条、第四条；财税〔2010〕121号

## 第一步：确定计税依据——房产余值

从价计征适用于自用房产（非出租）。计税公式：应纳税额 = 房产原值 ×（1 - 扣除比例）× 1.2%

扣除比例由各省自行确定（10%-30%），常见标准：北京/上海30%、广东/浙江30%、江苏20%-30%（各市不同）。

**房产原值包含的内容**（财税〔2010〕121号）：
- 房屋建造成本/购置价款
- 与房屋不可分割的附属设施（中央空调、电梯、消防设施等）
- 地价：宗地容积率低于0.5的，按房产建筑面积的2倍计算土地面积对应的地价计入原值；容积率≥0.5的，将全部地价并入房产原值

这是常见易错点：地价必须计入房产原值，很多企业遗漏导致少缴税。

## 第二步：特殊情况的原值确定

**改扩建**：房屋改扩建后增加的价值应调增房产原值。装修费用如果属于"不可分割的附属设备"则计入原值，独立可拆卸的装修不计入。

**融资租入房产**：以融资租赁合同约定的总价款（含利息）作为房产原值。

**投资性房地产**：按投资入账金额确定房产原值，后续公允价值计量不影响房产税原值。

**无偿使用他人房产**：使用人为纳税义务人，按房产余值从价计征。

## 第三步：纳税义务发生时间

按条例第七条及相关文件：
- 购置新建商品房：自交付使用之次月起
- 购置存量房：自办理房屋权属转移变更登记手续、取得房产证之次月起
- 自建房屋：自建成之次月起（以竣工验收备案为准）
- 委托施工房屋：从办理验收手续之次月起
- 在建工程提前使用：从使用之次月起

## 第四步：申报与常见风险点

房产税按年征收、分期缴纳（一般半年或按季）。各地申报期限不同，需确认当地规定。

常见风险点：
- 遗漏地价入房产原值（最高频稽查项目）
- 新购/新建房产未及时申报
- 房屋改扩建后未调增原值
- 免税项目（如自用办公）转为应税用途（如出租）未及时变更
- 部分自用部分出租的房产未分别计税

关键输出：房产原值计算表（含地价分摊）+ 房产清单 + 纳税义务时间确认 + 年度应纳税额计算。
