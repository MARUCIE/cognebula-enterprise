# 场景：业务重组涉税事项

**政策依据**：财税〔2009〕59号（企业重组业务企业所得税处理若干问题）；国家税务总局公告2010年第4号；财税〔2016〕36号

## 第一步：重组类型识别

| 重组类型 | 定义 | 常见场景 |
|----------|------|----------|
| 债务重组 | 债务人发生财务困难，债权人让步 | 以资抵债/债转股/减免债务 |
| 股权收购 | 一方购买另一方股权达控制 | 股权转让/增资扩股 |
| 资产收购 | 一方购买另一方实质经营性资产 | 业务转让/资产剥离 |
| 合并 | 两家以上企业合为一家 | 吸收合并/新设合并 |
| 分立 | 一家企业分为两家以上 | 存续分立/新设分立 |

## 第二步：一般性税务处理vs特殊性税务处理

**一般性税务处理**（默认）：
- 被收购/合并/分立企业在重组日按公允价值确认资产转让所得/损失
- 受让方以公允价值为计税基础
- 当期确认所得缴纳所得税

**特殊性税务处理**（递延纳税，需同时满足5个条件）：
1. 具有合理的商业目的，不以减免税为主要目的
2. 收购/合并/分立资产或股权比例符合规定（股权收购≥50%，资产收购≥50%）
3. 重组后12个月内不改变重组资产原来的实质性经营活动
4. 重组交易对价中股权支付比例≥85%
5. 重组后12个月内取得股权的股东不转让所取得的股权

**核心好处**：暂不确认资产转让所得，递延到未来处置时才缴税。

## 第三步：各税种处理要点

| 税种 | 处理要点 |
|------|----------|
| 企业所得税 | 一般处理当期征税；特殊处理递延征税 |
| 增值税 | 整体资产转让（全部/部分实质经营性资产+债权债务+劳动力）不征增值税 |
| 土地增值税 | 企业合并/分立暂不征土地增值税（需满足条件） |
| 契税 | 企业合并/分立承受土地房屋免征契税（特定条件下） |
| 印花税 | 新设/变更的营业账簿按实缴资本增加部分缴纳 |

## 第四步：风险评估

- **备案要求**：享受特殊性税务处理须在年度汇算清缴时提交《企业重组所得税特殊性税务处理报告表》
- **12个月锁定期**：特殊处理后12个月内违反条件的（如转让股权/改变经营），需补缴递延的税款
- **亏损弥补**：合并中被合并企业的亏损可由合并企业继承弥补（有限额限制）
- **评估风险**：资产评估增值可能导致重大税负，重组前必须做全面税务测算
