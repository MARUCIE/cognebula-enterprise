# 场景：新业务类型税务评估

**政策依据**：《增值税暂行条例》；财税〔2016〕36号；《企业所得税法》及实施条例

## 第一步：新业务类型税务画像

收到客户新业务咨询时，按以下维度建立税务画像：

| 维度 | 问题 | 影响 |
|------|------|------|
| 交易性质 | 销售货物/提供服务/转让资产/混合 | 决定增值税税率 |
| 交易对手 | 境内/境外，个人/企业 | 决定代扣义务和税收协定适用 |
| 收入确认 | 一次性/分期/持续性 | 决定纳税义务发生时间 |
| 成本结构 | 可取得进项发票的比例 | 决定税负率 |
| 税收优惠 | 是否属于优惠目录行业/项目 | 决定实际税负 |
| 合规要求 | 是否需要特殊资质/许可 | 决定经营范围是否需要变更 |

## 第二步：税种影响分析

**标准分析模板**：

```
新业务：[业务描述]
1. 增值税
   - 税目：□货物 □服务 □无形资产 □不动产
   - 税率：□13% □9% □6% □免税 □零税率
   - 进项来源：[主要进项构成及可抵扣比例]
   - 预估税负率：[X%]

2. 企业所得税
   - 收入确认方式：[时点/时段]
   - 成本扣除要点：[特殊扣除规则]
   - 可享受优惠：[高新/小微/加计扣除等]

3. 其他税种
   - 印花税：[合同类型及税率]
   - 个人所得税：[涉及劳务报酬/分红等]
   - 附加税费：[城建税+教育费附加]
```

## 第三步：税负测算

**简易测算模型**：
```
预估年收入 = A
预估可抵扣进项 = B
增值税税负 = (A × 适用税率 - B) ÷ A × 100%
企业所得税税负 = (A - 成本 - 费用) × 25% ÷ A × 100%
综合税负率 = 增值税税负 + 企业所得税税负 + 附加税费
```

**与行业均值比对**：测算结果与同行业税负率比对，偏离过大需复查假设。

## 第四步：评估结论

输出新业务税务评估报告，包含：
- 税种清单及预估税负
- 经营范围是否需要变更
- 发票票种是否需要调整
- 是否涉及特殊资质或许可
- 合规风险提示
- 税收筹划建议（合法范围内的节税路径）
