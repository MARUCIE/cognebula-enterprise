# 场景六：委外加工成本处理

**政策依据**：《企业会计准则第1号——存货》第8条；《增值税暂行条例》第1条（委托加工物资）；《消费税暂行条例》第4条第2款（委托加工消费品）

## 第一步：初步判别——委外加工的定义与范围

**委外加工**：企业将材料发给外单位加工，外单位按企业要求加工完成后收回成品或半成品

**判定要素**：
1. 原材料由委托方提供（区别于外购成品）
2. 受托方按委托方的技术要求加工
3. 加工完成后由委托方收回
4. 委托方支付加工费

**会计科目**：委托加工物资（资产类）

## 第二步：深入分析——成本归集与核算

**委托加工物资成本构成**：

| 成本项目 | 内容 | 会计处理 |
|---------|-----|---------|
| 发出材料成本 | 委托方发出的原材料实际成本 | 借：委托加工物资；贷：原材料 |
| 加工费 | 受托方收取的加工费 | 借：委托加工物资；贷：应付账款 |
| 运杂费 | 发出和收回的运输费 | 借：委托加工物资；贷：银行存款 |
| 消费税（如涉及） | 受托方代收代缴 | 收回后直接销售→计入成本；连续生产→可抵扣 |
| 增值税 | 加工费的进项税 | 一般纳税人可抵扣进项 |

**完整分录流程**：
```
1. 发出材料：
   借：委托加工物资 100,000
     贷：原材料 100,000

2. 支付加工费（含增值税13%）：
   借：委托加工物资 50,000
   借：应交税费—进项税 6,500
     贷：应付账款 56,500

3. 收回入库（用于继续生产）：
   借：原材料/库存商品 150,000
     贷：委托加工物资 150,000
```

## 第三步：佐证/验证——特殊事项处理

**消费税处理**（委托加工应税消费品）：
- **收回后直接销售**：代收代缴的消费税计入委托加工物资成本
- **收回后继续加工**：代收代缴的消费税可以抵扣（借记应交税费—消费税）
- **组成计税价格** = (材料成本 + 加工费) ÷ (1 - 消费税率)

**加工损耗处理**：
- 正常损耗：计入委托加工物资成本（不单独处理）
- 非正常损耗：受托方赔偿或计入营业外支出
- 需在合同中明确正常损耗率

**质量问题**：
- 加工质量不达标退回返工：返工成本由受托方承担
- 须保留质检记录和不合格报告

## 第四步：结论/建议

- **合同明确**：委外加工合同应明确加工费、交期、质量标准、损耗率、验收方式
- **成本对比**：定期比较委外加工成本 vs 自制成本，评估经济性
- **发票管理**：加工费发票与发出材料的品名/数量须逻辑匹配
- **存货盘点**：委外在途物资（已发出未收回）纳入期末存货盘点范围
- **税务风险**：虚假委外加工（无实际加工行为）可能被认定为虚开发票
