# 场景四：行政处罚裁量基准

**政策依据**：《税收征收管理法》第60-73条；《行政处罚法》（2021年修订）；各省税务局行政处罚裁量基准（各省独立制定）

## 第一步：初步判别——处罚类型与裁量空间

1. **法定罚款幅度**：税法规定的是"可以处以"还是"应当处以"？处[下限]以上[上限]以下罚款
2. **裁量基准层级**：国家税务总局指导 → 各省局制定具体标准
3. **从轻/减轻/从重情形**：直接影响罚款在法定幅度中的位置

**常见处罚条款与裁量区间**：
| 违法行为 | 法条 | 法定幅度 |
|---------|-----|---------|
| 未按期办理税务登记 | 第60条 | 2,000以下；情节严重2,000-10,000 |
| 未按期申报 | 第62条 | 2,000以下；情节严重2,000-10,000 |
| 偷税 | 第63条 | 不缴少缴税款50%-5倍罚款 |
| 编造虚假计税依据 | 第63条 | 50,000以下 |
| 逃避追缴欠税 | 第65条 | 欠缴税款50%-5倍罚款 |

## 第二步：深入分析——裁量基准三档制

多数省份采用"三档制"裁量：

**以偷税罚款（50%-5倍）为例**：
| 档次 | 裁量基准 | 适用情形 |
|-----|---------|---------|
| 从轻 | 0.5-1倍 | 首次违法、主动补缴、配合检查、违法金额小 |
| 一般 | 1-3倍 | 无从轻/从重情形 |
| 从重 | 3-5倍 | 拒不配合、伪造证据、情节严重、再犯 |

**《行政处罚法》第33条（首违不罚）**：
初次违法且危害后果轻微并及时改正的，可以不予行政处罚。
→ 部分省份已将此落实为"首违不罚清单"

## 第三步：佐证/验证——争取从轻/减轻的策略

1. **主动补缴**：发现问题后在税务机关通知前主动补缴 → 最有力的从轻情节
2. **配合调查**：全面配合税务检查，提供完整资料 → 从轻考量
3. **及时改正**：违法行为发现后立即整改，建立合规制度 → 减轻依据
4. **首次违法**：无以往同类违法记录 → 可能适用首违不罚
5. **经济困难**：生产经营严重困难 → 可申请分期缴纳罚款

**从重情形（须警惕）**：
- 拒绝/阻挠税务检查
- 伪造、变造、隐匿账簿凭证
- 对举报人实施打击报复
- 在税务机关责令改正后仍不改正
- 同一违法行为再犯

## 第四步：结论/建议

- **查阅当地基准**：各省裁量基准差异较大，务必查阅主管税务机关所在省的裁量文件
- **从轻证据准备**：收到《税务行政处罚事项告知书》后，立即整理从轻情节证据
- **陈述申辩权**：处罚前必有告知和陈述申辩环节，应充分行使（不影响处罚加重）
- **听证申请**：罚款金额较大的（各省标准不同，通常>10万），可申请听证
- **罚款独立性**：罚款与补税是独立的，缴纳罚款不等于认可补税金额
