# 场景五：碳排放配额会计处理

**政策依据**：财政部《碳排放权交易有关会计处理暂行规定》（财会〔2019〕22号）

## 第一步：初步判别——碳排放配额的会计定性

**碳排放配额**：政府分配给重点排放单位的排放权，允许在规定期间内排放一定量的温室气体。

**会计科目**：
- **碳排放权资产**（新增资产类科目）：用于核算通过碳排放权交易机制取得的碳排放配额
- 不作为金融资产、无形资产或存货处理

## 第二步：深入分析——全流程会计处理

**1. 无偿取得配额（政府免费分配）**
```
借：碳排放权资产  0元（名义金额/备查登记）
  贷：（不做账务处理，仅做备查登记）
```
→ 免费分配的配额不确认资产价值，仅做数量登记

**2. 有偿取得配额（拍卖/购买）**
```
借：碳排放权资产  XX元（购买价格）
  贷：银行存款/其他应付款  XX元
```

**3. 履约时（使用配额抵消排放）**
```
借：营业外支出—碳排放配额  XX元
  贷：碳排放权资产  XX元
```
→ 按加权平均法或先进先出法结转成本

**4. 出售配额（市场交易）**
```
借：银行存款  XX元
  贷：碳排放权资产  XX元（账面价值）
  贷：营业外收入  XX元（差额，如有收益）
```

**5. 配额不足时购买（补足履约缺口）**
```
借：碳排放权资产  XX元
  贷：银行存款  XX元
→ 随后用于履约：
借：营业外支出  XX元
  贷：碳排放权资产  XX元
```

## 第三步：佐证/验证——税务处理

**企业所得税**：
- 免费取得配额：不确认收入（零成本取得）
- 购买配额用于履约：购买支出可税前扣除
- 出售配额取得收益：确认应税收入
- 配额到期作废：已购买的未使用配额的损失可否扣除？→ 目前无明确规定，建议按资产损失申报

**增值税**：
- 碳配额交易目前在全国碳市场中暂不征收增值税（政策观望中）
- 区域碳市场部分地区有征税规定

## 第四步：结论/建议

- **备查登记**：免费配额虽不入账但须做详细的数量登记台账
- **市场关注**：关注碳配额价格趋势，决定出售/持有策略
- **税务政策跟踪**：碳交易税务政策仍在完善中，密切关注后续文件
- **履约规划**：提前评估年度排放量与配额缺口，决定购买时点
- **碳资产管理**：有条件的企业设立碳资产管理岗位，统筹配额获取/交易/履约
