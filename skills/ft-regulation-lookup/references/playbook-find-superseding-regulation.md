# 场景：查找替代/修订法规

**政策依据**：税收法规的废止替代机制；《税收规范性文件制定管理办法》

## 第一步：替代关系类型

| 类型 | 特征 | 查找方法 |
|------|------|----------|
| 全文替代 | 新文件明确废止旧文件 | 新文件末尾"本通知自X日起执行，XX号文同时废止" |
| 部分修订 | 新文件修改旧文件的特定条款 | 修订公告逐条列明修改内容 |
| 整合替代 | 多个旧文件被一个新文件统一替代 | 新文件附件列出全部废止文件清单 |
| 延续替代 | 原有效期到期，延续公告更新有效期 | 延续公告引用原文件号 |

## 第二步：查找替代法规的方法

**方法一：顺藤摸瓜法**
1. 在旧法规的官方法规库页面查看标注（通常标注"已废止"或"已修改"）
2. 点击废止/修改标注跳转到新法规
3. 阅读新法规，确认与旧法规的对应关系

**方法二：反向搜索法**
1. 搜索"废止 [旧文件号]"或"修改 [旧文件号]"
2. 搜索"替代 [旧文件关键词] [最新年份]"
3. 在国家税务总局"废止及失效文件目录"中查找

**方法三：上位法追溯法**
1. 确认旧法规的上位法（如条例→实施细则→具体公告）
2. 检查上位法是否已修订
3. 上位法修订后，查找配套的新实施文件

**方法四：KG关系查询**
```
查询：[旧法规] → 被替代为 → ?
查询：[旧法规条款] → 现行对应 → ?
```

## 第三步：替代关系映射示例

| 旧法规 | 替代法规 | 替代类型 |
|--------|----------|----------|
| 国税发〔2009〕2号（特别纳税调整） | 国家税务总局公告2016年第42号（关联申报） | 部分替代 |
| 财税〔2015〕119号（研发费加计） | 财政部 税务总局公告2023年第7号（比例调整） | 部分修订 |
| 《增值税暂行条例》（2008版） | 《增值税暂行条例》（2017修订版） | 全文修订 |
| 财税〔2018〕54号（一次性扣除） | 各年延续公告 | 延续替代 |

## 第四步：管理建议

- 在KG中为每个法规节点建立"替代关系"属性（被替代→替代者）
- 发现废止法规时，同步标记并添加替代法规链接
- 对高频使用的法规建立"法规演变链"（从最早版本到最新版本的完整历程）
- 每月关注财政部/税务总局新发公告中的废止清单
