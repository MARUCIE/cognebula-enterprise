# 场景：非同一控制下企业合并

**政策依据**：《企业会计准则第20号——企业合并》第十至十六条

## 第一步：确认合并日与合并成本

非同一控制下合并采用"购买法"，以购买日为基准。购买日的判断（准则第十一条）：同时满足合同/协议已获批准、办理了必要的财产权交接手续、支付了大部分对价、购买方实际控制了被购买方的财务和经营政策。

合并成本包括：购买方支付的现金、转让的非现金资产公允价值、发行或承担的债务公允价值、发行的权益性证券公允价值。审计费、法律费、评估费等直接费用不计入合并成本，计入当期损益。发行权益性证券的手续费冲减资本公积。

## 第二步：被购买方可辨认净资产公允价值计量

在购买日，对被购买方的可辨认资产和负债按公允价值重新计量（准则第十四条）。常见调整项：
- 存货：从账面成本调至可变现净值或市场价
- 固定资产/投资性房地产：按评估价值入账
- 无形资产：品牌、客户关系、专利等需单独确认（即使被购买方未入账）
- 或有负债：在购买日公允价值能可靠计量的，确认为负债
- 递延所得税：因公允价值调整产生的暂时性差异，确认递延所得税

关键要求：评估报告的基准日应为购买日或尽可能接近购买日。

## 第三步：商誉计算与确认

商誉 = 合并成本 - 购买方在被购买方可辨认净资产公允价值中享有的份额。

如果合并成本 < 享有份额（即"负商誉"），应首先复核合并成本和可辨认净资产公允价值的计量是否正确，经复核后仍有差额的，计入当期营业外收入（准则第十五条）。

商誉不摊销，但每年年末需进行减值测试（《资产减值准则》第二十三条），减值后不得转回。将商誉分配至相关资产组或资产组组合进行减值测试。

## 第四步：合并报表编制与后续处理

合并利润表：仅包含被购买方自购买日起至期末的收入和费用（与同一控制不同，不追溯期初）。购买日子公司所有者权益各项目以公允价值为基础确认，后续年度在此基础上连续调整。

购买价格分配（PPA）通常在购买日后12个月内完成（计量期调整）。计量期内取得的新信息如果表明购买日已存在的事实，可以追溯调整。超过12个月或非购买日已存在的事实，计入当期损益。

关键输出：购买日确认底稿 + 合并成本明细 + PPA公允价值评估 + 商誉计算 + 合并分录。
