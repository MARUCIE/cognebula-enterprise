# 场景：租赁会计处理（新准则）

**政策依据**：《企业会计准则第21号——租赁》（财会〔2018〕35号，2019年1月1日起施行）

## 第一步：判别租赁分类

新准则下承租人不再区分经营租赁和融资租赁，统一采用使用权资产模型。但可选用以下两项简化处理：
- **短期租赁豁免**：租赁期不超过12个月（且无购买选择权） → 直接费用化
- **低价值资产豁免**：单项租赁资产全新时价值不超过4万元（如打印机、小型办公设备） → 直接费用化

**决策规则**：代账企业中，租赁办公室通常不满足豁免条件（租期多为1-5年），必须确认使用权资产和租赁负债。

## 第二步：初始计量

**使用权资产初始成本** = 租赁负债初始金额 + 预付租金 + 初始直接费用 + 复原成本估计

**租赁负债初始金额** = 尚未支付的租赁付款额的现值

折现率选择优先顺序：
1. 租赁内含利率（如能确定）
2. 承租人增量借款利率（实务中最常用，参考同期银行贷款利率）

## 第三步：后续计量

每月会计分录：
```
借：财务费用（租赁负债×折现率/12）
    贷：租赁负债——未确认融资费用
借：管理费用——折旧费（使用权资产/租赁期）
    贷：使用权资产累计折旧
借：租赁负债——租赁付款额
    贷：银行存款
```

**使用权资产折旧**：按租赁期与资产剩余使用寿命的较短者计提。期满所有权不转移的，在租赁期内折旧完毕。

## 第四步：风险评估

- **中小企业过渡**：执行小企业会计准则的企业暂不强制执行新租赁准则，仍可按原经营租赁处理（直接费用化）。但执行企业会计准则的必须执行
- **税务处理**：税法上经营性租赁仍按实际支付的租金费用化扣除（《企业所得税法实施条例》第四十七条），与会计处理产生临时性差异，需确认递延所得税
- **常见错误**：免租期未分摊（应将总租金在含免租期的整个租赁期内均匀分摊）；租赁变更未重新计量租赁负债
