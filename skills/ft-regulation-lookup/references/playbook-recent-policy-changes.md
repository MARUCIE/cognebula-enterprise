# 场景：近期政策变动追踪

**政策依据**：财政部/国家税务总局定期发布的政策公告

## 第一步：政策追踪渠道

| 渠道 | 更新频率 | 内容 |
|------|----------|------|
| 国家税务总局官网 | 实时 | 总局公告/解读/政策问答 |
| 财政部税政司 | 实时 | 财税政策/部门规章 |
| 12366纳税服务热线 | 工作日 | 政策咨询/口径确认 |
| 各省税务局官网 | 每周 | 地方执行口径/操作指南 |
| 税务总局微信公众号 | 每日 | 政策速递/热点解读 |
| 中国税务报 | 每日 | 深度解读/案例分析 |

## 第二步：政策分类与优先级

**按影响面分级**：
| 级别 | 类型 | 响应时间 |
|------|------|----------|
| P0 | 税率变动/新税种/重大优惠调整 | 发布当日学习并通知客户 |
| P1 | 申报规则变更/发票管理变更 | 3日内学习并调整操作 |
| P2 | 操作流程优化/系统升级 | 1周内了解 |
| P3 | 解读类/案例类/培训类 | 有空时学习 |

**年度关键政策节点**：
| 时间 | 关注事项 |
|------|----------|
| 1月 | 新年度税收优惠延续/到期政策汇总 |
| 3月 | 全国两会税制改革方向/汇算清缴启动 |
| 5月 | 汇算清缴截止/年度政策汇编发布 |
| 7月 | 社保基数调整/半年度政策梳理 |
| 9-10月 | 第四季度预缴/年终奖筹划窗口 |
| 12月 | 年度政策到期前的延续公告密集发布 |

## 第三步：政策变动影响评估

收到新政策后按以下模板评估：

```
政策名称：[文号+标题]
发布日期：[YYYY-MM-DD]
生效日期：[YYYY-MM-DD]
影响税种：[增值税/所得税/...]
影响范围：[所有企业/特定行业/特定规模]
核心变化：[简要说明变了什么]
对客户影响：[哪些客户受影响/影响程度]
操作调整：[需要修改哪些操作流程]
通知优先级：[P0/P1/P2/P3]
```

## 第四步：知识沉淀

- 新政策发布后48小时内更新知识图谱中对应的法规节点
- 标注旧法规状态（废止/修订/延续）
- 更新受影响Skill中的政策引用
- 对P0/P1级政策，生成客户通知摘要
- 每季度整理一次"季度税收政策变动汇总"
