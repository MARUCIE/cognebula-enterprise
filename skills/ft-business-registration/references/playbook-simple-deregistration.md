# 场景：简易注销条件与流程

**政策依据**：《市场主体登记管理条例》第三十三条至第三十六条；国家市场监管总局关于全面推进企业简易注销登记改革的指导意见

## 第一步：判断是否适用简易注销

**适用条件**（同时满足）：
1. 未发生债权债务，或已将债权债务清算完结
2. 全体投资人书面承诺
3. 领取营业执照后未开展经营活动（未开业）或者开业后无债权债务的

**不适用简易注销的情形**：
- 存在未结清的税款、罚款、滞纳金
- 涉及司法案件/行政处罚尚未结案
- 股东/出资人无法联系或拒不配合
- 被列入经营异常名录且未移出
- 被列入严重违法失信名单
- 存在股权被质押或冻结

## 第二步：简易注销流程

1. **全体投资人签署承诺书**：承诺企业无债权债务并愿意承担法律责任
2. **公示期**：在国家企业信用信息公示系统发布简易注销公告（公示期20天，2023年起从45天缩短）
3. **公示期满**：无异议的，向市场监管局提交注销申请
4. **提交材料**：注销申请书+全体投资人承诺书+营业执照正副本
5. **核准注销**：市场监管局1-3个工作日内核准

**税务处理**：简易注销中，市场监管局将与税务部门信息共享核查。税务状态异常的将被驳回。部分城市要求先完成税务注销。

## 第三步：税务简易注销条件

| 条件 | 说明 |
|------|------|
| 未办理过涉税事宜 | 直接免办税务注销 |
| 办理过涉税事宜但未领用发票且无欠税 | 可免办清税证明 |
| 已缴销发票、无欠税、无未结事项 | 可申请即时办结清税 |

**注意**：如有未申报的税款或已领用但未缴销的发票，须先处理完毕才能注销。

## 第四步：风险提示

- **虚假承诺责任**：全体投资人签署的承诺书具有法律效力。如果实际存在债权债务但签署了"无债权债务"承诺，债权人可要求投资人承担清偿责任
- **注销后债务**：简易注销后被发现有未清偿债务的，投资人在承诺范围内承担无限连带责任
- **撤销注销**：利害关系人可申请撤销注销登记
- **建议**：即使满足简易注销条件，也建议委托会计师进行简单的税务清算确认，避免遗漏
