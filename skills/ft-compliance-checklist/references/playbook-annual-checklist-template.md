# 场景三：年度合规检查模板

**政策依据**：《企业所得税法》第54条（汇算清缴）；《税收征收管理法》第25条

## 第一步：初步判别——年度检查的全面性要求

年度检查是最全面的税务健康体检，覆盖所有税种和全年周期，为汇算清缴做准备。

## 第二步：深入分析——年度检查清单


## 第三步：佐证/验证——年度检查报告

包含：全年指标汇总、纳税调整明细、优惠享受情况、问题清单和整改措施。

## 第四步：结论/建议

- **1月启动**：年度检查从次年1月开始，3月底前完成
- **审计协调**：与年度财务审计同步进行，数据口径一致
- **调整底稿**：所有纳税调整事项须有底稿和政策依据
- **专业复核**：建议委托税务师对重大调整事项出具鉴证报告
- **问题闭环**：上年度检查发现的问题本年度是否已整改
