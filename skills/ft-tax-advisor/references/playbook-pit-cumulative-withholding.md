# 场景：个税累计预扣法

**政策依据**：《个人所得税法》第十一条；国家税务总局公告2018年第61号；国家税务总局公告2020年第13号

## 第一步：理解累计预扣法原理

自2019年起，居民个人取得工资薪金所得，扣缴义务人按月采用累计预扣法计算预扣税款。

**核心公式**：
```
本期应预扣预缴税额 = (累计预扣预缴应纳税所得额 × 预扣率 - 速算扣除数) - 累计减免税额 - 累计已预扣预缴税额

累计预扣预缴应纳税所得额 = 累计收入 - 累计免税收入 - 累计减除费用 - 累计专项扣除 - 累计专项附加扣除 - 累计依法确定的其他扣除
```

其中：累计减除费用 = 5000元/月 × 当年截至本月的月数

## 第二步：预扣率表（综合所得适用）

| 累计预扣预缴应纳税所得额 | 预扣率 | 速算扣除数 |
|-------------------------|--------|-----------|
| ≤ 36,000 | 3% | 0 |
| 36,000-144,000 | 10% | 2,520 |
| 144,000-300,000 | 20% | 16,920 |
| 300,000-420,000 | 25% | 31,920 |
| 420,000-660,000 | 30% | 52,920 |
| 660,000-960,000 | 35% | 85,920 |
| > 960,000 | 45% | 181,920 |

**特征**：年初几个月预扣税额少，下半年随累计所得增加而跳档，预扣额逐月增大。这是正常现象。

## 第三步：扣除项目清单

**专项扣除**（强制缴纳）：养老保险、医疗保险、失业保险、住房公积金

**专项附加扣除**（7项）：
1. 子女教育：2000元/月/孩（2023年起提标）
2. 继续教育：400元/月（学历）或3600元/年（职业资格）
3. 大病医疗：年度自付超15000元部分，限额80000元（汇算清缴时扣）
4. 住房贷款利息：1000元/月（首套，最长240个月）
5. 住房租金：800/1100/1500元/月（按城市规模分档）
6. 赡养老人：3000元/月（2023年起提标）
7. 3岁以下婴幼儿照护：2000元/月/孩（2023年起提标）

## 第四步：风险提示

- **扣缴义务**：扣缴义务人必须按月办理全员全额扣缴申报，未按规定扣缴的，处应扣未扣税款50%-3倍罚款（《税收征管法》第六十九条）
- **年度汇算**：取得两处以上工资、专项附加扣除未享受完整、年收入超12万元且补税超400元的，须办理年度汇算清缴（次年3月1日-6月30日）
- **首份工作优化**：当年首次取得工资的新入职员工，扣缴义务人可按实际工作月份数计算累计减除费用（国家税务总局公告2020年第13号）
- **常见错误**：跨公司任职未合并计算；专项附加扣除重复享受；离职月份漏扣
