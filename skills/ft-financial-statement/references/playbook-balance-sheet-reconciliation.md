# 场景：资产负债表勾稽检查

**政策依据**：《企业会计准则——基本准则》第二十条至第三十七条；《会计基础工作规范》

## 第一步：基本平衡公式验证

**核心恒等式**：资产总额 = 负债总额 + 所有者权益总额

**不等则说明**：科目分类错误/漏记/重复记账/计算错误。必须在此步骤全部排查通过后才能继续。

## 第二步：报表间勾稽关系

| 勾稽关系 | 验证公式 | 不等原因 |
|----------|----------|----------|
| 资产负债表↔利润表 | 期末未分配利润 = 期初未分配利润 + 本期净利润 - 利润分配 | 利润结转分录有误 |
| 资产负债表↔现金流量表 | 期末货币资金 = 期初货币资金 + 现金净增加额 | 现金流量表编制有误 |
| 利润表↔所得税申报表 | 利润总额应与季度预缴申报表一致 | 申报数据有误 |
| 资产负债表↔增值税申报表 | 应交增值税余额应与申报表应纳税额匹配 | 增值税结转有误 |

## 第三步：科目余额合理性检查

| 科目 | 检查要点 | 异常信号 |
|------|----------|----------|
| 货币资金 | 与银行对账单核对 | 负数余额/与银行严重不符 |
| 应收账款 | 账龄分析 | 大额长期挂账/贷方余额 |
| 存货 | 期末数与盘点核对 | 负数/异常增长 |
| 固定资产 | 与台账核对 | 无折旧变动/净值为负 |
| 应付账款 | 与供应商对账 | 借方余额（实为预付） |
| 其他应收款 | 明细核查 | 股东大额借款/长期挂账 |
| 其他应付款 | 明细核查 | 大额不明款项 |
| 应交税费 | 与申报表核对 | 长期贷方大额余额/负数 |

## 第四步：常见错误与整改

- **应收/应付贷方余额**：应收贷方余额应重分类至预收账款；应付借方余额应重分类至预付账款
- **其他应收款长期挂账**：股东借款超过一年未归还的，税务上视同分红缴纳20%个税
- **存货负数**：发出数量超过入库数量，通常是暂估入库遗漏或发出计价有误
- **应交税费负数**：一般为多缴税或留抵税额，应确认是否正确
- **月度报告**：建议每月出具资产负债表勾稽检查清单，标记异常项及整改状态

**风险提示**：
- 报表间勾稽不平直接影响审计意见，年报审计前必须全部排查通过
- 其他应收款长期挂账（股东借款超1年）将被税务机关视同分红征收20%个税
- 应交税费科目异常（长期负数/长期大额贷方余额）是税务预警的高频触发项

**关联技能协同**：
- 应收账龄异常需评估坏账 --> `ft-accountant` 计提坏账准备（playbook-provision-for-bad-debts）
- 应交税费与申报表不符 --> `ft-tax-advisor` 核对增值税/所得税申报数据
- 股东借款风险 --> `ft-risk-assessment` 评估股东借款税务风险（playbook-shareholder-loan-risk）
- 存货异常 --> `ft-internal-audit` 安排存货盘点审计（playbook-inventory-count-audit）
