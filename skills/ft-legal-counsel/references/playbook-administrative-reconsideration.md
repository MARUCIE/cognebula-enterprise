# 场景二：行政复议程序及材料

**政策依据**：《行政复议法》（2023年修订）；《税务行政复议规则》（国家税务总局令第39号）

## 第一步：初步判别——复议受理条件

1. **申请人资格**：与具体行政行为有利害关系的公民/法人/其他组织
2. **被申请人**：作出行政行为的税务机关
3. **复议机关**：被申请人的上一级税务机关（省级以下）；国家税务总局（省级税务局作出的）
4. **申请期限**：知道行政行为之日起60日内
5. **前置条件**：征税争议须先缴清税款或提供担保

**不予受理情形**：超过申请期限、不属于复议范围、已向法院起诉、申请人不适格

## 第二步：深入分析——复议程序全流程

```
准备阶段（建议15天）
  → 收集证据、起草申请书、评估缴税/担保方案
      ↓
申请阶段
  → 提交复议申请书+证据材料 → 复议机关5日内决定受理
      ↓
审理阶段（60日内，最长延长30日）
  → 书面审查（原则）/ 听取意见 / 调查取证
  → 申请人可补充证据和陈述
      ↓
决定阶段
  → 维持 / 撤销 / 变更 / 确认违法 / 责令履行
```

## 第三步：佐证/验证——申请材料清单

**必备材料**：
1. 《税务行政复议申请书》（载明申请人信息、被申请人、请求事项、事实和理由）
2. 税务机关作出的行政行为文书（税务处理决定书/处罚决定书等）
3. 申请人身份证明（营业执照/身份证复印件）
4. 授权委托书（如委托代理人）
5. 缴税凭证或纳税担保书（征税争议的）

**建议附送材料**：
6. 证据目录及证据材料（按时间顺序编号）
7. 相关法律法规条文索引
8. 类案参考（同类型争议的复议/诉讼案例）
9. 第三方专家意见书（如涉及专业技术问题）

**申请书写作要点**：
- 事实部分：按时间线陈述，附证据编号引用
- 理由部分：逐条反驳税务机关的认定，指出事实错误或法律适用错误
- 请求部分：明确具体（撤销某号决定书 / 减少罚款金额至X元）

## 第四步：结论/建议

- **60日不可突破**：申请期限为不变期间，不可恢复（2023年新复议法有特殊情形除外规定）
- **担保替代缴税**：如现金流紧张，优先考虑不动产抵押或银行保函方式提供担保
- **证据质量**：复议成功率与证据充分性直接相关，重点准备交易真实性、税务处理合规性的证据
- **和解调解**：2023年新复议法增设调解制度，部分自由裁量权范围内的争议可协商解决
- **诉讼衔接**：如复议不利，15日内须决定是否起诉（时效极短，须提前准备诉讼方案）
