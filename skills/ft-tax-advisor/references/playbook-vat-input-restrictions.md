# 场景：不得抵扣进项的6种情形

**政策依据**：《增值税暂行条例》第十条；财税〔2016〕36号附件1第二十七条

## 第一步：明确6种不得抵扣情形

1. **用于简易计税项目**：选择简易计税方法的项目对应进项不得抵扣
2. **用于免征增值税项目**：免税业务对应的进项不得抵扣
3. **用于集体福利或个人消费**：如员工食堂采购、节日福利等
4. **非正常损失的购进货物及其运输**：因管理不善造成的被盗、丢失、霉烂变质，以及违法违规被依法没收、销毁的
5. **非正常损失的在产品、产成品所耗用的购进货物**：同上
6. **贷款服务、餐饮服务、居民日常服务、娱乐服务**：这四类取得的进项税额一律不得抵扣

## 第二步：重点判定规则

**"非正常损失"的严格定义**（财税〔2016〕36号）：
- 仅限于因管理不善造成的被盗、丢失、霉烂变质损失
- 被执法部门依法没收或强令自行销毁的货物
- **不包括**：自然灾害损失、合理损耗、技术淘汰报废

**决策规则**：地震、洪水、火灾（非人为）等自然灾害造成的损失，对应进项税额可以继续抵扣。生产过程中的合理损耗也可以抵扣。

**"集体福利和个人消费"判定**：
- 公司购车用于经营 → 可抵扣
- 公司购车供老板个人使用 → 不得抵扣
- 办公室装修 → 可抵扣
- 员工宿舍装修 → 不得抵扣（集体福利）

## 第三步：兼营情形的进项分摊

企业同时存在可抵扣和不可抵扣业务的，需按比例划分：

**不得抵扣进项 = 当期无法划分的全部进项税额 ×（当期简易计税/免税/集体福利销售额 ÷ 当期全部销售额）**

能够明确归属的进项直接归集，不需要分摊。

## 第四步：风险评估

- **已抵扣需转出**：已经抵扣的进项税额，后续转用于不得抵扣项目的，需在转用当期做进项税额转出
- **固定资产/不动产转用**：不动产已抵扣进项后转用于不得抵扣项目，按净值比例转出（不按原值全额转出）
- **餐饮/娱乐全面禁抵**：即使是商务宴请、客户招待，取得的餐饮/娱乐发票进项也不得抵扣——这是最常见的错误
- **专票不等于可抵扣**：取得专票不代表一定能抵扣。属于上述6种情形的，即使取得专票也必须做进项转出
