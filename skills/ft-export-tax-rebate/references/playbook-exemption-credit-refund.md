# 场景一：免抵退税完整计算

**政策依据**：财税〔2012〕39号；国家税务总局公告2012年第24号；《出口货物劳务增值税和消费税管理办法》

## 第一步：初步判别——是否适用免抵退税

**适用对象**：生产企业自营或委托出口自产货物
- 生产企业：有生产能力（含视同自产货物）
- 一般纳税人
- 出口货物适用退（免）税政策

**不适用**：外贸企业（适用免退税，见playbook-exemption-refund）

## 第二步：深入分析——免抵退税五步计算

**假设**：某生产企业当期内销收入600万，出口收入400万（FOB），征税率13%，退税率10%，上期留抵税额5万，进项税额80万

**Step 1：计算不得免征和抵扣税额**
= 出口FOB × (征税率 - 退税率)
= 400 × (13% - 10%) = 12万

**Step 2：计算当期应纳税额**
= 内销销项 - (进项 - 不得免抵退税额) - 上期留抵
= 600×13% - (80 - 12) - 5
= 78 - 68 - 5 = 5万

**情况A：当期应纳税额≥0（本例=5万）**
→ 当期无退税，免抵税额 = 免抵退税额

**Step 3：计算免抵退税额**
= 出口FOB × 退税率
= 400 × 10% = 40万

**Step 4：比较确定退税额**
- 当期应纳税额5万>0 → 应退税额=0
- 免抵税额 = 免抵退税额 = 40万

**情况B：若当期应纳税额<0（假设=-15万）**
→ 当期期末留抵税额=15万
→ 比较：留抵15万 vs 免抵退40万
→ 应退税额 = min(15, 40) = 15万
→ 免抵税额 = 40 - 15 = 25万

## 第三步：佐证/验证——常见计算错误

1. **征退税率差额**：遗漏"不得免征和抵扣税额"的转出 → 多退
2. **FOB价格**：必须折算为FOB（CIF需扣除运费保费）
3. **汇率**：出口销售额按月初人民银行中间价折算
4. **进料加工**：有进料加工手册的须扣减免税进口料件（另行计算）
5. **视同内销**：超期未收汇的出口按内销征税

## 第四步：结论/建议

- **月度申报**：生产企业免抵退税按月计算申报
- **免抵税额影响**：免抵税额须缴纳城建税和教育费附加（容易遗漏）
- **进项管理**：出口占比高时注意进项与留抵的关系，避免长期大额留抵
- **退税率差额**：密切关注退税率调整公告，征退差额大时评估出口业务税务成本
