# 场景三：年度申报时间线

**政策依据**：《企业所得税法》第54条（汇算清缴5月31日前）；《个人所得税法》第11条（综合所得3月1日-6月30日）

## 第一步：初步判别——年度重要时间节点总览

```
1月份：
  ● 1月15日前：12月份月度申报+Q4季度预缴
  ● 1月31日前：上年度个税专项附加扣除确认
  ● 1月内：上年度工商年报开始准备

2月份：
  ● 2月15日前：1月份月度申报
  ● 春节假期顺延规则关注

3月份：
  ● 3月1日：个人所得税综合所得汇算清缴开始
  ● 3月15日前：2月份月度申报
  ● 3月底前：上年度财务报表审计完成（建议）

4月份：
  ● 4月15日前：Q1季度预缴
  ● 4月增值税纳税申报期内：出口退税凭证收齐后申报截止（2022年起放宽，不再是硬性4月截止，但超期需提供举证材料）

5月份：
  ● 5月31日前：企业所得税汇算清缴★（最重要）
  ● 5月31日前：关联申报（年度关联业务往来报告表）

6月份：
  ● 6月30日前：个人所得税综合所得汇算清缴截止★
  ● 6月30日前：工商年报公示截止
  ● 6月30日前：转让定价同期资料（主体文档/本地文档）准备完毕

7月-12月：
  ● 各季度预缴按期申报
  ● 高新技术企业到期复审（按有效期安排）
  ● 12月31日前：股东借款清理截止
  ● 12月31日前：各项税收优惠年度条件确认
```

## 第二步：深入分析——汇算清缴核心工作

**企业所得税汇算清缴（1月1日-5月31日）**：

| 工作项 | 完成时限 | 负责人 |
|-------|---------|-------|
| 年度财务报表定稿 | 3月底 | 财务+审计 |
| 纳税调整事项梳理 | 4月15日 | 税务 |
| 税收优惠享受确认 | 4月底 | 税务 |
| 关联交易申报准备 | 5月15日 | 税务 |
| 汇算清缴申报表填报 | 5月25日 | 税务 |
| 申报提交+税款补退 | 5月31日 | 税务+出纳 |

## 第三步：佐证/验证——年度关键截止日对照表

| 事项 | 截止日 | 逾期后果 |
|-----|-------|---------|
| 企业所得税汇缴 | 5月31日 | 滞纳金+罚款 |
| 个税综合所得汇缴 | 6月30日 | 补税加收滞纳金 |
| 工商年报 | 6月30日 | 列入经营异常名录 |
| 出口退税申报 | 次年4月申报期内（2022年起放宽，收齐凭证后均可申报） | 超期未申报需举证，否则按免税/内销处理 |
| 高新复审 | 证书到期前 | 失去15%优惠 |

## 第四步：结论/建议

- **倒排计划**：每年1月份制定全年税务日历，标注每项截止日
- **审计前置**：财务报表审计建议3月底前完成，为汇缴预留2个月
- **暂估清理**：5月31日前确保上年暂估入库的发票全部到位
- **多部门协作**：汇算清缴涉及财务/税务/业务/人力多部门，需提前协调
- **补退税安排**：汇缴补税在5月31日前缴纳，退税可在汇缴申报后申请

**风险提示**：
- 逾期未完成汇算清缴将产生滞纳金（每日万分之五）并可能影响纳税信用等级
- 工商年报逾期将被列入经营异常名录，影响企业信用和招投标资格
- 出口退税逾期未申报将按内销处理或免税处理，无法享受退税
