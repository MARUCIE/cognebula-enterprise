# 场景：报关差错补救

**政策依据**：《海关法》第二十六条；《海关进出口货物报关单修改和撤销管理办法》（海关总署令第220号）

## 第一步：识别报关差错类型

常见报关差错：
- **归类错误**：HS编码选错，导致税率不正确
- **价格申报错误**：完税价格申报偏高或偏低
- **数量/重量错误**：与实际到货不符
- **原产地错误**：影响协定税率适用
- **贸易方式错误**：一般贸易/加工贸易/暂时进出口申报错误
- **其他要素**：收发货人、运输方式、成交方式、币制等

差错的法律后果取决于性质：非主观故意的申报不实→补税+滞纳金；涉及逃避海关监管→行政处罚或刑事追诉。

## 第二步：主动披露与补正

《海关法》鼓励企业主动纠正错误。主动披露（海关总署公告2022年第54号）：企业在海关发现前主动向海关报告违规行为，如实披露违规事实，海关可以从轻或减轻处罚。

主动披露的条件：
- 在海关发现（立案调查/稽查通知）前主动报告
- 如实提供全部违规情况及相关证据
- 主动消除或减轻违法后果（补缴税款及滞纳金）

优惠：对主动披露且没有走私故意的，海关可以不予处罚或从轻处罚。补税金额50万以下的违规行为，主动披露且补缴税款的，免于处罚。

## 第三步：报关单修改/撤销流程

按220号令，报关单修改/撤销需向海关提出申请：

**放行前修改**：向现场海关申请修改，提交《进出口货物报关单修改/撤销申请表》及相关材料。

**放行后修改**：向报关单原申报口岸海关申请。材料包括：情况说明（错误原因、正确申报内容）、证明材料（合同、发票、运单等原始单证）、修改对照表。海关审核后决定是否同意修改。

修改涉及税款变化的，需补缴或退还差额税款。补缴的需加收滞纳金（自缴税期限届满之日起，每日万分之五）。

## 第四步：预防机制与合规管理

建立报关差错预防机制：
- **归类预审**：新品种进口前先确认HS编码（申请预裁定或咨询专业报关行）
- **双重校验**：报关单提交前经报关员和关务主管双重审核
- **系统校验**：在ERP/报关系统中设置逻辑校验规则（如价格区间校验、税号校验）
- **定期自查**：每季度抽查报关单，核对申报要素与实际是否一致
- **培训制度**：报关人员定期参加海关政策培训，了解最新归类/审价动态

关键输出：差错识别分析 + 主动披露申请书 + 报关单修改申请 + 补税/退税计算 + 预防措施方案。
