# 场景：行业分类与涉税特征

**政策依据**：《国民经济行业分类》（GB/T 4754-2017）；各行业增值税税率适用规定

## 第一步：行业大类与增值税税率映射

| 行业门类 | 典型业务 | 增值税税率 |
|----------|----------|-----------|
| A 农林牧渔 | 种植/养殖/捕捞 | 9%（自产免税） |
| B 采矿业 | 煤炭/石油/矿石 | 13% |
| C 制造业 | 生产加工 | 13% |
| E 建筑业 | 施工/安装/装修 | 9% |
| F 批发零售 | 商品流通 | 13% |
| G 交通运输 | 物流/客运/货运 | 9% |
| H 住宿餐饮 | 酒店/餐饮 | 6% |
| I 信息技术 | 软件/互联网/IT服务 | 6%（软件即征即退超3%部分） |
| K 房地产 | 开发/销售/租赁 | 9%（销售）/9%（出租） |
| L 租赁和商务服务 | 咨询/广告/人力资源 | 6% |
| M 科学研究 | 研发/技术服务 | 6% |
| R 文体娱乐 | 文化/体育/娱乐 | 6% |

## 第二步：行业涉税特征分析

| 行业 | 关键涉税特征 | 高频风险点 |
|------|-------------|-----------|
| 建筑 | 异地预缴/简易计税/分包扣除 | 挂靠经营/虚开材料票 |
| 餐饮 | 进项抵扣受限/农产品收购 | 现金收入不入账 |
| 电商 | 平台代扣代缴/跨境税务 | 刷单虚增收入/个人账户收款 |
| 房地产 | 土地增值税/预缴制度 | 关联交易转移利润 |
| 软件 | 即征即退/加计扣除 | 研发费归集不规范 |
| 医疗 | 免税项目多/混合销售 | 免税与应税未分别核算 |
| 物流 | 进项分散/油料抵扣 | 私车费用/虚开油票 |

## 第三步：行业平均税负率参考

税负率 = 实际缴纳增值税 ÷ 不含税收入。偏离行业均值30%以上可能触发预警。

| 行业 | 增值税税负率参考 | 所得税贡献率参考 |
|------|----------------|----------------|
| 商贸批发 | 0.8%-1.5% | 0.5%-1% |
| 商贸零售 | 1.5%-2.5% | 1%-2% |
| 制造业 | 2%-4% | 1.5%-3% |
| 建筑业 | 1.5%-3% | 1.5%-2.5% |
| 服务业 | 3%-5% | 2%-4% |
| 软件业 | 1%-3%（即征即退后） | 2%-5% |

**注意**：以上为全国参考值，各地区存在差异。具体标准以当地税务机关公布的行业税负率为准。

## 第四步：建档应用

- 新客户接入时先确认行业分类（以营业执照经营范围为主）
- 对照行业涉税特征建立针对性审核清单
- 每月/每季核对客户实际税负与行业均值的偏离度
- 偏离超过30%时主动排查原因并形成书面说明备查
