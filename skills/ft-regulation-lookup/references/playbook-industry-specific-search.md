# 场景：行业专项法规检索

**政策依据**：各行业税收政策汇编；国家税务总局行业管理文件

## 第一步：行业专项法规分布

| 行业 | 专项法规领域 | 核心文件举例 |
|------|-------------|-------------|
| 房地产 | 土地增值税清算/预售预缴/成本分摊 | 国税发〔2006〕187号/国税函〔2010〕220号 |
| 建筑 | 异地预缴/简易计税/分包扣除 | 国家税务总局公告2016年第17号 |
| 金融 | 利息收入确认/资管产品增值税 | 财税〔2016〕140号/财税〔2017〕56号 |
| 软件 | 增值税即征即退/研发费加计 | 财税〔2011〕100号/财税〔2012〕27号 |
| 医疗 | 免税项目/药品流通 | 财税〔2019〕20号 |
| 农业 | 自产免税/收购发票/特殊扣除 | 财税〔2017〕37号 |
| 电商 | 平台代缴/跨境电商综合税 | 财税〔2016〕18号/海关总署公告 |
| 文化 | 出版物增值税优惠/文化事业建设费 | 财税〔2018〕53号 |

## 第二步：检索路径

**标准检索流程**：
1. 确定客户行业分类（参照GB/T 4754-2017）
2. 检索该行业的增值税专项政策
3. 检索该行业的企业所得税专项优惠
4. 检索该行业的特殊税种规定（如房地产→土地增值税）
5. 检索该行业的发票管理特殊规定
6. 检索该行业的地方执行口径

**检索关键词模板**：
```
"[行业名称] 增值税 [优惠/征收/管理]"
"[行业名称] 企业所得税 [税率/扣除/优惠]"
"[行业名称] 发票 [开具/抵扣/特殊规定]"
"[行业名称] [年份] 最新政策"
```

## 第三步：行业法规合集维护

为高频行业建立法规速查表：

```
## 建筑行业法规速查
- 一般计税vs简易计税：财税〔2016〕36号附件2
- 异地预缴：国家税务总局公告2016年第17号
- 分包扣除：国家税务总局公告2016年第17号
- 工程发票备注栏：国家税务总局公告2016年第23号
- 跨区域涉税事项报告：国家税务总局公告2017年第38号
```

## 第四步：注意事项

- **行业交叉**：一家企业可能涉及多个行业（如制造+销售+服务），须分别检索各行业法规
- **行业专属优惠**：部分优惠政策有行业限制（如集成电路企业"两免三减半"），须确认客户是否属于目标行业
- **行业排除**：部分通用优惠明确排除特定行业（如研发费加计扣除排除住宿餐饮等7个行业）
- **KG行业标签**：在知识图谱中为法规节点添加行业标签，实现按行业快速筛选
