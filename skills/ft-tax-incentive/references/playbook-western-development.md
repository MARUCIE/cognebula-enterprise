# 场景五：西部大开发税收优惠

**政策依据**：财税〔2011〕58号；财政部公告2020年第23号（延续至2030年12月31日）；《西部地区鼓励类产业目录》

## 第一步：初步判别——适用条件

1. **注册地**：西部12省区市（重庆、四川、贵州、云南、西藏、陕西、甘肃、青海、宁夏、新疆、内蒙古、广西）
2. **主营业务**：属于《西部地区鼓励类产业目录》中的产业项目
3. **收入占比**：主营业务收入占企业收入总额≥60%

**优惠内容**：企业所得税减按15%征收（标准25%，节省10个百分点）

## 第二步：深入分析——鼓励类产业目录

**目录构成**：
1. **国家现有产业结构调整指导目录中的鼓励类产业**（全国通用部分）
2. **西部地区新增鼓励类产业**（各省区市专属部分）

**常见鼓励类产业**：
| 行业 | 示例产业 |
|-----|---------|
| 信息技术 | 软件开发、信息系统集成、数据处理 |
| 先进制造 | 新能源设备、智能装备、航空航天零部件 |
| 现代农业 | 农产品深加工、生物农药、有机农业 |
| 文旅健康 | 旅游开发（部分省区）、中医药种植 |
| 新能源 | 光伏、风电、储能 |
| 环保 | 污水处理、固废处理、生态修复 |

**60%收入占比计算**：
- 分子：鼓励类产业相关的主营业务收入
- 分母：企业全部收入总额（含其他业务收入、营业外收入）
- 按自然年度计算

## 第三步：佐证/验证——实务风险点

1. **实质性经营判定**（最大风险）：
   - 税务机关重点核查：注册地是否有真实办公/生产场所
   - 核心管理人员是否在当地
   - 社保/公积金缴纳地与注册地是否一致
   - "空壳注册"享受优惠的将被取消资格+补税+罚款

2. **产业目录匹配**：
   - 须逐条比对目录编码，不能"大概对应"
   - 主营业务变化后可能不再匹配

3. **60%比例波动**：
   - 某年非主营收入激增（如资产处置收入）可能导致比例不达标
   - 不达标年度不得享受15%优惠

4. **备案要求**：
   - 自行判别、申报享受、相关资料留存备查
   - 无需事前审批，但税务机关可后续核查

## 第四步：结论/建议

- **选址规划**：新设企业如业务匹配鼓励目录，优先选择西部城市（成都/重庆/西安最成熟）
- **实质化运营**：确保注册地有真实经营活动（办公场所+人员+银行+社保）
- **收入管控**：年度内监控鼓励类收入占比，确保≥60%
- **10年红利**：政策延续至2030年底，长期规划价值显著
- **叠加可能**：西部优惠15%可与研发加计扣除叠加（但不可与高新15%同时适用，因税率相同）
