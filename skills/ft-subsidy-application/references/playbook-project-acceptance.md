# 场景：项目验收流程

**政策依据**：《科技计划项目验收管理办法》；各省市专项资金项目验收实施细则

## 第一步：确认验收时间与组织方式

项目验收通常在项目实施期满后3-6个月内完成。验收组织方式：
- **主管部门组织验收**：由项目管理部门（科技局/发改委/工信局）组织专家组验收
- **委托第三方验收**：委托独立评估机构进行验收评审
- **自行验收**：小额项目由企业自行验收后报主管部门备案

验收形式：会议验收（专家组实地考察+听取汇报+质询）、通讯验收（专家远程审阅材料出具意见）、现场验收（大型项目必须现场核查）。

如需延期验收，应在项目到期前提出书面申请，说明延期原因和预计完成时间（通常最多延期一年）。

## 第二步：准备验收材料

验收材料清单（按重要性排序）：
1. **项目总结报告**：项目实施过程、技术路线、取得的成果、经济社会效益
2. **技术成果证明**：专利证书、软件著作权、产品检测报告、用户使用报告
3. **财务决算报告**：经审计的专项资金使用报告（通常需委托会计师事务所审计）
4. **资金使用明细**：与预算对照的实际支出明细表
5. **产出指标完成对照表**：将任务书/合同书中约定的指标与实际完成情况逐项对照
6. **过程材料**：研发日志、试验数据、会议纪要、阶段性成果

## 第三步：验收评审关键环节

专家评审重点关注：
- **任务完成度**：合同约定的技术指标和经济指标是否达标（最核心）
- **资金使用合规性**：是否专款专用、预算执行偏差是否合理
- **成果创新性**：是否具有自主知识产权、是否达到国内/国际先进水平
- **产业化前景**：成果是否具有市场应用价值和推广潜力

验收结论通常分为：通过、不通过（需整改后复验）、终止。不通过的常见原因：核心技术指标未达标、资金使用存在严重问题、项目内容与任务书偏差过大。

## 第四步：验收后续工作

验收通过后：
- 将验收材料归档（保存至少10年）
- 办理资产确权（项目形成的固定资产的权属确认）
- 知识产权归属确认（按合同约定，通常归承担单位所有但需标注政府资助信息）
- 配合成果登记（在科技成果登记系统中登记）

如项目涉及后补助（验收通过后拨付尾款），应在验收通过后及时提交拨款申请。项目成果的后续产业化情况可能需要在2-3年内持续报告。

关键输出：验收总结报告 + 技术成果证明材料 + 财务决算审计报告 + 指标完成对照表 + 延期/整改申请（如需）。
