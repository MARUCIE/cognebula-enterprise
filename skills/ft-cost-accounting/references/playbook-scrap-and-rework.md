# 场景七：废料与返工成本

**政策依据**：《企业会计准则第1号——存货》第9条；《企业产品成本核算制度（试行）》（财会〔2013〕17号）第15条

## 第一步：初步判别——废料/返工的分类

**废料（Scrap）**：生产过程中产生的边角料、下脚料，有一定残值但不能继续加工为产品

**废品（Spoilage）**：
- **正常废品**：生产过程中不可避免的次品（在合理废品率范围内）
- **非正常废品**：超出正常废品率的次品（管理/操作失误导致）

**返工品（Rework）**：通过返工可修复达到产品标准的不合格品

## 第二步：深入分析——各类处理方式

**正常废品/废料处理**：
| 情形 | 会计处理 |
|-----|---------|
| 正常废品净损失 | 计入产品成本（由合格品承担） |
| 废料残值（可出售） | 冲减生产成本或计入其他业务收入 |
| 废料残值（自用） | 按估计价值入库，冲减生产成本 |

**非正常废品处理**：
| 情形 | 会计处理 |
|-----|---------|
| 非正常废品净损失 | 计入营业外支出（不得计入产品成本） |
| 可追偿部分 | 计入其他应收款 |

**返工品处理**：
| 情形 | 会计处理 |
|-----|---------|
| 正常返工（合理范围内） | 返工成本计入产品成本 |
| 非正常返工 | 返工成本计入当期损益 |

## 第三步：佐证/验证——成本核算细节

**正常废品净损失计算**：
```
废品总成本（材料+人工+制造费用到废弃点为止）
 - 废品残值（可出售价值）
 - 可追偿金额（保险赔偿/责任人赔偿）
= 废品净损失（分摊至合格品成本）
```

**返工成本计算**：
```
返工消耗的材料（额外投入）
+ 返工耗用的人工（额外工时×工资率）
+ 返工分摊的制造费用
= 返工总成本
```

**税务影响**：
1. 正常损耗的进项税可以抵扣（无需转出）
2. 非正常损失的进项税须转出（《增值税暂行条例》第10条）
3. 非正常损失包括：管理不善导致的变质/被盗/霉烂/丢失
4. **自然灾害等不可抗力损失不属于非正常损失**（2016年36号文后的口径）

**检查要点**：
- 废品率是否在行业合理范围内
- 是否建立了废品/返工台账
- 废料出售是否开具发票确认收入

## 第四步：结论/建议

- **废品率标准**：制定各产品线的正常废品率上限，超出部分按非正常处理
- **返工台账**：每次返工记录原因、工序、耗用工时和材料，归集返工成本
- **废料管理**：建立废料出入库制度，定期清理出售，收入不可忽略
- **进项税转出**：非正常废品对应的进项税必须转出（管理不善导致的）
- **持续改善**：废品/返工数据是质量管理的关键输入，按月统计分析并反馈给生产部门
