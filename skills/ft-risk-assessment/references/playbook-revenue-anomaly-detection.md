# 场景二：收入异常指标检测

**政策依据**：国税发〔2005〕43号（纳税评估管理办法）；《增值税暂行条例》第19条（纳税义务发生时间）

## 第一步：初步判别——收入异常的信号

1. **收入与发票不匹配**：开票金额远低于实际经营规模
2. **收入增长异常**：同比大幅下降但成本未同步下降
3. **收入与资金流不匹配**：银行入账金额远大于申报收入
4. **收入结构突变**：应税收入占比突然下降，免税/不征税收入占比飙升

## 第二步：深入分析——六大异常指标

| 指标 | 公式 | 异常信号 |
|-----|-----|---------|
| 收入变动率 | (本期-上期)÷上期×100% | 下降>20%且无合理原因 |
| 收入成本配比率 | 收入变动率÷成本变动率 | 明显偏离1（收入降成本不降） |
| 存货与收入比 | 期末存货÷年收入 | 持续升高（可能隐匿收入） |
| 往来款异常 | 预收账款÷收入 | 长期大额预收未转收入 |
| 银行流水比 | 银行收入流水÷申报收入 | 比值>1.5（体外循环嫌疑） |
| 关联交易价差 | 关联方交易价÷市场公允价 | 偏离>20%（转让定价风险） |

## 第三步：佐证/验证——隐匿收入的常见模式

1. **账外经营**：现金收款不入账，通过个人银行账户收款
   - 检测：银行流水与申报收入比对
2. **延迟确认**：已发货/已完成服务但不开票不确认收入
   - 检测：出库单/运输单与收入确认的时间差
3. **收入挂往来**：将销售收入挂入"预收账款""其他应付款"
   - 检测：预收账款/其他应付款余额异常增大
4. **关联方低价销售**：向关联方低于市场价销售转移利润
   - 检测：关联交易价格与市场价偏离度
5. **视同销售遗漏**：自用/赠送/投资等未按视同销售处理
   - 检测：存货减少但无对应收入

**税务机关常用检查手段**：
- 金税系统发票数据与申报数据交叉比对
- 银行账户查询（包括法定代表人和关联人员个人账户）
- 上下游企业数据互查（进销项匹配）

## 第四步：结论/建议

- **月度检测**：每月计算收入变动率和收入成本配比率，异常波动立即排查
- **银行对账**：企业银行流水与账面收入定期核对（至少季度一次）
- **视同销售自查**：每年检查是否有应做视同销售未做的情形
- **预收清理**：长期挂账的预收账款须逐笔核实是否应转收入
- **关联定价**：关联交易价格保留定价依据（可比非受控价格法/成本加成法）
