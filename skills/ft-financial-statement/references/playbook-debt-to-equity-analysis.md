# 场景：资产负债率预警

**政策依据**：《企业会计准则——基本准则》；银行授信评估标准；税务风险指标参考

## 第一步：指标计算与分级

**资产负债率** = 负债总额 / 资产总额 × 100%

| 区间 | 风险等级 | 建议 |
|------|----------|------|
| <40% | 保守 | 可适当增加杠杆提升ROE |
| 40%-60% | 适中 | 健康区间 |
| 60%-70% | 偏高 | 关注偿债能力 |
| 70%-80% | 高风险 | 警惕流动性危机 |
| >80% | 极高风险 | 可能资不抵债 |

**行业差异**：
- 房地产/建筑：70%-80%属于正常（行业特征高杠杆）
- 制造业：50%-65%为正常
- 科技/软件：30%-50%为正常（轻资产）

## 第二步：负债结构分析

| 分析维度 | 健康信号 | 风险信号 |
|----------|----------|----------|
| 流动负债占比 | <60%（有充足长期资金支持） | >80%（短期偿债压力大） |
| 有息负债率 | 利息保障倍数>3 | 利息保障倍数<1.5 |
| 经营性负债 | 应付账款/预收账款为主 | 短期借款为主 |
| 隐性负债 | 无表外担保 | 大额对外担保/关联方借款 |

**流动负债vs非流动负债**：
- 短期借款到期集中 → 续贷风险
- 应付账款持续增长 → 可能是供应商账期被动拉长
- 预收账款增长 → 好信号（已锁定未来收入）

## 第三步：偿债能力综合评估

**短期偿债**：
```
流动比率 = 流动资产 / 流动负债（>1.5为佳）
速动比率 = (流动资产 - 存货) / 流动负债（>1.0为佳）
现金比率 = (货币资金 + 交易性金融资产) / 流动负债（>0.25为底线）
```

**长期偿债**：
```
利息保障倍数 = EBIT / 利息费用（>3倍为安全）
经营现金流/总负债 = 经营活动现金流净额 / 负债总额（>0.2为健康）
```

## 第四步：预警与税务关联

**经营预警**：
- 资产负债率连续3个季度上升 → 启动现金流压力测试
- 流动比率<1 → 警告短期可能无法偿付到期债务
- 利息保障倍数<1.5 → 盈利无法覆盖利息，续贷困难

**税务关联**：
- 资产负债率过高的企业是税务风险管理关注对象（可能通过虚增成本减少利润来降低偿债压力）
- 股东借款（其他应付款）过大 → 可能被认定为关联融资，利息扣除受债资比限制（2:1）
- 资不抵债但仍正常经营 → 税务可能质疑是否有隐匿收入
