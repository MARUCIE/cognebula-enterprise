# 场景：全电发票处理流程

**政策依据**：国家税务总局公告2021年第23号（全面数字化的电子发票试点）；国家税务总局关于进一步开展全电发票试点的公告

## 第一步：全电发票基础识别

**全电发票特征**：
- 无纸质形态，以电子文件（XML格式+PDF版式文件）交付
- 票面信息简化：取消发票代码，仅有20位发票号码
- 票种统一：不再区分增值税专用发票和普通发票，统一为"电子发票（增值税专用发票）"和"电子发票（普通发票）"
- 无需税控设备，通过电子发票服务平台开具

**与传统电子发票的区别**：
| 维度 | 全电发票 | 传统电子发票 |
|------|----------|-------------|
| 开具平台 | 电子发票服务平台 | 税控系统 |
| 发票代码 | 无 | 有（12位） |
| 发票号码 | 20位 | 8位 |
| 认证抵扣 | 无需认证，自动归集 | 需勾选认证 |
| 交付方式 | 平台推送/下载 | 邮件/二维码 |

## 第二步：接收与入账流程

1. **接收**：通过电子发票服务平台"税务数字账户"查看收到的全电发票
2. **查验**：在平台上查验发票真伪和状态（正常/红冲/异常）
3. **下载**：下载XML源文件和PDF版式文件
4. **入账**：将XML文件导入财务系统或手工录入
5. **归档**：按《会计档案管理办法》电子归档（XML文件为法定归档格式）

**报销审核要点**：
- 确认发票状态为"正常"（非红冲/异常）
- 核对购方信息（名称、税号）是否正确
- 查验备注栏必填信息是否完整
- 确认是否已被重复报销（通过发票号码查重）

## 第三步：进项税额抵扣

全电发票抵扣流程简化：
- 无需手动勾选认证
- 进入"税务数字账户"→"发票勾选确认"→ 按期确认用途
- 选择"抵扣勾选"或"退税勾选"或"代办退税勾选"
- 确认后自动计入当期可抵扣进项

## 第四步：风险提示

- **纸电并行期**：目前全电发票与纸质发票并行使用，收票方需兼容两种格式
- **防重复报销**：全电发票易被重复打印和报销，必须建立发票号码台账进行查重
- **电子归档**：全电发票不需要打印纸质件归档，但需确保电子文件完整保存且可调取（保存期不少于10年）
- **开票限额**：全电发票采用"授信制"（总额度控制），开票金额占用额度，红冲后释放额度
