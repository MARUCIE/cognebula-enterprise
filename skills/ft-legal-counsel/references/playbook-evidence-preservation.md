# 场景九：税务争议证据留存

**政策依据**：《行政诉讼法》第34条（被告举证责任）；《税务稽查工作规程》（国税发〔2009〕157号）；《会计档案管理办法》（财会〔2015〕25号）

## 第一步：初步判别——何时需要启动证据留存

1. **常规留存**：日常经营中建立证据档案（预防性）
2. **风险预警**：收到税务风险提示函/自查通知（加强留存）
3. **稽查启动**：收到税务稽查通知书（全面固定证据）
4. **争议进行**：复议/诉讼阶段（证据收集和组织）

**核心原则**：证据在产生时即应规范留存，事后补做证据效力大打折扣。

## 第二步：深入分析——税务证据分类与留存标准

**书证类**：
| 证据类别 | 具体内容 | 保存期限 |
|---------|---------|---------|
| 会计凭证 | 记账凭证+原始凭证 | 30年 |
| 会计账簿 | 总账/明细账/日记账 | 30年 |
| 财务报表 | 月报/季报/年报 | 永久 |
| 税务申报表 | 增值税/所得税/个税等 | 永久（建议） |
| 合同协议 | 购销合同/服务合同 | 合同终止后10年 |
| 发票 | 增值税专票/普票/电子发票 | 30年 |

**电子证据类**：
- 电子税务局申报记录及回执截图
- 金税系统开票数据和认证记录
- 银行电子回单和流水
- ERP/财务系统操作日志

**言词证据类**：
- 与税务机关的沟通记录（书面优于口头）
- 税务约谈的记录和签字确认文件
- 内部决策的会议纪要

## 第三步：佐证/验证——证据固定操作规范

**稽查通知后的紧急证据固定**：

1. **不要做的事**：
   - 销毁、篡改、隐匿任何账簿凭证（构成犯罪情节）
   - 与相关方串通统一口径（可能加重处罚）
   - 对电子数据进行删除或覆盖操作

2. **立即要做的事**：
   - 复印所有将被调取的账簿凭证（保留副本）
   - 截图保存电子税务局历史申报数据
   - 整理银行流水并核对资金往来
   - 收集所有合同协议原件
   - 拍照/录像保存实物证据（如库存、固定资产）

3. **系统备份**：
   - 财务系统数据库完整备份
   - ERP系统相关模块数据导出
   - 邮件系统中相关业务往来邮件归档

4. **证人证言**：
   - 关键业务经办人的书面说明
   - 业务流程和决策过程的记录

## 第四步：结论/建议

- **日常制度**：建立税务档案管理制度，指定专人负责，定期检查完整性
- **电子化备份**：纸质凭证扫描电子化存储（双重保险）
- **时间戳**：电子证据注意保留时间戳（证明形成时间）
- **第三方存证**：重要的电子证据可使用区块链/公证存证（增强证明力）
- **律师指导**：收到稽查通知后，在律师指导下进行证据整理，避免无意中损害自身利益
