# 场景：知识图谱关系探索

**政策依据**：《税收征收管理法》全文（18个税种的法规体系）；《立法法》第九十二条（法律效力层级与适用规则）

## 第一步：KG核心实体类型

| 实体类型 | 说明 | 示例 |
|----------|------|------|
| 税种(TaxType) | 中国现行18个税种 | 增值税、企业所得税 |
| 法规(Regulation) | 法律/条例/公告/通知 | 《企业所得税法》 |
| 条款(Clause) | 法规中的具体条/款/项 | 第四十三条 |
| 税目(TaxItem) | 税种下的征税项目 | 销售货物、提供服务 |
| 税率(TaxRate) | 适用税率/征收率 | 13%、9%、6% |
| 行业(Industry) | 国民经济行业分类 | 制造业、建筑业 |
| 优惠政策(Incentive) | 减免/返还/加计等 | 小微企业5%优惠 |
| 业务场景(Scenario) | 具体业务场景 | 固定资产一次性扣除 |

## 第二步：核心关系类型

```
税种 -[包含]-> 税目
税目 -[适用]-> 税率
法规 -[规定]-> 条款
条款 -[引用]-> 其他条款
法规 -[废止]-> 旧法规
优惠政策 -[适用于]-> 行业
优惠政策 -[依据]-> 条款
业务场景 -[涉及]-> 税种
业务场景 -[适用]-> 优惠政策
```

## 第三步：常用查询模式

**场景驱动查询**（最常用）：
```
Q: 某业务场景涉及哪些税种？
路径: 业务场景 → [涉及] → 税种 → [适用] → 税率

Q: 某行业可享受哪些优惠？
路径: 行业 → [适用] ← 优惠政策 → [依据] → 条款

Q: 某法规被哪些新法规替代？
路径: 旧法规 → [被废止] ← 新法规
```

**条件筛选查询**：
```
Q: 所有适用于小规模纳税人的增值税优惠
条件: 税种=增值税 AND 适用对象=小规模纳税人 AND 状态=有效

Q: 2023年后新发布的企业所得税相关法规
条件: 税种=企业所得税 AND 发布日期>=2023-01-01
```

**关系链探索**：
从一个节点出发，沿着关系链逐步展开：
```
起点: "业务招待费"
→ [扣除限额] → 60%和5‰双限额
→ [依据] → 企业所得税法实施条例第43条
→ [属于] → 企业所得税
→ [相关场景] → 年终汇算清缴纳税调整
```

## 第四步：探索技巧

- **广度优先**：不确定时先看一个节点的所有直接关系，再决定深入方向
- **深度优先**：已知目标时沿特定关系链快速到达目标节点
- **反向查询**：从结果出发反向追溯（如"哪些法规提到了20%税率"）
- **模式匹配**：寻找类似场景的处理模式（如"与业务招待费类似的限额扣除项目有哪些"）
