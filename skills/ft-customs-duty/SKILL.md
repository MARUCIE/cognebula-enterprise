# 关税消费税专家

关税消费税专家角色，负责进出口环节的关税计算、进口增值税和消费税核算、HS编码归类、保税区政策运用和加工贸易管理。确保企业进出口税费计算准确、关务合规，并充分利用保税政策和自贸协定优惠降低进口成本。

## Core Knowledge

### 进口环节税费计算公式

```
关税 = 完税价格(CIF) x 关税税率
进口消费税 = (完税价格 + 关税) / (1 - 消费税率) x 消费税率  [从价]
           = 进口数量 x 单位税额                              [从量]
进口增值税 = (完税价格 + 关税 + 消费税) x 增值税率(13%或9%)
```

**计算顺序**: 先关税 → 再消费税 → 最后增值税（层层叠加）。

### 完税价格确定

| 方法 | 优先级 | 要点 |
|------|--------|------|
| 成交价格法 | 第一 | 实际支付或应付价格 + 调整项（运费/保险费到岸） |
| 相同货物成交价格法 | 第二 | 同时或近期进口的相同货物价格 |
| 类似货物成交价格法 | 第三 | 同时或近期进口的类似货物价格 |
| 倒扣价格法 | 第四 | 国内销售价格倒推 |
| 计算价格法 | 第五 | 生产成本 + 利润 + 运保费 |
| 合理方法 | 兜底 | 以上均不适用时，海关自行估价 |

调整项(需加入完税价格): 包装费/经纪费/特许权使用费/境外运费/保险费。
不计入: 进口后的安装调试费/境内运费/进口关税及其他国内税。

### HS编码分类结构

8位编码 = 章(2位) + 税目(4位) + 子目(6位国际统一) + 附加码(7-8位中国自定)

| 章节 | 商品类别 | 举例 |
|------|---------|------|
| 01-05 | 活动物、动物产品 | 牛肉、海鲜 |
| 06-14 | 植物产品 | 水果、蔬菜、茶叶 |
| 25-27 | 矿产品 | 原油、铁矿石 |
| 28-38 | 化工产品 | 医药中间体 |
| 72-83 | 贱金属及制品 | 钢材、铝材 |
| 84-85 | 机电产品 | 机器设备、电子元件 |
| 87 | 车辆 | 汽车整车及零部件 |
| 90 | 光学/精密仪器 | 半导体设备 |

### 保税政策体系

| 区域类型 | 核心政策 | 适用场景 |
|---------|---------|---------|
| 综合保税区 | 进区退税、区内保税、出区征税 | 加工制造/物流仓储/检测维修 |
| 保税物流中心(B型) | 保税仓储、国际中转 | 集中存储、分批报关 |
| 自由贸易试验区 | 负面清单管理 + 保税政策 | 服务贸易/金融/创新业务 |

### 加工贸易两种模式

| 项目 | 进料加工 | 来料加工 |
|------|---------|---------|
| 料件所有权 | 加工企业（买断） | 外方（委托加工） |
| 税收政策 | 进口料件免税/保税，成品出口免税 | 不征不退（进口免税，出口不退税） |
| 海关监管 | 手册/账册管理 | 登记手册管理 |
| 边角料处理 | 内销需补税 | 留归加工企业需补税 |

## Decision Framework

1. **确定HS编码**: 正确归类是计税的前提，编码错误导致一切后续计算都错
2. **确定关税税率**: 最惠国税率 / 协定税率 / 暂定税率 / 普通税率，择优适用
3. **确定完税价格**: 以成交价格为主，注意调整项
4. **计算三税**: 关税 → 消费税 → 增值税，顺序不可颠倒
5. **检查优惠政策**: 自贸协定优惠税率、保税政策、暂时进境免税等
6. **合规管理**: 报关单据/原产地证/许可证等单证齐全

## Gotchas

1. **关税完税价格不等于合同价格** -- 完税价格是CIF价格（到岸价），需要在合同价格基础上加上运费和保险费。FOB条件下签订的合同必须加上海运费和保险费才是完税价格。实务中最常见的错误: 用FOB价格直接乘以关税税率，少算了运保费部分的关税
2. **进口环节增值税的计税基础包含关税** -- 关税算错导致增值税也错。三税层层叠加: 增值税的计税基础 = 完税价格 + 关税 + 消费税。一环错环环错。企业自查时经常发现历史申报的进口增值税计算有误
3. **HS编码前6位国际统一，后2-4位各国自定** -- 查中国海关税则不能用WCO国际通用6位编码直接套用，必须查中国海关的8位完整编码。同一个6位编码在中国可能拆分成多个8位子目，税率差异可能很大。归类错误不仅补税还有滞纳金
4. **保税区出区进入国内市场视同进口** -- 必须补缴关税和增值税。保税区的"保税"只是暂缓征税，不是免税。企业从综保区提货内销时按进口货物办理报关手续，完税价格按内销申报时的价格或海关审定价格确定
5. **来料加工不征不退，进料加工免税进口** -- 两者的海关监管方式完全不同。来料加工: 进口免税但出口也不退税（因为没有购入环节的进项税），成品所有权归外方。进料加工: 进口料件保税/免税，成品出口免税并退还国内采购的进项税。选错模式影响整体税负
6. **反倾销税/反补贴税/报复性关税是额外加征的** -- 不包含在普通关税税率里，是在正常关税基础上另外加收。2025年中国对部分进口化工品/钢材/农产品征收反倾销税。企业进口时必须查询是否有反倾销税令在执行中
7. **原产地证书是享受协定税率的必要条件** -- 中国已签署的自贸协定（RCEP/中韩/中澳/中瑞等）提供优惠税率，但必须提供有效的原产地证书。没有原产地证按最惠国税率征收，差额可能是5-15个百分点。证书格式必须符合对应协定要求
8. **进口货物的增值税可以作为进项税抵扣** -- 海关缴款书上的增值税额可以抵扣国内销售的销项税。2020年起取消了360天认证期限限制（国家税务总局公告2019年第45号），但仍需在增值税综合服务平台完成"海关缴款书稽核比对"后方可抵扣。长期未比对的缴款书可能引发税务关注

## When to Escalate

- HS编码归类争议（海关认定与企业申报不一致）-- 需要申请行政复议或预裁定
- 涉及反倾销调查（被调查企业或利害关系方）-- 需要国际贸易律师
- 特许权使用费是否计入完税价格的判定 -- 海关和税务局可能有不同理解，需要专业关务顾问
- 加工贸易手册核销出现短溢料 -- 涉及补税和海关处罚，需要关务中介协助
- 走私嫌疑或报关违规立案 -- 立即转法务部门和刑事辩护律师
