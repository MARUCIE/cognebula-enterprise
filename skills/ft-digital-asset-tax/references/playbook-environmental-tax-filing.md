# 场景七：环保税申报流程

**政策依据**：《环境保护税法》（2018年1月1日起施行）；《环境保护税法实施条例》

## 第一步：初步判别——是否为环保税纳税人

**纳税人**：在中华人民共和国领域和管辖的其他海域，直接向环境排放应税污染物的企业事业单位和其他生产经营者

**应税污染物**（四类）：
1. **大气污染物**：二氧化硫、氮氧化物、颗粒物等
2. **水污染物**：化学需氧量、氨氮、总磷等
3. **固体废物**：煤矸石、尾矿、粉煤灰等
4. **噪声**：工业噪声超标分贝数

**不征环保税的情形**：
- 排入集中式污水处理/生活垃圾集中处理设施的（符合环境标准的）
- 农业生产排放的
- 机动车/铁路/船舶/航空器排放的
- 依法设立的生活垃圾焚烧发电厂等

## 第二步：深入分析——税额计算

**计税依据**：
| 污染物类别 | 计税单位 | 税额标准（国家下限） |
|-----------|---------|-------------------|
| 大气污染物 | 每污染当量 | 1.2-12元（各省自定） |
| 水污染物 | 每污染当量 | 1.4-14元（各省自定） |
| 固体废物 | 每吨 | 5-1,000元（按类别） |
| 噪声 | 每月每超标分贝 | 350-11,200元 |

**各省差异极大**：
- 北京：大气12元/当量，水14元/当量（全国最高）
- 多数省份：大气1.2-3.6元，水1.4-3.5元

**减免优惠**：
- 排放浓度值低于国家标准30% → 减按75%征收
- 排放浓度值低于国家标准50% → 减按50%征收

## 第三步：佐证/验证——申报操作

**申报周期**：按月计算，按季申报（季度终了15日内）

**申报数据来源**：
1. **自动监测数据**：安装自动监测设备的（优先采用）
2. **监测机构数据**：委托第三方环境监测机构
3. **排污系数法**：按国务院环境保护主管部门规定的排污系数计算
4. **物料衡算法**：按物料衡算方法计算

**申报流程**：
```
季度末 → 汇总本季度排放数据
  → 登录电子税务局环保税申报模块
    → 填写各污染物排放量
      → 系统自动计算税额
        → 核对确认后提交
          → 税款缴纳
```

## 第四步：结论/建议

- **监测设备**：安装在线自动监测设备的数据最具公信力
- **省级税额**：务必确认所在省的环保税税额标准（省间差距可达10倍）
- **减排优惠**：投资减排设施可降低排放浓度，享受减征优惠
- **环保+税务协同**：环保部门监测数据与税务申报数据须一致
- **主动减排**：环保税的立法目的是"多排多缴、少排少缴、不排不缴"，减排是根本
