# 场景：成本费用真实性核查

**政策依据**：《企业所得税法》第八条；《企业所得税法实施条例》第二十七条；国家税务总局公告2018年第28号（企业所得税税前扣除凭证管理办法）

## 第一步：核查原则——三性审查

成本费用税前扣除必须同时满足"三性"：
1. **真实性**：支出确实已经发生（有合同/发票/付款凭证/交付记录）
2. **相关性**：与取得收入直接相关，属于企业正常经营必要支出
3. **合理性**：金额、频次、比例符合商业常规和行业惯例

**核心规则**：发票只是扣除凭证之一，不是真实性的唯一证据。有发票但无真实业务的，不得扣除（反之，真实业务取得非发票凭证但符合28号公告的，可以扣除）。

## 第二步：高风险费用项目重点核查

| 费用类型 | 核查要点 | 常见问题 |
|----------|----------|----------|
| 咨询费/顾问费 | 咨询合同+咨询报告+服务过程记录 | 无实际服务内容，虚列费用 |
| 会议费 | 会议通知+签到表+议程+场地发票 | 以会议费列支餐饮娱乐 |
| 差旅费 | 出差审批单+行程单+住宿发票 | 个人旅游费用列入差旅 |
| 办公费 | 采购清单+入库单+领用记录 | 金额异常偏高或品名不合理 |
| 劳务费 | 劳务合同+成果交付+付款凭证 | 无真实劳务关系 |
| 佣金/手续费 | 合同+合法经营资质+比例合理 | 超过合同金额5%限额 |

## 第三步：凭证审查标准（28号公告）

**必须取得发票的**：境内增值税应税交易，以发票作为税前扣除凭证。

**可不取得发票的**：
- 对方为无需办理税务登记的单位或从事小额零星经营业务的个人 → 收据+身份信息即可（每次≤500元）
- 境外支出 → 对方开具的收款凭证+合同+付汇凭证
- 员工出差的火车票、飞机票等运输凭证 → 可直接作为扣除凭证
- 工资薪金 → 工资表+银行付款记录

## 第四步：不合规后果

- 不符合三性要求的费用不得税前扣除，须做纳税调增
- 虚列成本费用金额较大的，按偷税处理（《税收征管法》第六十三条）
- 取得虚开发票列支费用的，除不得扣除外还面临进项转出、发票违法处罚
- **补救措施**：汇算清缴前发现凭证不合规的，应要求对方换开或补开发票；对方无法补开的，可凭相关资料证明真实性后扣除（28号公告第十四条）
